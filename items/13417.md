## Summary

Current main still does not implement WhatsApp Newsletter/Channel JID support for the message tool. The issue has concrete reproduction detail, the related PR #13424 was closed unmerged, and current code/docs/tests still describe and enforce WhatsApp targets as E.164 numbers or group JIDs only.

## Best Possible Solution

Keep #13417 open until the bundled WhatsApp plugin supports Newsletter/Channel targets end to end: add newsletter JID recognition and normalization, define outbound allowlist behavior, add a consistent session/chatType model, route sends through the active listener, add focused tests, and update WhatsApp/message docs. Newsletter metadata/follow/react actions can be separate follow-up work after send-target support is correct.
