## Summary

Keep open. Current main still allows caller-supplied temperature to reach the Anthropic Messages request for non-thinking Claude Opus 4.7 runs, so this PR targets a real remaining bug from #72123 rather than obsolete work. The submitted patch still needs cleanup and focused regression coverage before merge.

## Best Possible Solution

Keep this PR open or land an equivalent small fix on main: strip temperature for claude-opus-4-7 and claude-opus-4.7 in the Anthropic Messages transport option and payload paths, preserve temperature forwarding for supported Anthropic models, add focused regression tests for omission and pass-through behavior, and decide whether the Anthropic Vertex path needs the same capability filter or a separate follow-up.
