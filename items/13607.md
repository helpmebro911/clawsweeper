## Summary

Keep #13607 open. Current main still lacks a schema-backed global `agents.defaults.reasoningDefault`; the implemented surface is per-agent `agents.list[].reasoningDefault` plus session/directive overrides. The open related PR #71817 is explicitly partial Telegram/Feishu per-agent work and does not supersede this global-default request.

## Best Possible Solution

Keep #13607 open until OpenClaw adds a schema-backed `agents.defaults.reasoningDefault` with clear precedence: inline `/reasoning` and session override > per-agent `agents.list[].reasoningDefault` > global `agents.defaults.reasoningDefault` > model-capability/off fallback. The fix should update config types, Zod schema, generated schema/help/labels, docs, reply resolution, current-level/status paths, relevant channel adapters, and regression tests. Treat #71817 as useful partial per-agent channel work, not the canonical global-default implementation.
