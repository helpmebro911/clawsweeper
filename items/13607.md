## Summary

Keep #13607 open. Current main still does not implement a schema-backed global `agents.defaults.reasoningDefault`; it only supports per-agent `agents.list[].reasoningDefault` and session/directive overrides. The related open PR #71817 is explicitly partial for Telegram/Feishu per-agent behavior and does not supersede the requested global default.

## Best Possible Solution

Keep #13607 open until OpenClaw adds a schema-backed global `agents.defaults.reasoningDefault` with clear precedence: inline `/reasoning` and session override > per-agent default > global default > model-capability/off fallback. The implementation should update config types, strict zod schema, generated schema/help/labels, docs/status output, the main reply and current-level resolution paths, relevant channel adapters, and regression tests. Treat #71817 as useful partial Telegram/Feishu per-agent work, not as the canonical global-default fix.
