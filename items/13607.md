## Summary

Keep #13607 open. Current main still supports `reasoningDefault` only as a per-agent `agents.list[]` setting, while the requested global `agents.defaults.reasoningDefault` is absent from the defaults type, strict defaults schema, generated schema/help metadata, docs, and runtime resolution path. The linked implementation PRs #24132 and #28704 were closed unmerged, and open PR #71817 is explicitly only a partial per-agent Telegram/Feishu fix, not the requested global default.

## Best Possible Solution

Keep #13607 open until OpenClaw either implements schema-backed `agents.defaults.reasoningDefault` with clear precedence, updates generated schema/help/labels, runtime resolution, status/directive tests, and public docs, or a maintainer explicitly decides reasoning visibility should remain per-agent only. Treat #71817 as a partial per-agent/channel fix, not as the canonical resolution for this global default request.
