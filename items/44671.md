## Summary

Keep open. Current `main` still rejects whitespace in Docker config/workspace/socket paths and in generated bind-mount specs, so the Windows path bug described by this PR is not implemented on `main`. The PR discussion also surfaced credible YAML-quoting regressions that would need follow-up before merging.
