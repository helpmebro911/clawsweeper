## Summary

Keep open. Current `main` does not implement the linked fix: doctor/status still report "No active memory plugin is registered" whenever no memory runtime is registered, and the bundled `memory-lancedb` plugin still does not register a memory runtime or unified memory capability runtime. The issue discussion's proposed fix in PR #65066 is therefore not yet reflected on the checked-out `main` or the latest shipped release `v2026.4.23`.
