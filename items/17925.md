## Summary

Keep open. Current main implements the Google/Gemini side as a bundled `gemini` web_search provider using Google Search grounding, but the requested ZAI/GLM native web_search passthrough remains missing. The related Zhipu/ZAI PRs cited in the issue were closed unmerged, and the maintainer comment already said partial search-provider work did not satisfy this broader request.
