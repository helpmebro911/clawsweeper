## Summary

Keep #39543 open. Current main has a PDF analysis tool that can summarize local paths, URLs, and managed media refs, but the requested local PDF upload workflow is still not implemented in the Control UI/Gateway chat attachment path: the UI only accepts image attachments and the Gateway intentionally drops non-image/PDF attachment payloads.
