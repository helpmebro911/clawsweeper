## Summary

Current `main` already addresses this report: the gateway pre-auth/CLI handshake timeout now defaults to 10 seconds, supports a runtime override via `OPENCLAW_HANDSHAKE_TIMEOUT_MS`, and the fix is documented as shipped in `v2026.3.22`.
