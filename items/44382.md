## Summary

Keep open. Current `main` separately fixed the light-terminal readability problem with a built-in light/dark theme heuristic, but it does not implement this PR's larger 5-theme system, `--theme` flag, or runtime `/theme` switching. The PR also bundles unrelated protocol/security changes, so there is no strong auto-close basis.
