## Summary

Keep open. Current `main` still appears to lack production wiring for Nextcloud Talk message reactions: `sendReactionNextcloudTalk` exists, but the bundled plugin only wires outbound text/media and defines no message-action handler for `react`.

## Evidence

- **Reaction helper exists but appears unused in production:** `sendReactionNextcloudTalk` is implemented, but a repo search only found references in its own test file, not in production code.
  - file: [extensions/nextcloud-talk/src/send.ts:192](https://github.com/openclaw/openclaw/blob/24e9924d6abd93560fb7bf721909257de533e57a/extensions/nextcloud-talk/src/send.ts#L192)
