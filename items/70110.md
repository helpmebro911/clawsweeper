## Summary

Current main still lacks the Nextcloud Talk message-action adapter needed to route shared `message` tool `react` actions to `sendReactionNextcloudTalk`. The linked PR #72348 is open and unmerged with closing syntax, so this issue should stay open until that PR or an equivalent fix lands.

## Best Possible Solution

Keep this issue open and review/land PR #72348, or an equivalent fix that adds a Nextcloud Talk `ChannelMessageActionAdapter`, registers it from `extensions/nextcloud-talk/src/channel.ts`, dispatches `react` to `sendReactionNextcloudTalk`, and adds regression coverage for discovery, dispatch, message-id fallback, and reaction request shape.
