## Summary

Keep #69030 open. Current main partially addresses the reported context leak by routing `infer model run` through the agent path with `modelRun: true`, `promptMode: "none"`, disabled tools, and bundled MCP cleanup, but it does not implement the requested `--raw`/raw model capability and still uses the agent/session runtime rather than a direct simple-completion path.

## Best Possible Solution

Keep the issue open and implement a true raw inference path for `openclaw infer model run`, either as `--raw` or a separate `model.raw` capability. The implementation should use the existing simple-completion helpers instead of `agentCommand`/gateway `agent`, avoid default-agent prompt/session/tool side effects, update capability metadata/CLI docs, and add tests that prove no agent prompt, tools, MCP runtime, or session transcript path is used in raw mode.
