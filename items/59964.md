## Summary

Keep open. Current `main` already covers the startup and hot-reload auto-recovery motivation via last-known-good restore, `.clobbered.*` preservation, notices, docs, and tests, but it does not implement the PR’s `openclaw config recover` CLI or the proposed `recoverConfigFile` export surface. The PR discussion also still contains unresolved correctness findings against the branch.
