## Summary

Keep open. Current main has related LaunchAgent PATH/environment hardening, but I did not find a source-backed fix that makes the LaunchAgent-managed gateway equivalent to a foreground gateway for LAN GitLab SSH operations or automatically supplies SSH_AUTH_SOCK/network context. The issue is recent, actionable, not protected by maintainer labels, and not maintainer-authored.
