## Summary

Keep open. Current `main` still rejects extra fields under `agents.defaults.models` and `applyModelDefaults()` only normalizes built-in model defaults plus aliases, so the requested passthrough behavior is not implemented. The PR discussion also identified a normalized-ref lookup gap in the proposed approach, which means there is not strong evidence for a conservative close.

## Evidence
