## Summary

Keep open. Current `main` still runs local `node-llama-cpp` embeddings in-process, still batches local embedding calls with `Promise.all`, and the shipped config/docs expose no OpenClaw-level CPU-only or Metal/GPU guidance for Apple Silicon. A fresh 2026-04-24 comment reports the same failure class and a `NODE_LLAMA_CPP_GPU=false` workaround, so this is not a strong close candidate.

## Evidence
