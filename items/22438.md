## Summary

Keep open. Current main partially addresses the original symptom by giving subagent/cron sessions a built-in middle bootstrap set (`AGENTS.md`, `TOOLS.md`, `SOUL.md`, `IDENTITY.md`, `USER.md`), but the requested configurable `bootstrapTier` surface is not implemented. The remaining request is still real, and docs/runtime are currently inconsistent enough that closing as implemented would be risky.
