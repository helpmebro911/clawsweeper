## Summary

Keep #22438 open. Current main has related partial context controls, including a fixed subagent/cron bootstrap allowlist and per-run `lightContext`, but it still does not implement the requested schema-backed `agents.defaults.bootstrapTier` option or a `minimal | standard | full` tier resolver. The supplied GitHub context also shows #22439 remains the active open implementation PR for this exact issue.

## Best Possible Solution

Keep #22438 open until #22439 or a successor lands, or until maintainers explicitly decide that fixed allowlists plus per-run `lightContext` are the intended design. A complete implementation should wire `agents.defaults.bootstrapTier` through config schema/types/help/labels/docs, workspace filtering, run resolution, hook semantics, subagent/cron behavior, and focused tests for `minimal | standard | full`.
