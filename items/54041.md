## Summary

Keep open. Current main still does not implement the requested Anthropic context-management beta passthrough/native support. Anthropic beta header handling exists via `anthropicBeta` and `context1m`, but there is no Anthropic `context_management` payload shaping, no compaction/context-editing replay or `pause_after_compaction` behavior, and no provider-defined `memory_20250818` tool path. Related #10213 and #48558 remain open and cover narrower slices, but they do not supersede this broader integration request.

## Best Possible Solution

Keep this issue open for maintainer/product design in the bundled Anthropic provider. A complete fix should add documented Anthropic model params for beta/header handling, validated `context_management` payload shaping for compaction and context editing, stream and replay support for compaction blocks and `pause_after_compaction`, coverage for both direct Anthropic transport and provider-wrapper paths, and a provider-owned design for `memory_20250818` or a generic provider-defined-tool SDK seam if that is required.
