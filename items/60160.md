## Summary

Keep open. Current `main` already ships a related subagent-context feature, but it is a different contract (`context: "isolated"|"fork"`) from this PR's proposed `includeContext: "none"|"summary"|"full"`, so this PR is not clearly implemented on `main`. The PR discussion also contains unresolved review findings, including a likely test-helper control-flow bug and a spawn-cleanup risk on transcript-read failures, so conservative cleanup should not auto-close it.
