## Summary

Keep #16387 open. Current main still treats model provider headers as static string values; I found no implementation or docs for runtime interpolation of `{{session.key}}`, `{{agent.id}}`, or `{{run.id}}`. The prior discussion is useful and concrete, and the closed PRs linked from the timeline appear unmerged rather than superseding the issue.
