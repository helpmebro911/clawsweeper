## Summary

Keep #8582 open. Current main implements the ElevenLabs realtime STT and provider-generic Voice Call streaming portion, but the issue bundled three asks and the duplicate-plugin warning control plus focused custom configSchema extension docs are still not implemented or split into canonical follow-ups.

## Best Possible Solution

Keep #8582 open unless a maintainer splits it. Best path is to add a tested per-id duplicate override or suppression acknowledgement in plugin config and manifest-registry diagnostics, document that behavior, and add focused plugin docs explaining that custom config keys require updating `openclaw.plugin.json` `configSchema` when extending or forking a bundled plugin. Alternatively, create canonical follow-up issues for those two remaining DX asks before closing this umbrella item.
