## Summary

Keep open. Current main still lacks the Open WebUI visible-history sync requested by #49205: `chat.send` updates OpenClaw transcript/broadcast state, but there is no Open WebUI DB mirror hook or sync helper, and existing Open WebUI coverage only verifies the normal OpenAI-compatible proxy path. The issue is recent, concrete, and distinct from the related session-key split issue #49202.
