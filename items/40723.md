## Summary

Keep this PR open. Current main still lacks the central user-facing change: a distinct Agents overview control that edits the global default primary model at `agents.defaults.model` separately from per-agent `agents.list[].model` overrides.

## Best Possible Solution

Keep this PR open for maintainer review or a rebased/fresh port. The concrete product path is to add an explicit global default primary-model selector in the Agents overview, wire it to `agents.defaults.model`, preserve fallback lists in canonical `fallbacks` form, keep per-agent model overrides unchanged, and include focused unit coverage for default-vs-override model state resolution.
