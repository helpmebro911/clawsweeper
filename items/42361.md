## Summary

Keep open. Current `main` still does not implement the PR's titled `web_fetch` network-error message improvement, and the branch bundles several unrelated fixes. Some of those side fixes have already landed independently on `main`, while others now conflict with current OpenClaw behavior/docs or with current package guardrails, so this PR is still useful but not mergeable as-is.
