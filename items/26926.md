## Summary

Keep open. Current main still only models and parses Synology Chat inbound webhooks as text messages; the existing media support is outbound URL-based delivery, which does not satisfy the issue's request for uploaded images, CSVs, and PDFs to reach the agent from Synology Chat.

## Evidence
