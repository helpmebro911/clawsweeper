## Summary

Keep this PR open. Current main still does not implement Slack message search: Slack discovery does not advertise `search`, Slack dispatch/runtime do not handle a search action, the Slack Web API wrapper has no `search.messages` helper, and the CLI/docs still present message search as Discord-only. The closed #58143 is prior unmerged work, while this PR is the active rebuild with review follow-up commits, so it is not obsolete or superseded.

## Best Possible Solution

Keep this PR open for maintainer review or a successor patch. The best path is to finish the bundled Slack plugin feature in `extensions/slack`: add a `search.messages` wrapper, map generic `search` through dispatch/runtime, advertise it only for runnable Slack accounts with a user token, preserve Discord search validation in the shared CLI, and add focused tests plus user-facing docs for Slack token/scope requirements.
