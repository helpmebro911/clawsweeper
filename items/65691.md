## Summary

Keep #65691 open. Current main still lacks Slack message search support: Slack message actions do not advertise `search`, Slack dispatch does not map `search` to a Slack runtime action, the Slack runtime has no `searchSlackMessages`/`searchMessages` path, and CLI/docs still describe message search as Discord-only. The PR is an active unmerged rebuild of closed PR #58143 and contains useful implementation work plus focused tests; review comments describe actionable fixups rather than evidence that the PR is obsolete.

## Best Possible Solution

Keep #65691 open for maintainer review. The best path is to finish the bundled Slack plugin implementation on this PR or a successor: add the `search.messages` wrapper and runtime/dispatch action, expose `search` only for runnable Slack accounts, preserve Discord search UX, and add focused tests/docs for Slack search behavior and token requirements.
