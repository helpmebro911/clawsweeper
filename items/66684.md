## Summary

Keep open. The report is recent and does not meet the security-policy bar for a complete vulnerability report, but one cited finding is still a real shipped pattern: current `main` and the latest release both bootstrap Bun with `curl ... | bash` in the Docker build path. The other cited hits are false positives or docs/changelog text, so there is not enough evidence to close as implemented, cannot_reproduce, incoherent, or stale.
