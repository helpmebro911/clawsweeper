## Summary

Keep open. Current main still uses static OpenRouter attribution, now as documented `X-OpenRouter-Title: OpenClaw` plus fixed referer/categories, and the request/header resolution path has no agent-aware input for formatting or overriding the title per agent. The related #50117 broader per-agent LLM header request is also still open, so it does not supersede this item. Ryce's cost-attribution/session-state comment remains relevant because current code does not preserve per-agent provider usage metadata as an alternate solution.
