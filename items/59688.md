## Summary

Keep PR #59688 open. Current main still has the strict resolved-realpath containment check without the PR's allowed-symlink-target-root escape valve, and managed skills are still loaded before the other known skill roots without passing any allowlist. The PR remains a plausible, narrowly scoped cleanup for noisy managed-skill symlink warnings.
