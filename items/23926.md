## Summary

Keep open. Current main has a partial plugin `before_install` hook for installer/security-scan paths, but the specific request remains meaningfully unmet: there is no `skill:pre-install`/`skill:post-install` internal hook event pair, no post-install observational hook, and ClawHub skill download/update paths still install archives directly without the scan/hook path that dependency installs use. Related PR #29426 was unmerged and, per the provided context, explicitly left hook enforcement as future work.
