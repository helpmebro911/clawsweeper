## Summary

Current main already bounds the exec approval modal and makes the long command area scrollable, so the reported 'buttons pushed off-screen by long command text' behavior appears implemented in shipped code.

## Evidence

- **Exec approval dialog layout is viewport-bounded:** The modal card uses a fixed max height and column layout, which prevents long content from expanding past the viewport.
