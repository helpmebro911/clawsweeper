## Summary

Keep #39341 open. Current main still appears to miss the requested top-level `channels.discord.allowFrom` check for Discord native slash-command allowlist auditing: the audit path only counts legacy `dm.allowFrom` plus the credential-store allowFrom. The linked PR #39387 was closed unmerged, and the latest release tag inspected still has the same behavior.
