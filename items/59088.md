## Summary

Current `main` already has a Control UI Agents dashboard with an inline editor for core agent files, but it is explicitly limited to allowlisted bootstrap files plus `MEMORY.md`. The issue asks for a broader workspace browser/editor (`memory/*.md`, file tree, arbitrary text-ish files), so this is only partially implemented and should stay open unless the request is narrowed to core files only.
