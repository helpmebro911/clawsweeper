## Summary

Current main still does not implement #7909. The Control UI assistant-message action remains a single “Copy as markdown” button that writes the current markdown string to the clipboard; there is no plain-text option, dropdown, preference, or test/docs hook for plain-text message copy. The latest issue discussion also includes a maintainer Codex review saying to keep this open.

## Best Possible Solution

Keep #7909 open until the Control UI adds a real plain-text copy path for assistant messages. The likely implementation belongs in `ui/src/ui/chat/copy-as-markdown.ts` and `ui/src/ui/chat/grouped-render.ts`: expose a dropdown or adjacent action with “Copy as Markdown” and “Copy as Plain Text”, add a tested Markdown-to-plain-text conversion helper that strips formatting while preserving readable code/list/link text, and cover the new behavior with focused UI/helper tests.
