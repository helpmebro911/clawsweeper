## Summary

Keep open. Current `main` still routes `logs.tail` output through the shared regex redactor, but the default redaction patterns do not cover the reported generic JWT, `Authorization: Basic`, `X-OpenClaw-Token`, or `x-pomerium-jwt-assertion` cases, and the existing tests only cover narrower token formats. The issue is also recent and already has an open follow-up PR in the discussion.
