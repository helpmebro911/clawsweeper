## Summary

Current `main` has avatar serving and string-based identity updates, but it does not implement the reported feature: there is no Agents-tab avatar upload UI and no dedicated upload RPC that saves a selected image into the agent workspace and updates `IDENTITY.md`.

## Evidence

- **Agents overview has no avatar picker or upload control:** The Agents overview renders workspace/model fields and model save controls only. There is no avatar preview, clickable avatar, file input, or upload handler in the Agents tab.
