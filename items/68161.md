## Summary

Keep this PR open. Current main still has the stale optional `hello-ok.auth` schema and matching optional Swift artifacts, while the runtime handler now always emits `auth`; this PR remains a valid implementation candidate for the open same-author bug #68160. The related PR #68227 is closed unmerged, so it does not supersede this PR.

Keep this PR open for maintainer review. The best path is to land one complete implementation that makes `hello-ok.auth` required in the TypeBox schema, regenerates both Swift protocol models, updates Swift consumers/tests, and then closes #68160 after merge; maintainers can optionally borrow any useful docs or focused protocol test coverage from the closed unmerged #68227.
