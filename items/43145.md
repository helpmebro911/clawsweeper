Keep this PR open. Current main still lacks the retained run-trace timeline store and `agent.timeline` gateway RPC that are the central feature. Existing trajectory bundles are related but not equivalent, and #68204 is the right design target but not enough to auto-close this implementation candidate.

## Best Possible Solution

Keep this PR open for explicit maintainer handling, but require it to be split or substantially reduced. Drop or separate already-shipped sub-fixes and unrelated autonomy/sandbox/cron/planner work, then rework the remaining trace feature against #68204 with a canonical schema, safe gateway/API exposure, ownership checks, redaction, retention/span caps, generated client updates, docs, and focused tests.
