## Summary

Current main does not implement #24118. The Signal channel still lacks a `channels.signal.linkPreview` schema/config surface, and the Signal send path does not pass `previewUrl`/preview metadata or call `updateConfiguration`. The related #29466 context is webchat UI preview work and does not supersede the Signal JSON-RPC request.
