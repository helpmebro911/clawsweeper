## Summary

Keep open. Current main still does not show a standalone `openclaw nodes audio record` / `audio.*` / `microphone.*` node command surface; the available surfaces are camera clip and screen recording with optional audio, plus generic `nodes invoke`. That makes this a plausible feature request or docs/expectation mismatch, not a conservative close.
