## Summary

Keep open. Current `main` still has the `isTextBlock` bug described in the PR, so this is not implemented or obsolete. The PR is also not merge-ready as submitted because it bundles unrelated changes, including a Feishu key-validation relaxation that would weaken current slash blocking and a `package.json` override removal without the matching `pnpm-lock.yaml` update required by frozen-lockfile CI.
