- The PR is size XL and touches core agent tools, A2A behavior, gateway lane policy, tests, and docs; it needs maintainer product/API review rather than automated cleanup.
- The PR's global `CommandLane.Nested` widening conflicts with current main's more conservative per-session nested-lane direction and could reintroduce shared-lane behavior if merged as-is.
