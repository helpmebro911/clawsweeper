## Summary

Keep open. Current main still has the same explicit `host=node` cwd approval path that can return the reported `SYSTEM_RUN_DENIED: approval requires an existing canonical cwd` before execution, and I do not see a current-main macOS launchd/APFS firmlink normalization or fallback that would make this safe to auto-close.
