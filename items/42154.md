## Summary

Current `main` has a newer sandbox-specific fix path (`tools.sandbox.tools.alsoAllow`) with docs/tests/changelog support, but it still keeps `tools.alsoAllow` and the sandbox step separate. That means PR #42154's exact behavior is not implemented on `main`, and there is not strong enough evidence for an automatic close reason.
