## Summary

Keep this PR open. Current main still does not include the proposed launcher-backed npm bin, local stable tarball upgrade script, live memory-pro host smoke, or launcher e2e coverage. The PR is also not conservative cleanup material because it changes the published npm executable path and adds global-install/live-execution scripts while unresolved review comments identify release-preflight, entry-bootstrap, gateway-detection, and regression-script reliability issues.

## Best Possible Solution

Keep this PR open for maintainer review rather than closing it as obsolete.
