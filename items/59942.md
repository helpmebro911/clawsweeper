## Summary

Current `main` still contains the exact fallback paths behind the report: `status` shows memory as enabled-but-unavailable when the selected memory slot yields no snapshot, and `doctor` still reports no active memory plugin when memory runtime resolution returns null. With no follow-up discussion disproving the report and no fix visible on `main`, this is not safe to close.
