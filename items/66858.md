## Summary

Keep open. Current `main` still contains one of the reported child behaviors explicitly (`missing-session-entry` subagent recovery is still treated as an orphan and pruned on resume), and the Codex harness docs still state that native compaction only exposes start/completion signals without a readable summary or kept-entry audit.
