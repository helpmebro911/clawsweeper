## Summary

Keep open. Current `main` does not contain this macOS watchdog runbook, and the remaining review concerns are real against current behavior: `openclaw health` treats gateway reachability as success even when channel issues exist, and the CLI already recognizes profile/config-path mismatches as a real service-targeting problem. That means this docs-only PR still needs revision rather than conservative auto-close.
