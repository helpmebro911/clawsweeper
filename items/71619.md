## Summary

Keep this PR open. Current main still includes Linux nvm/fnm/Volta/asdf/Bun paths in the daemon minimal service PATH, and the current tests/docs still describe or assert that behavior, so the PR remains a live implementation candidate for the #71578 PATH audit contradiction.

## Best Possible Solution

Keep this PR open for maintainer review or land an equivalent patch: make Linux minimal service PATH include stable user bins, PNPM/NPM prefix, Nix profiles, and system dirs while excluding Linux version-manager directories; preserve macOS version-manager behavior; update daemon service-env, service-audit, and public docs for the #71578 hardened-PATH expectation.
