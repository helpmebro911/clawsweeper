## Summary

Keep open. This PR is a focused Feishu bug fix that still covers behavior missing on current main: first-turn topic sessions without root_id do not persist a usable MessageThreadId, and several outbound threadId fallback paths still omit replyInThread. The author also addressed bot review feedback by extending the media/local-image paths and adding regression coverage.

## Best Possible Solution

Keep this PR open as a plausible implementation candidate. The best path is to rebase it onto current main, reconcile any overlap with narrower Feishu threading PRs such as #67069, then review and run the targeted Feishu extension tests plus the changed gate before landing or requesting changes.
