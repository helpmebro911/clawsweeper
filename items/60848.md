## Summary

Keep this PR open. Current main still lacks the Feishu topic-thread MessageThreadId fallback and still omits replyInThread propagation on several outbound threadId-fallback paths; related #67069 only overlaps the inbound metadata area.

## Best Possible Solution

Keep this PR open as a plausible Feishu bug-fix candidate.
