## Summary

Keep open. Current main still has the `tools.sessions.visibility` default of `tree` and `sessions_send` still returns the same cross-agent visibility error called out in the issue. The config docs/changelog now mention the default, but I did not find source evidence for the requested migration warning, doctor/startup validation, or richer OC-07-specific runtime error. Related #18967 is a different open feature request, while #19564 and #20939 do not supersede this migration/UX report.
