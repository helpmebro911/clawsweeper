## Summary

Current `main` already contains loopback Control UI auto-approval logic for token/password-auth browser connections and has regression tests covering the first-connect success path plus later scope-upgrade pairing. The implementation appears to have landed after the reported 2026.3.13 release, so the issue is fixed on `main` but not attributable to a released tag from the evidence available here.
