## Summary

Keep open. Current `main` intentionally keeps the Gateway on loopback by default and treats `0.0.0.0`/LAN exposure as an explicit, security-sensitive opt-in, so this is not an implemented or obsolete cleanup candidate. It reads as a live product/security request, even if the current labels probably fit a feature/security-defaults discussion better than a reproducible regression.
