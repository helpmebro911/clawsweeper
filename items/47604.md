## Summary

Current main still does not contain the Wear OS companion app, phone-side Wear proxy, shared Android gateway module, Wear Data Layer dependency, tile/complication services, or Wear documentation proposed here. The PR is large and contributor-abandoned, but it is concrete platform feature work, the latest head is still published, and a maintainer removed the stale label and added `no-stale`, so there is no conservative cleanup close reason.

## Best Possible Solution

Keep this PR open unless maintainers make an explicit product decision not to ship Wear OS in core. The practical path is to turn it into maintainer-owned Wear OS work and split it into reviewable stages: shared Android gateway helpers, phone-side Wear proxy, standalone Wear app module, then tile/complication/notification polish with focused tests and fresh device proof.
