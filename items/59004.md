## Summary

Keep open. Current `main` still classifies `camera.snap` and `camera.clip` as dangerous node commands, so the gateway filters them out by default on macOS even though the macOS node app implements and advertises them when camera is enabled. The report matches current code, and the docs still read as if those commands should just be available, so this is not a safe cleanup close.
