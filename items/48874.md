## Summary

Keep #48874 open. Current main has partial session architecture work, including canonicalized session keys, `sessions.create`, compaction checkpoint branch/restore, and a memory plugin/search layer. The RFC’s broader archive/restore/pin session state model, archived-session API/listing, and Control UI archive/restore workflow are still not implemented, and no single canonical superseding item covers the remaining work.
