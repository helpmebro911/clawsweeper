## Summary

Keep open. Current main still lacks the PR's central Google Vertex implementation: no dedicated `google-vertex` provider registration, no Vertex project/location request URL builder, and no ADC-to-Bearer token exchange. The PR is also paired with open same-author #60736 and open regression #58775, so it remains an active implementation candidate rather than cleanup-close material.

## Best Possible Solution

Keep this PR open for normal maintainer review, or replace it with a maintainer-owned implementation that registers `google-vertex` as its own bundled Google plugin provider, exchanges ADC credentials for Bearer auth, builds Vertex project/location URLs, preserves existing Gemini model normalization and thought-signature replay, and adds focused Google plugin tests plus docs for the supported Vertex setup path. After merge, let the linked Google Vertex issues close through the implementation.
