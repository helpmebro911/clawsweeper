## Summary

Keep open. Current main does not ship Matrix or Cyberpunk as first-class Control UI themes, and the related Matrix PR was closed unmerged. The existing custom theme slot can import safe tweakcn token palettes, but it is browser-local and does not cover the requested built-in selector entries, digital-rain/glitch/scanline animations, or a plugin/theme extension surface for these themes.

## Best Possible Solution

Leave this open for maintainer product/design judgment. A complete core implementation would add Matrix and Cyberpunk as first-class Control UI theme families in ui/src/ui/theme.ts, ui/src/styles/base.css, the full and quick settings selectors, and the theme tests. If maintainers prefer external distribution, the better path is to define or document a supported custom-theme/plugin surface that can cover more than safe color tokens before closing this as external/ClawHub work.
