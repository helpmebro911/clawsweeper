## Summary

Keep open. Current main still has only the built-in Claw, Knot, and Dash theme families plus one browser-local tweakcn custom token slot; Matrix and Cyberpunk are not implemented or externally registerable as first-class Control UI themes. The linked Matrix PR is closed unmerged in the provided context, and Cyberpunk only appears as fork work, not shipped main behavior.

## Best Possible Solution

Leave this issue open for maintainer product/design judgment. If accepted as core UI work, implement Matrix and Cyberpunk as first-class Control UI theme families across ui/src/ui/theme.ts, ui/index.html early theme bootstrapping, base theme CSS, full and quick settings selectors, and focused accessibility/reduced-motion tests. If maintainers prefer external distribution, first add and document a supported Control UI theme-pack/plugin registration API; the current browser-local tweakcn custom slot is too narrow for the requested animated theme families.
