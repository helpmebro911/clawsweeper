## Summary

Keep this open. Current main still does not ship Matrix or Cyberpunk as first-class Control UI themes, the linked Matrix PR was closed unmerged, and the existing custom theme path is a bounded tweakcn token import rather than a supported animated theme or theme-plugin surface.

## Best Possible Solution

Leave this issue open for maintainer product/design judgment. If accepted as core UI work, add Matrix and Cyberpunk as first-class Control UI theme families across the theme model, base CSS, full and quick settings selectors, docs if user-facing, and focused tests for accessibility and prefers-reduced-motion behavior. If maintainers prefer external distribution, first define or document a supported custom-theme/plugin surface that safely covers more than bounded color/font tokens, including animation/effect policy.
