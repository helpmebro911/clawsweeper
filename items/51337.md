## Summary

Current `main` at `608c08fc54da2f86338366f8eeafd7e2a02713aa` does not implement this PR's hook `sessionTarget` resume path. The hook gateway still hardcodes hook-triggered agent jobs to `sessionTarget: "isolated"`, and the hook config/API/docs/tests on `main` only support `sessionKey`, not `sessionTarget`. This PR still represents unmerged behavior, so it should stay open.
