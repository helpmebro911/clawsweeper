## Summary

Keep open. Current `main` still resolves execution models in `src/agents/agent-command.ts` from the agent default plus stored/explicit overrides only, while the channel-specific `modelByChannel` resolver is wired into `/status` display code instead of the runtime selection path. The issue is recent, specific, and supported by the current code.

## Evidence

- **Runtime model selection still skips channel overrides:** `agentCommandInternal` starts from `resolveDefaultModelForAgent`, then applies stored session overrides and explicit run overrides. In the inspected selection block there is no call to `resolveChannelModelOverride`, so channel-pinned models are not part of the execution priority chain.
  - file: [src/agents/agent-command.ts:652](https://github.com/openclaw/openclaw/blob/a155b84cdadef068de50274318952032cd867788/src/agents/agent-command.ts#L652)
  - command: `nl -ba src/agents/agent-command.ts | sed -n '648,735p'`
  - sha: [a155b84cdade](https://github.com/openclaw/openclaw/commit/a155b84cdadef068de50274318952032cd867788)
- **Status path does use channel override resolution:** `/status` explicitly calls `resolveChannelModelOverride` using the session channel/group metadata and only then labels the chosen model as a `channel override`, matching the reporter's claim that status and runtime diverge.
  - file: [src/status/status-message.ts:816](https://github.com/openclaw/openclaw/blob/a155b84cdadef068de50274318952032cd867788/src/status/status-message.ts#L816)
  - command: `nl -ba src/status/status-message.ts | sed -n '808,845p'`
  - sha: [a155b84cdade](https://github.com/openclaw/openclaw/commit/a155b84cdadef068de50274318952032cd867788)
- **Channel override helper exists on main:** `resolveChannelModelOverride` is implemented in core and supports direct and fallback matching using `channel`, `groupId`, and `parentSessionKey`, so the missing behavior is not due to absent infrastructure.
  - file: [src/channels/model-overrides.ts:157](https://github.com/openclaw/openclaw/blob/a155b84cdadef068de50274318952032cd867788/src/channels/model-overrides.ts#L157)

  - file: [src/auto-reply/reply/session.ts:530](https://github.com/openclaw/openclaw/blob/a155b84cdadef068de50274318952032cd867788/src/auto-reply/reply/session.ts#L530)
