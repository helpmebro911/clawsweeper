## Summary

Keep open. Current `main` still hardcodes `google-generative-ai` to tagged reasoning output, still injects the `<think>/<final>` prompt text that says only `<final>` text is shown, and the Google transport still only turns native `functionCall` parts into tool invocations. I found no main-branch evidence of a fix, so the report remains plausible and current rather than cleanup-safe to close.
