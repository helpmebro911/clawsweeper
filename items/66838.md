## Summary

Keep open. Current `main` still uses the old cron-event wrapper that tells the model to relay reminders to the user, and the current tests still assert that behavior, so this PR is neither implemented on `main` nor obsolete. The existing heartbeat ack normalization also makes the PR’s token-handling discussion materially relevant rather than stale.
