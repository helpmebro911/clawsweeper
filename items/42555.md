## Summary

Keep open. Current `main` still hardcodes Slack assistant thread status to `"is typing..."`, the global agent identity config/schema does not expose `typingStatuses`, and docs/tests still describe/assert the fixed default. The submitted patch also targets pre-plugin-move Slack paths, so it likely needs rework for current `extensions/slack` architecture rather than auto-close.
