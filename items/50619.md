## Summary

Keep open. Current main has useful post-switch safeguards, including effective context-token budgeting and pre-prompt overflow compaction, but the requested `/model` behavior is still not implemented: the slash command applies the model switch immediately, and the gateway patch path validates/stores the model override without comparing current session size to the target model budget or warning about rate-limit/TPM risk. The related #58957 tracks a broader carried-over-context failure mode, but it does not fully supersede this specific pre-switch warning/auto-compact request.

## Evidence
