## Summary

Keep this PR open. Current main still has the short bundled skill descriptions for all seven targeted skills, and those descriptions are active model-facing metadata used for skill selection. The PR is a focused edit to existing bundled skills, so VISION.md's ClawHub-first guidance for new skills does not cleanly close it, and the broader open umbrella issue #43410 does not supersede this concrete implementation candidate.

## Best Possible Solution

Keep this PR open for maintainer review or a current-main refresh. The best path is to preserve the useful trigger-phrase and NOT FOR improvements for the seven existing bundled skills, copyedit inaccurate cross-skill references such as Telegram, and link it as a concrete partial implementation of #43410 rather than closing it as shipped, obsolete, or ClawHub-only work.
