## Summary

Keep open. This is a recent, plausible feature request for a core gateway heartbeat behavior. Current main documents and types support interval, active-hours, delivery, light-context, isolated-session, and busy-queue skipping, but I found no user-facing idle-only or trigger-when-idle option equivalent to the requested behavior.
