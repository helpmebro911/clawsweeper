## Summary

Keep open. Current main only surfaces model fallback through lifecycle events, verbose-mode inline notices, and Control UI indicators; it does not provide the requested built-in, configurable channel/DM/system announcement feature.

## Evidence

- **Fallback events exist, but chat notice is gated by verbose mode:** `runReplyAgent` always emits `lifecycle` `fallback` / `fallback_cleared` events, but only appends a user-visible fallback notice when `verboseEnabled` is true.
  - file: [src/auto-reply/reply/agent-runner.ts:1493](https://github.com/openclaw/openclaw/blob/c6684af682bcb59bd1ee85cb49f72d4074007e2c/src/auto-reply/reply/agent-runner.ts#L1493)
  - sha: [c6684af682bc](https://github.com/openclaw/openclaw/commit/c6684af682bcb59bd1ee85cb49f72d4074007e2c)
- **Test locks current behavior to verbose-only announcements:** The e2e test asserts `Model Fallback:` appears when verbose is `on`, and does not appear when verbose is `off`, even though a fallback lifecycle event still fires.
  - file: [src/auto-reply/reply/agent-runner.runreplyagent.e2e.test.ts:622](https://github.com/openclaw/openclaw/blob/c6684af682bcb59bd1ee85cb49f72d4074007e2c/src/auto-reply/reply/agent-runner.runreplyagent.e2e.test.ts#L622)
  - sha: [c6684af682bc](https://github.com/openclaw/openclaw/commit/c6684af682bcb59bd1ee85cb49f72d4074007e2c)
- **Current UX is a transient Control UI indicator, not a channel message feature:** The UI renders fallback status as a temporary indicator (`Fallback active` / `Fallback cleared`) from agent events; this is separate from sending a message into Discord/Slack/etc.
  - file: [ui/src/ui/chat/status-indicators.ts:40](https://github.com/openclaw/openclaw/blob/c6684af682bcb59bd1ee85cb49f72d4074007e2c/ui/src/ui/chat/status-indicators.ts#L40)
  - sha: [c6684af682bc](https://github.com/openclaw/openclaw/commit/c6684af682bcb59bd1ee85cb49f72d4074007e2c)
- **No built-in config knob matching the proposal is present:** A repo search for `fallbackAnnounce` / `throttleSeconds` returned no matches, so current main does not expose the proposed configurable announcement settings.
  - command: `rg -n "fallbackAnnounce|throttleSeconds" src docs ui extensions packages`
  - sha: [c6684af682bc](https://github.com/openclaw/openclaw/commit/c6684af682bcb59bd1ee85cb49f72d4074007e2c)
