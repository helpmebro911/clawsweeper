## Summary

Keep open. Current `main` does not implement this UI behavior: long markdown is still hard-truncated at 140k chars, markdown parsing still falls back at 40k chars, JSON auto-parse is still capped at 20k chars, and chat messages still render directly as `.chat-text` without any progressive-disclosure wrapper/button. This PR is therefore not implemented on `main`, not obsolete, and not safe to close under the allowed reasons.
