## Summary

Keep #39461 open. Current main still does not read `QQBOT_DATA_DIR` for QQ Bot data storage: `getQQBotDataPath()` hard-codes `<home>/.openclaw/qqbot`, and diagnostics still tells Windows users to set `QQBOT_DATA_DIR` even though that variable is not implemented.
