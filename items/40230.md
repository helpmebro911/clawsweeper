## Summary

Current `main` does not implement PR #40230. `src/agents/workspace.ts` still reads bootstrap files only through `openBoundaryFile(...)` rooted at the workspace and immediately returns failures, so a workspace `AGENTS.md` symlink resolving outside the workspace root would still be treated as missing rather than loaded. The checkout also lacks the PR’s symlink-specific tests; current tests cover hardlink rejection and extra-file traversal exclusion, not the requested symlink-following behavior.
