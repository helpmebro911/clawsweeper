## Summary

Keep open. Current `main` still does not expose this plugin runtime/session-history API, and the PR itself now documents unresolved transcript-contract limitations that match real complexity in OpenClaw's current branch-aware transcript and session-file resolution behavior. This is not conservative maintainer-cleanup material.
