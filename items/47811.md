## Summary

Keep open. Current main has useful improvements for multi-gateway/status visibility, but the issue's profile-normalization/casing footgun is still present: custom profile names are accepted with uppercase characters and then preserved into state directories and LaunchAgent labels, so `--profile Eagle` and `--profile eagle` still target different homes/services instead of normalizing or failing with an explicit casing error.

## Evidence

- **Profile parser preserves custom profile casing:** `applyCliProfileEnv` trims the supplied profile, stores it unchanged in `OPENCLAW_PROFILE`, derives `OPENCLAW_STATE_DIR` from that exact value, and sets `OPENCLAW_CONFIG_PATH` under that state dir.
  - file: [src/cli/profile.ts:79](https://github.com/openclaw/openclaw/blob/d228463120a5057f7ccf9a5c31af62fe4a922088/src/cli/profile.ts#L79)
  - command: `nl -ba src/cli/profile.ts | sed -n '63,100p'`
  - sha: [d228463120a5](https://github.com/openclaw/openclaw/commit/d228463120a5057f7ccf9a5c31af62fe4a922088)
- **Uppercase profile names are valid and returned unchanged:** The profile-name regex is case-insensitive, and `normalizeProfileName` returns the trimmed custom profile rather than lowercasing it.
  - file: [src/cli/profile-utils.ts:3](https://github.com/openclaw/openclaw/blob/d228463120a5057f7ccf9a5c31af62fe4a922088/src/cli/profile-utils.ts#L3)
  - command: `nl -ba src/cli/profile-utils.ts | sed -n '1,120p'`
  - sha: [d228463120a5](https://github.com/openclaw/openclaw/commit/d228463120a5057f7ccf9a5c31af62fe4a922088)
- **LaunchAgent labels preserve profile casing:** Gateway profile normalization treats only `default` case-insensitively; other profiles are returned unchanged and used in `ai.openclaw.<profile>` labels. That leaves `Eagle` and `eagle` as distinct service targets.
  - file: [src/daemon/constants.ts:17](https://github.com/openclaw/openclaw/blob/d228463120a5057f7ccf9a5c31af62fe4a922088/src/daemon/constants.ts#L17)

  - file: [src/cli/daemon-cli/status.gather.ts:197](https://github.com/openclaw/openclaw/blob/d228463120a5057f7ccf9a5c31af62fe4a922088/src/cli/daemon-cli/status.gather.ts#L197)
