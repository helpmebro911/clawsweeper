## Summary

Keep open: current `main` still has explicit preflight-compaction dead zones for heartbeat turns and for sessions with fresh persisted token counts, matching the later issue investigation. There is better overflow messaging now, but not enough evidence that the model-switch/carried-context failure is resolved end-to-end.
