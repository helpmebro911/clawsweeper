## Summary

Current main does not implement this PR’s migration. The active bundled Anthropic plugin still adds the legacy `context-1m-2025-08-07` beta for `context1m: true`, still strips that beta for OAuth tokens, and tests/docs still assert or describe the pre-GA behavior. The PR targets an older core path, so the work would need to be rebased onto the current Anthropic plugin rather than closed as implemented or obsolete.

## Best Possible Solution

Keep the PR open or ask the contributor/maintainer to rebase the GA migration onto the current Anthropic plugin: update `extensions/anthropic/stream-wrappers.ts`, plugin tests, docs, and any native Anthropic transport path that still depends on legacy beta/OAuth assumptions. The canonical issue #45550 should remain the tracker for the broader migration until current main actually stops sending and documenting the legacy beta header.
