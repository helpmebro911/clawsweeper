## Summary

Keep open. Current `main` still sends `memory-lancedb` embedding requests for `baseUrl` providers through the OpenAI SDK path, and the existing regression coverage only checks that `dimensions` are passed into the SDK call, not that custom `baseUrl` providers preserve those dimensions end-to-end. The PR appears to address a real gap rather than a duplicate or already-landed fix.

## Evidence
