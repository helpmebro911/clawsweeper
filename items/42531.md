## Summary

Keep open. Current main does not contain this PR's baseUrl-specific fix: `memory-lancedb` still sends embeddings through the OpenAI SDK even when `embedding.baseUrl` is configured, and the current tests only cover forwarding `dimensions` into the SDK call, not preserving dimensions on the custom-`baseUrl` path.
