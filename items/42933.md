## Summary

Keep open. Current `main` does not implement the proposed session-level circuit breaker: there is no `circuitBreaker` config surface, no persisted `cb*` session state, and the shipped failure mitigation remains auth-profile cooldown plus heartbeat `isolatedSession`. The PR discussion also still contains unresolved behavioral concerns, so conservative cleanup should not close it.
