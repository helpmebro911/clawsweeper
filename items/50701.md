## Summary

Keep open. Current main has partial transcript-backed usage work: reset/deleted archives are counted and discoverable. But the central request remains missing: no logical-session lineage/family API, no groupBy/includeHistorical contract, no per-family rollup fields, and no WebUI current-instance vs historical-lineage toggle or 90d/1y/all presets. The KeepMyClaw comment is a useful durability workaround, but it does not implement OpenClaw’s in-product WebUI lineage rollups.
