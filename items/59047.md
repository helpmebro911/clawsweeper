## Summary

Keep open. Current main appears to have addressed the external plugin SDK alias-resolution/load half of the report, but the local `agent`/embedded TUI path still does not thread `allowGatewaySubagentBinding` into `runEmbeddedPiAgent`, while plugin tool registry reuse is still gated on that flag. That matches the reporter’s follow-up analysis closely enough that there is not strong evidence to close this conservatively.
