## Summary

Keep #48420 open. Current main appears to have fixed the main Anthropic streaming transport by sending `sk-ant-oat*` credentials as `Authorization: Bearer`, and OpenAI Codex OAuth is now a separate supported route. However, the issue discussion specifically called out direct Anthropic HTTP callsites, and current main still has the native PDF Anthropic path sending the resolved credential as `x-api-key` with no OAuth/Bearer branch. That is a meaningful remaining part of the reported regression surface, so this should not be conservatively closed yet.
