The linked bug #64453 was closed as superseded by this PR, and the earlier same-scope PR #65569 is closed unmerged, leaving this PR as the active implementation candidate.

The best path is to land a refined memory-wiki fix that adds normalized token-overlap matching across digest and full-scan paths with focused regression tests, while addressing or explicitly accepting the digest fallback and minimum-threshold review concerns.
