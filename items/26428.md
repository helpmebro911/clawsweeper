## Summary

Current main does not implement the requested fish completion install behavior. The CLI still treats fish like profile-based shells by writing a source block into the fish config file, and docs still describe profile installation. The linked PR #27135 directly addressed this request but the provided GitHub context shows it was closed unmerged, so it does not supersede the open issue.
