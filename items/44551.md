## Summary

Keep open. Current `main` already has `zh-CN` locale support and auto-selects it from the browser language, so much of the Gateway UI is localized. But the request is not fully implemented: tool/audit-style labels such as `Read`, `Browser`, and `Exec` still come from shared English-only display metadata, and the Logs view still hardcodes multiple English strings.

## Evidence
