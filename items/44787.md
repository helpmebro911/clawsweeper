## Summary

Current `main` already contains the memory-flush fixes that match this report: silent pre-compaction flush turns are suppressed from external replies, and daily memory-flush writes are append-only to the canonical `memory/YYYY-MM-DD.md` file. The report was against `2026.3.1`; these fixes are documented in the `2026.4.2` changelog.

Closing from a Codex review because current `main` already implements the fix for this report. This was shipped after the reported `2026.3.1` version; the stable `2026.4.2` changelog includes both the append-only memory-flush fix and the silent-turn fix (`CHANGELOG.md:1456`, `CHANGELOG.md:1477`).
