## Summary

Keep open. Current main still preserves the reported Discord voice join abort path, the same-author focused Discord voice report remains open, and this issue is still the canonical 2026.4.8 gateway CPU/RPC-stall umbrella after the lower-detail CPU duplicate was closed in favor of it. The agent CLI wrapper timeout has improved, and post-release gateway startup/liveness mitigations exist, but those are partial and do not prove the gateway CPU/RPC stall, Discord voice, or Messenger bridge symptoms are fixed.
