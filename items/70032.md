## Summary

Keep open. Current `main` has generic typing-indicator infrastructure, but the bundled Nextcloud Talk channel still does not wire it up, expose a typing config option, or document typing support. This is an unimplemented enhancement on an in-repo bundled channel plugin, not a ClawHub-only item and not something already fixed on `main`.
