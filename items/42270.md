## Summary

Keep open. Current `main` has related fixes for network-interface discovery and OpenAI-compatible local-provider handling, but I did not find strong evidence that issue #42270's reported regression is fully resolved. The code still has uncaught network-interface enumeration paths in gateway logic, so the host-network/Tailscale angle is not clearly eliminated, and no changelog or code reference ties a shipped fix directly to this issue.
