## Summary

Keep #39141 open. Current main does not implement the requested session-scoped activity watchdog/alerts, and its stated prerequisite (#39127, sessions.activity.get/list plus WS statechange events) remains open in the provided GitHub context. This also should not be closed as ClawHub yet because the missing core/session activity API is exactly the blocker a plugin would need.
