## Summary

Keep open. Current `main` has already absorbed some of this PR's ideas through different implementations, but it does not implement the full requested behavior: Google Chat processing still ignores `ADDED_TO_SPACE`, and Google Chat reply threading still defaults to `off` unless configured.
