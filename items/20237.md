## Summary

Keep #20237 open. Current main still only partially covers the request: Control UI has cron edit/run/history/delete-after-run, browser Web Push, and an inline context usage notice with Compact, but it still lacks the central native in-app toast/popup system, cron run-result popups with actions, and snooze/reschedule controls. The existing maintainer discussion also explicitly kept this issue open for the same remaining work.

## Best Possible Solution

Keep this open until the missing Control UI notification work is implemented or explicitly descoped. The likely path is a native in-app toast/popup layer, cron run-completion events wired into actionable result notifications, snooze/reschedule controls backed by cron.update schedule patches or a dedicated validated RPC, and a product decision on whether context warnings should remain inline only or also produce toast-style Flush/Save or Compact prompts.
