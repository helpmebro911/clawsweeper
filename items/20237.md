## Summary

Keep #20237 open. Current main still only partially covers the request: cron edit/run/history/delete-after-run, browser Web Push, and an inline context usage notice with Compact are present, but the central native in-app toast/popup notification system, cron run-result popups with actions, and snooze/reschedule controls remain missing. The existing maintainer-context Codex review comment on the issue also explicitly kept it open for the same remaining work.

## Best Possible Solution

Keep this open until the missing Control UI notification work is implemented or explicitly descoped. The likely path is a native in-app toast/popup layer, cron run-completion events wired into actionable result notifications, snooze/reschedule controls backed by cron.update or a dedicated validated RPC, and a product decision on whether context warnings should remain inline only or also produce toast-style Flush/Save prompts alongside existing Compact behavior.
