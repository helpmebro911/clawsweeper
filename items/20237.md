## Summary

Keep #20237 open. Current main still only partially covers the request: cron edit/run/history/delete-after-run, browser Web Push, and an inline context notice exist, but the requested native in-app toast/popup layer, actionable cron run-result popups, and snooze/reschedule controls are still missing. The issue also has a recent maintainer comment explicitly keeping it open for those gaps.

## Best Possible Solution

Keep this open until the missing Control UI notification work is implemented or explicitly descoped. A good path is a native in-app toast/popup layer, cron run-completion events wired into actionable result notifications, snooze/reschedule controls backed by validated cron.update schedule patches or a dedicated RPC, and a product decision on whether context warnings should remain inline-only or also trigger toast-style Compact/Flush prompts.
