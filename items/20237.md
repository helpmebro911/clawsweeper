## Summary

Keep #20237 open. Current main only partially covers the request: Control UI has cron edit/run history/delete-after-run, browser Web Push, Gateway cron finished events, and an inline high-context notice with Compact. The central requested work is still missing: a native in-app toast/popup layer, actionable cron result popups, and snooze/reschedule controls. The existing 2026-04-26 Codex review comment also explicitly kept this open for the same gaps, and the relevant files have not changed since that review.

## Best Possible Solution

Keep #20237 open until the missing Control UI notification work is implemented or explicitly descoped. A good implementation path is a native in-app toast/popup layer, cron finished events wired into actionable result notifications, snooze/reschedule controls backed by validated cron.update schedule patches or a dedicated RPC, and a product decision on whether context warnings should remain inline-only or also trigger toast-style Compact/Flush prompts.
