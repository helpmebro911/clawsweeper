## Summary

Keep open. Current `main` and the shipped `v2026.4.23` release still run `check:docs` through `git ... | xargs oxfmt`, so the PowerShell problem from #44293 is not implemented on main. The PR direction is useful, but the review comments identify a likely Windows regression in the proposed `spawnSync("oxfmt")` approach, and the PR only validated syntax (`node --check`) rather than Windows behavior.
