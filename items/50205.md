## Summary

Keep open. Current main still has no supported `requestLabels`/`labels` configuration for Gemini or `google-vertex` provider entries, and the Google Generative AI request builder does not inject a `labels` field before sending `streamGenerateContent` requests. The issue is recent, author is not a maintainer, and it has no protected label, so there is no conservative cleanup reason to close it.
