## Summary

Keep #23353 open. Current main still lacks Anthropic native server-side web_search/web_fetch/code_execution support, Programmatic Tool Calling allowed_callers, and response handling for Anthropic server-tool content blocks. The related implementation PRs were closed unmerged, and the latest issue discussion explicitly keeps this tracker open.

## Best Possible Solution

Keep #23353 open as the broad Anthropic native server-tools tracker. The best implementation path is an explicit Anthropic provider/runtime design covering config/schema/docs, request payload injection for web_search/web_fetch/code_execution server tools, suppression or coexistence rules for managed web tools, response block preservation/rendering, Programmatic Tool Calling allowed_callers semantics, and focused tests. If maintainers want smaller work units, split web_search/web_fetch and code_execution/PTC into canonical follow-up issues rather than closing this tracker.
