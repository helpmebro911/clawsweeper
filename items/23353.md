## Summary

Keep #23353 open. Current main still does not implement Anthropic native server-side web_search, web_fetch, code_execution, Programmatic Tool Calling allowed_callers, or response handling for Anthropic server-tool content blocks. The related implementation attempts are closed unmerged, and the latest issue discussion explicitly keeps this tracker open.

## Best Possible Solution

Keep #23353 open as the broad Anthropic native server-tools tracker. The best implementation path is an explicit Anthropic provider/runtime design covering config/schema/docs, request payload injection for web_search/web_fetch/code_execution server tools, managed tool suppression or coexistence rules, response block preservation/rendering, Programmatic Tool Calling allowed_callers behavior, and focused unit/integration tests. If maintainers want smaller work units, split web_search/web_fetch and code_execution/PTC into canonical follow-up issues rather than closing this tracker.
