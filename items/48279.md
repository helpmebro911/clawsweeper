## Summary

Keep open. The reported ACP `session/list` regression still appears present on current `main`: OpenClaw advertises session list capability and pins `@agentclientprotocol/sdk` 0.20.0, but `AcpGatewayAgent` still implements `unstable_listSessions` instead of the SDK-dispatched `listSessions` method. The related PR #48292 was closed unmerged in the provided context, and the current source confirms the rename has not landed.
