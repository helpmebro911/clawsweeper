## Summary

Keep open. Current `main` does not implement this PR’s direct-first sudo behavior; it explicitly does the opposite by routing non-root `SUDO_USER` calls straight to `systemctl --machine <user>@ --user`. The provided review discussion also identified plausible regressions in the PR’s fallback logic, so there is not strong evidence to close it as implemented, obsolete, or invalid.
