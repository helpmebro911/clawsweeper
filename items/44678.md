## Summary

Keep open. Current `main` still does not honor configured Ollama `params.num_ctx` on the native path, and this PR is neither merged nor replaced by shipped behavior. The linked bug #44550 is still open, with a newer PR (#69464) discussed as the more complete fix, so there is not strong evidence to close #44678 conservatively.
