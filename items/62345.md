## Summary

Keep open. Current main has Discord proxy support and a merged multipart attachment fix, but the reported failure is JSON final-reply/slash-command delivery with undici v8 plus a global EnvHttpProxyAgent workaround. The available fix evidence is scoped to multipart uploads, and the broader proxy-fetch normalization remains tracked by open PR #66243 rather than implemented on main.
