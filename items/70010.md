## Summary

Keep PR #70010 open. Current main still does not document Google Tasks in the bundled `gog` skill: the skill metadata/prose omit Tasks, the setup command omits the `tasks` service, and the common command list has no `gog tasks ...` examples or `gog tasks clear`. The related issue #69094 is closed, and duplicate PR #69257 was closed unmerged, so this PR remains the active implementation path rather than being obsolete.

## Best Possible Solution

Keep the PR open and have maintainers either merge or refine it as the focused update to the existing bundled `skills/gog/SKILL.md`: include Tasks in metadata/prose, include `tasks` in the OAuth service setup, and document the useful Google Tasks command examples including `gog tasks clear <tasklistId>`. If maintainers later decide `gog` should leave bundled skills, that should be a separate product/plugin packaging decision rather than an auto-close of this focused docs fix.
