## Summary

Keep open. Current `main` has partial thinking-block handling, but the issue’s requested auto-recovery is not implemented in the production reply runner. The code preserves provider-owned thinking replay for newer Claude models, yet `runAgentTurnWithFallback` still only auto-resets for compaction failures and role-ordering conflicts, not Anthropic `thinking`/`redacted_thinking` modification errors.
