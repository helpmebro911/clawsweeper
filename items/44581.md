## Summary

Current `main` does not show the PR’s replay-boundary fix on the new provider-hook architecture. The code still lacks history-aware preservation of signed thinking blocks for non-Anthropic replay targets, and the wrapped-Mistral `strict9` case is only covered for the native Mistral provider, not OpenRouter-style `mistralai/...` ids.
