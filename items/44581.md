## Summary

Current `main` does not contain the PR’s core fixes. Replay policy is still chosen from the target provider/model only, without inspecting signed history, OpenAI-boundary sanitization still only handles `rs_*` JSON reasoning signatures, and OpenRouter still uses passthrough-Gemini replay hooks rather than a Mistral `strict9` path for wrapped `mistralai/*` models. This PR should stay open and be refreshed onto the newer provider-owned replay-policy structure.
