## Summary

Keep this PR open. Current main still has no Linux, Windows, or shared desktop companion app, while the vision and platform docs explicitly treat Linux and Windows companion apps as planned first-party platform work. The submitted Tauri implementation has functional and security blockers, and there is overlapping open work in #59075 plus canonical #75, but those call for maintainer review/consolidation rather than conservative auto-close.

## Best Possible Solution

Keep this PR open for maintainer/product/security review or author follow-up. The best path is to coordinate it with #75 and #59075, choose a shared Linux/Windows desktop companion architecture, rebase onto current main, fix the Tauri build/runtime/security blockers, add workspace/release/docs/labeler/CI wiring, and validate Linux and Windows builds before merge consideration.
