## Summary

Keep open. Current main fixes the unresolved SecretRef crash and plain `agents list --json` plugin preload path, but the reported CLI exit-hang surface is not fully resolved: human `agents list` and `agents add` still load plugins, and CLI shutdown only has a memory-runtime cleanup path, not a generic external-plugin teardown path.

## Best Possible Solution

Keep this issue open but narrow maintainer action to the remaining CLI exit-hang surface. The best fix is to keep config-only `agents` paths, especially non-interactive `agents add` and human `agents list` where feasible, off broad runtime plugin loading; if a path truly needs plugins, add a short-lived CLI plugin-runtime teardown contract that external plugins can implement. Treat the SecretRef fallback, no-plugin bind/delete overrides, and plain JSON list fast path as resolved subparts.
