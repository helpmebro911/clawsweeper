## Summary

Keep open. Current main partially addresses one part of the request with `agents.defaults.experimental.localModelLean`, but it is an experimental nested flag that only drops a few heavyweight tools. The requested stable top-level `localMode` switch still has meaningful missing scope: timeout defaults, compaction tuning, model/embedding preferences, installer/config CLI exposure, and hot reload behavior. The issue is recent and the reporter supplied concrete before/after data and a working manual workaround, so it is not stale or unactionable.
