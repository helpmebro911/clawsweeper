## Summary

Current `main` does not implement the PR's Langfuse-oriented trace-tree enrichment. The requested config surface, outbound diagnostic event, agent-event bridge, and enriched OTEL spans are absent from the shipped code/docs, so there is no strong basis to close this PR as already implemented or obsolete.
