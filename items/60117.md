## Summary

Current `main` does not contain the proposed contributor-credit/deduplication guidance in `CONTRIBUTING.md` or the matching reminder in `AGENTS.md`. There is partial related implementation in maintainer tooling (`Co-authored-by` and `thanks @...` checks), but that does not make this docs PR implemented; it still represents potentially useful contributor-facing documentation and now mainly needs rebasing onto the newer docs files added on `main`.
