## Summary

Keep open. Current main has partial related work for managed inbound media and iMessage attachment-root validation, but it does not implement the requested configurable `tools.image.mediaLocalRoots`/equivalent for the `image` tool. The image tool still resolves local roots from the workspace/default OpenClaw roots only, and the tool config schema has no image-tool local-root field.
