## Summary

Keep open. Current `main` still routes memory embedding HTTP calls through the default strict guarded-fetch mode, so env proxy settings are not enabled on that path. The PR's original file target is stale after the Apr 25 refactor into `src/memory-host-sdk/host/remote-http.ts`, but the underlying behavior was copied forward unchanged rather than implemented on `main`.
