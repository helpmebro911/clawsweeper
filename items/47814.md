Keep #47814 open. Current main still does not contain the named DM session key helpers, active named-session store field, store helpers, docs, or tests proposed by the PR, and the PR discussion has unresolved correctness concerns rather than evidence that the work is obsolete or already shipped.

## Best Possible Solution
