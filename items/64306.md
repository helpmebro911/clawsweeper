## Summary

Keep open. Current main still bootstraps most CLI commands through plugin-registry loading, the linked `activationSourceConfig` fix is not present on `main` or the latest release tag in this checkout, and a newer Windows issue (#70451 on 2026-04-23) reports the same 60-second CLI hang on OpenClaw 2026.4.21.
