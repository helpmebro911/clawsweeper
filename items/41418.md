## Summary

Keep open. Current main still lacks a global `openclaw agent --dry-run`, and the universal tool adapter continues to call `tool.execute()` rather than returning preview payloads. Existing `disableTools` behavior removes tools instead of allowing the model to plan tool calls safely.
