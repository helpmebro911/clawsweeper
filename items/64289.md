## Summary

Keep open. Current `main` weakens the reported root-cause analysis, but it does not provide strong evidence that the idle-time reconnect/flash is fixed or obsolete. The startup path reloads deferred startup plugins before WebSocket handlers are attached, and the logged `code=1001` disconnect is client-originated rather than a server-side close. The named plugins from the report are not present in this repository, so the exact repro path cannot be verified from repo code alone.

## Evidence

- **Deferred startup plugins are reloaded before WS handlers attach:** Gateway startup reloads deferred configured channel plugins and updates gateway methods before calling `attachGatewayWsHandlers`, so the specific claim that startup leaves those plugins to initialize only on the first later webchat interaction is not supported by current `main`.
  - file: [src/gateway/server.impl.ts:770](https://github.com/openclaw/openclaw/blob/11804a484ded89374afa55020813cfa265aa143a/src/gateway/server.impl.ts#L770)
  - sha: [11804a484ded](https://github.com/openclaw/openclaw/commit/11804a484ded89374afa55020813cfa265aa143a)
- **Startup readiness test blocks WS until startup is ready:** The gateway startup readiness test asserts the port is still refused before startup resolves, and then immediately accepts a WS connection once startup completes.
  - file: [src/gateway/server.startup-websocket-race.test.ts:48](https://github.com/openclaw/openclaw/blob/11804a484ded89374afa55020813cfa265aa143a/src/gateway/server.startup-websocket-race.test.ts#L48)
  - sha: [11804a484ded](https://github.com/openclaw/openclaw/commit/11804a484ded89374afa55020813cfa265aa143a)
- **Server logs webchat close codes; it does not synthesize 1001 here:** The webchat disconnect log path records the received close code/reason. In the same codebase, the explicit server-side restart close uses `1012`, not `1001`, so the reported `code=1001` is not explained by an obvious current server close path.
  - file: [src/gateway/server/ws-connection.ts:332](https://github.com/openclaw/openclaw/blob/11804a484ded89374afa55020813cfa265aa143a/src/gateway/server/ws-connection.ts#L332)

  - file: [src/gateway/server-close.ts:200](https://github.com/openclaw/openclaw/blob/11804a484ded89374afa55020813cfa265aa143a/src/gateway/server-close.ts#L200)
