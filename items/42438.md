## Summary

Current `main` does not implement the requested owner-notify/suppress-pairing-response feature. WhatsApp DM policy still only supports `pairing`, `allowlist`, `open`, and `disabled`; `allowlist` already silently blocks unauthorized senders, but there is no `silent`/`notify` DM policy, no `pairingResponse`/`notifyOnBlocked` config, and no blocked-DM notification path for the owner.
