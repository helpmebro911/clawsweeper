## Summary

Keep this PR open. Current main still does not wire `task` or `team` tools into the OpenClaw core tool set, the owner-only core list still contains only `cron`, `gateway`, and `nodes`, and no `task-tool.ts` or `team-tool.ts` implementation exists on main. The PR remains a plausible implementation candidate, but it is stacked on open dependency #67331 and has unresolved review feedback in security-sensitive agent/tool and filesystem-mutation code paths.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to resolve or land #67331 first, rebase or split this PR so the task/team tool wiring is isolated, fix the schema and runtime error-handling review comments, then validate owner-only behavior through focused agent-tool and `/tools/invoke` tests before considering merge.
