## Summary

Current `main` does not appear to implement this request. For Telegram voice notes, media is fetched before the message enters dispatch, and audio preflight transcription can run while building message context before the reply pipeline starts the typing indicator. The documented `typingMode: "instant"` behavior is tied to model-loop start, not inbound voice receipt, so the reported UX gap remains plausible and should stay open.
