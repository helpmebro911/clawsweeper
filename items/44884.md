## Summary

Current `main` does not contain the PR’s gateway hardening work. The new config surface (`gateway.tls.terminatedUpstream`, `gateway.ipAllowlist`, `gateway.ipBlocklist`, `gateway.requestRateLimit`), the HTTP/WS enforcement paths, the stronger doctor/audit checks, and the bind-mode hard failures described in PR #44884 are all still missing, so there is no basis to close this PR as already implemented or obsolete.
