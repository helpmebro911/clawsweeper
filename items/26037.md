## Summary

Keep open. Current main has first-class Qwen/Alibaba DashScope Coding Plan support, but the specific requested capability, official reasoning/thinking enablement for the cited Coding Plan models, is not implemented out of the box. The bundled Qwen catalog still marks qwen3.5-plus, glm-5, glm-4.7, qwen coder models, and kimi-k2.5 as non-reasoning, while the OpenAI-compatible transport only sends reasoning_effort when the selected model has reasoning: true.
