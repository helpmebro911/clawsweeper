## Summary

Keep open. Current `main` still only supports a string `memorySearch.fallback` and routes fallback embeddings through the shared primary `remote` config, so the requested separate fallback remote/model config is not implemented. The PR also carries unrelated/stale changes and unresolved review findings, which argues against conservative auto-close rather than for it.
