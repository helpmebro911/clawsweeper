## Summary

Keep open. Current `main` and the latest shipped release still only subscribe the WhatsApp integration to `creds.update`, `connection.update`, and `messages.upsert`; there is no `call` event listener, no call-tracking config surface, and no call-event tests. The April 25 maintainer comment is consistent with the current code, so this is still a live enhancement request rather than something already fixed or obsolete.

## Evidence
