## Summary

Keep open. Current main already has Mattermost post update/delete helpers for preview streaming and interaction cleanup, but the shared Mattermost message action surface still advertises and handles only send/react. The PR targets a real remaining Mattermost capability, but the branch needs a focused refresh before maintainer review because it mixes unrelated changes and includes a production extension import from core internals.

Keep this PR open and request a focused refresh: wire Mattermost edit/delete through describeMattermostMessageTool(), supportsAction(), and handleAction(); reuse the existing updateMattermostPost()/deleteMattermostPost() helpers; validate required edit content; add targeted Mattermost action tests; and remove unrelated infra/test/changelog changes plus the core-internal production import before maintainer review.
