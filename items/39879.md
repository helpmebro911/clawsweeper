## Summary

Current main does not implement WhatsApp outbound mention injection. The WhatsApp send path still builds plain Baileys text/media payloads without deriving or passing `mentionedJids`, and the existing mention handling is inbound-only. The issue should remain open as a bundled WhatsApp channel-plugin feature gap.
