## Summary

Keep #18160 open. Current main still has no direct cron exec/shell payload: cron payloads are limited to systemEvent and agentTurn, isolated/current/session jobs are required to be agentTurn, and the runtime still routes detached jobs through runIsolatedAgentJob. The linked implementation PR #51276 was closed unmerged, while related #37651 and #50558 are duplicate open requests rather than a clear canonical superseding item.

## Evidence

- **Cron payload type excludes exec:** `CronPayload` is only `{ kind: "systemEvent"; text: string } | CronAgentTurnPayload`; `CronPayloadPatch` has the same two-kind shape. No `exec` or `shell` payload exists on current main.
  - file: [src/cron/types.ts:112](https://github.com/openclaw/openclaw/blob/bd796d1c85c7dec0dcb89ed11502e4271ded1314/src/cron/types.ts#L112)
