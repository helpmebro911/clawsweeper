## Summary

Keep open. Current `main` preserves inter-session provenance on injected sub-agent/main-agent messages, but the chat UI still renders groups and avatars from coarse message roles only. That means cross-session steer/announce traffic can still collapse into generic `user`/`assistant` presentation instead of distinct main-agent vs sub-agent identities.

## Evidence

- **Subagent announce writes inter-session provenance:** Subagent announce agent calls attach `inputProvenance` with `kind: "inter_session"`, `sourceSessionKey`, and `sourceTool: "subagent_announce"`, which is the metadata the UI would need to distinguish injected agent-originated messages.
  - file: [src/agents/subagent-announce.ts:193](https://github.com/openclaw/openclaw/blob/a958b6e72328d1d922a6d133b1a70e866faf4013/src/agents/subagent-announce.ts#L193)
  - sha: [a958b6e72328](https://github.com/openclaw/openclaw/commit/a958b6e72328d1d922a6d133b1a70e866faf4013)
- **Current tests expect parent-session subagent messages to carry provenance:** The e2e test for direct completion into the main session asserts `inputProvenance.kind === "inter_session"` and `sourceSessionKey === "agent:main:subagent:test"`, confirming current main still models these messages as cross-session injections rather than a distinct chat role.
  - file: [src/agents/subagent-announce.format.e2e.test.ts:611](https://github.com/openclaw/openclaw/blob/a958b6e72328d1d922a6d133b1a70e866faf4013/src/agents/subagent-announce.format.e2e.test.ts#L611)
  - sha: [a958b6e72328](https://github.com/openclaw/openclaw/commit/a958b6e72328d1d922a6d133b1a70e866faf4013)
- **User-message provenance is preserved in stored messages:** `applyInputProvenanceToUserMessage` adds `provenance` onto `role: "user"` messages, so cross-session agent messages are persisted as user-role messages with extra provenance metadata.
  - file: [src/sessions/input-provenance.ts:43](https://github.com/openclaw/openclaw/blob/a958b6e72328d1d922a6d133b1a70e866faf4013/src/sessions/input-provenance.ts#L43)

  - file: [src/gateway/server-methods/chat.ts:1052](https://github.com/openclaw/openclaw/blob/a958b6e72328d1d922a6d133b1a70e866faf4013/src/gateway/server-methods/chat.ts#L1052)
