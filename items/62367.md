## Summary

Keep #62367 open. Current main still treats `openai-codex/gpt-5.4-mini` as a Codex OAuth forward-compat route, but there is no equivalent `openai-codex/gpt-5.4-nano` dynamic model, modern-model policy, or catalog augmentation. The related #50265 model-list work does not supersede this runtime regression report because the requested nano Codex route remains missing or unsupported on current main.
