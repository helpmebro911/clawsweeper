## Summary

Current main still renders the Channels settings page from backend-provided loaded-plugin order only, so the requested built-in-plus-custom display behavior is not implemented.

## Evidence

- **UI still trusts backend order exclusively:** `resolveChannelOrder` returns `snapshot.channelMeta.map(...)` or `snapshot.channelOrder` whenever present, and only falls back to the built-in hardcoded channel list when both are absent. That means a snapshot containing only a custom loaded channel still hides bundled built-ins.
  - file: [ui/src/ui/views/channels.ts:97](https://github.com/openclaw/openclaw/blob/c6684af682bcb59bd1ee85cb49f72d4074007e2c/ui/src/ui/views/channels.ts#L97)
  - sha: [c6684af682bc](https://github.com/openclaw/openclaw/commit/c6684af682bcb59bd1ee85cb49f72d4074007e2c)
- **Gateway snapshot is built from loaded channel plugins:** `channels.status` calls `listChannelPlugins()` and feeds that directly into `buildChannelUiCatalog(plugins)`, then returns `channelOrder` and `channelMeta` from that catalog.
  - file: [src/gateway/server-methods/channels.ts:162](https://github.com/openclaw/openclaw/blob/c6684af682bcb59bd1ee85cb49f72d4074007e2c/src/gateway/server-methods/channels.ts#L162)
  - sha: [c6684af682bc](https://github.com/openclaw/openclaw/commit/c6684af682bcb59bd1ee85cb49f72d4074007e2c)
- **Loaded-plugin registry excludes disabled bundled channels:** `listChannelPlugins()` is currently just `listLoadedChannelPlugins()`, so bundled channels that are not loaded cannot appear in the UI catalog returned by `channels.status`.
  - file: [src/channels/plugins/registry.ts:8](https://github.com/openclaw/openclaw/blob/c6684af682bcb59bd1ee85cb49f72d4074007e2c/src/channels/plugins/registry.ts#L8)
  - sha: [c6684af682bc](https://github.com/openclaw/openclaw/commit/c6684af682bcb59bd1ee85cb49f72d4074007e2c)
- **UI catalog helper does not merge fallback bundled entries:** `buildChannelUiCatalog` only maps over the `plugins` argument it receives; it does not append built-in catalog entries from the broader channel catalog.
  - file: [src/channels/plugins/catalog.ts:279](https://github.com/openclaw/openclaw/blob/c6684af682bcb59bd1ee85cb49f72d4074007e2c/src/channels/plugins/catalog.ts#L279)
  - sha: [c6684af682bc](https://github.com/openclaw/openclaw/commit/c6684af682bcb59bd1ee85cb49f72d4074007e2c)
- **No landed fix found in local history:** A local `git log --oneline --all --grep='channel cards' --grep='always include built-in' --grep='merge builtin custom' --grep='resolveChannelOrder'` search returned no matching fix commit in the checked-out history.
  - command: `git log --oneline --all --grep='channel cards' --grep='always include built-in' --grep='merge builtin custom' --grep='resolveChannelOrder'`
