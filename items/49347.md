## Summary

Keep open. The root/published npm install blocker was addressed by the related dependency-split PRs, but the reported full source/workspace `pnpm install` path still includes extension workspaces and still has the WhatsApp Baileys -> libsignal GitHub codeload tarball in the current lockfile. A maintainer comment explicitly distinguished those cases and left this issue open for the remaining source-install reliability problem.
