## Summary

Keep this PR open. Current main still lacks Telegram forum-topic deletion: the Telegram message action map exposes `delete`, `topic-create`, and `topic-edit`, but not `topic-delete` or `deleteForumTopic`; docs and tests likewise only cover message deletion plus create/edit forum-topic behavior. The PR is stale against the current Telegram plugin layout and needs security hardening before merge, but it addresses a real unshipped capability rather than something implemented, superseded, or out of scope.

## Best Possible Solution

Keep the PR open as useful but request a rebase/port to the current Telegram plugin layout. The rebased implementation should add `topic-delete` to the shared action list and target spec, map it in `extensions/telegram/src/channel-actions.ts`, implement `deleteForumTopicTelegram` in `extensions/telegram/src/send.ts`, add a `deleteForumTopic` branch in `extensions/telegram/src/action-runtime.ts`, update docs/tests, and harden destructive ID parsing to accept only base-10 integer IDs.
