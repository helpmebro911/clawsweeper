## Summary

Keep open. Current `main` already dedupes `PluginManifestRegistry.plugins` down to one row per duplicate id, but it still unconditionally warns for bundled-vs-global/workspace shadows and still builds loader lookups from the deduped `plugins[]` array. That means the PR's main remaining value, preserving loser roots for disabled `overridden by ...` rows via a per-root index and suppressing benign bundled-shadow warnings, is not implemented on `main`.
