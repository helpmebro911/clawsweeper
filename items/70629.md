## Summary

Keep #70629 open. Current main still maps Gemini 2.5 `reasoning: "minimal"` to `thinkingBudget: 128`; the PR addresses a real remaining Google plugin path, though maintainers should verify whether the blanket 512 floor should apply to Pro/Flash or only Flash-Lite.

## Best Possible Solution

Keep the PR open for maintainer review. The best path is to land a targeted Google plugin fix that prevents invalid `thinkingBudget: 128` for `gemini-2.5-flash-lite`, with regression coverage for the affected Gemini 2.5 model ids; if live Google API evidence confirms Pro/Flash also reject 128, the current broader 512 floor is reasonable, otherwise narrow it to avoid unnecessary reasoning-token budget increases.
