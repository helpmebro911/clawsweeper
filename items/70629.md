## Summary

Keep PR #70629 open. Current main still maps `reasoning: "minimal"` for Gemini 2.5 thinking-budget models to `thinkingBudget: 128`, and that still hits Flash-Lite because the `2.5-flash` branch also matches `gemini-2.5-flash-lite`. The PR addresses a real remaining gap, though maintainers may want to narrow or justify the Pro/Flash floor change because official Gemini docs currently require 512 specifically for Flash-Lite, while listing lower valid ranges for Pro and Flash.

## Best Possible Solution

Keep the PR open for maintainer review. The core fix should ensure `gemini-2.5-flash-lite` no longer sends `thinkingBudget: 128` for minimal reasoning, with a regression test on `buildGoogleGenerativeAiParams`; maintainers should decide whether to keep the PR's conservative 512 floor for all Gemini 2.5 budget models or narrow it to Flash-Lite while preserving the documented Pro/Flash minima.
