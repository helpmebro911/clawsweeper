## Summary

Keep this PR open. Current main still sends Control UI tool-event arguments and some derived tool output paths without secret redaction, while the PR is an active fix candidate for the linked open security report #72283.

## Best Possible Solution

Keep the PR open and review it as a plausible core security fix. The best path is to land this PR or a maintainer-approved variant that redacts sensitive text at the agent-event producer boundary for tool start args, partial/final tool results, result details used by command_output and patch summary derivation, plus focused regression tests for the Control UI event path. Close the linked security report only after the fix is merged to main.
