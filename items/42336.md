## Summary

Keep open. Current main still archives pre-reset transcripts to `.jsonl.reset.*`, but the session selector only renders `sessions.list`, and `sessions.list` only enumerates live session-store entries. I found no code on `main` that indexes reset archive files back into the session list, so the reported behavior is still not implemented.
