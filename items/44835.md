## Summary

Keep open. Current `main` has code and tests that suggest the form no longer depends on blur to persist text fields, and Moonshot/Kimi routing is now documented more clearly, but I did not find issue-specific proof tying #44835’s `models.*.id/name undefined` report on `v2026.3.12` to a shipped fix on `main`. With no clean release provenance, no runnable repro in this read-only checkout, and the issue only 42 days old, the conservative bar for closure is not met.
