## Summary

Keep #50865 open. Current main has a Sessions UI for active store entries and archives transcripts on `/new`, `/reset`, and delete, but it does not expose archived `.reset.`/`.deleted.` transcripts as restorable old sessions in the WebUI. The KeepMyClaw comment is a useful external workaround, but it does not make the requested OpenClaw WebUI restore/access flow available. Related PR #51105 was closed unmerged and only proposed CLI export, not GUI restore.
