## Summary

Current main has a newer SecretRef system with env/file/exec providers, and it documents using 1Password through the generic exec provider. It does not add native keyring or 1Password providers, and the broad #13610 tracker does not exactly cover that remaining keyring/1Password work, so #17311 should stay open.
