## Summary

Keep #27445 open. Current main still lacks the requested native `announceTarget` config default and `sessions_spawn` per-call override for routing native sub-agent completion announces into the parent session. Adjacent work exists, including active requester steering, channel/thread fallback fixes, ACP-only `streamTo:"parent"`, and related #33478 callback/thread hardening, but those do not provide the requested native parent/channel announce policy.

## Best Possible Solution

Keep #27445 open as a core subagent announce policy request. The likely maintainer path is to add a public `channel`/`parent` announce policy to native `sessions_spawn` and `agents.defaults.subagents`, update config schema/help/generated docs and https://docs.openclaw.ai/tools/subagents, and cover top-level parent-only routing, current channel delivery, active requester steering, nested subagent delivery, and interactions with #33478 callback/thread hardening.
