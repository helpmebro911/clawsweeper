## Summary

Keep #11977 open. Current main still has Docker release jobs only for linux/amd64 and linux/arm64, and the manifest job only assembles those two digests. Related PR #12033 is useful prior art, but it was closed unmerged, and the current Dockerfile still has riscv64-specific prerequisite concerns such as unconditional Bun bootstrap and native-addon verification.

## Best Possible Solution

Keep #11977 open until a maintainer lands a Docker release workflow change that adds a non-blocking linux/riscv64 image build, includes the riscv64 digest in the multi-platform manifest when available, handles Dockerfile prerequisites on riscv64, and updates Docker release validation or docs as needed. Closed PR #12033 should be treated as prior art, not as a canonical superseding item.
