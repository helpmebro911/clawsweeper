## Summary

Current main already prevents the startup config-clobber outcome this PR targets. OpenClaw now rejects destructive internal config rewrites before they replace `openclaw.json`, keeps `.rejected.*` artifacts for inspection, and documents/restores last-known-good configs when `gateway.mode` disappears or the file sharply shrinks.
