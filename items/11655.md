## Summary

Keep #11655 open. Current main still does not implement Slack App Home/Home tab support: the bundled Slack plugin enables only the App Home messages tab, does not subscribe to `app_home_opened`, does not register an App Home monitor handler, and has no `views.publish`/Home tab action or config surface. The related implementation PRs in the provided context were closed unmerged, and duplicate #52020 points back to #11655 as the canonical tracker.

## Best Possible Solution

Keep #11655 as the canonical tracker. Implement the feature in the bundled Slack plugin by adding `app_home_opened` to the generated/setup docs, registering a handler under `extensions/slack/src/monitor/events`, publishing a safe default Home view with `app.client.views.publish`, and adding focused tests for event filtering, publish payloads, and the chosen config/tool customization surface.
