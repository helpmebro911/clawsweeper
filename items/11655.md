## Summary

Keep #11655 open. Current main at 4a3030df9efa72c44bab567a4390ac0c4876a73f still does not implement Slack App Home support: there is no `app_home_opened` event subscription or handler, no `views.publish` runtime path, and no Home-tab config/tool customization surface. The prior related PRs were closed unmerged, and duplicate #52020 was closed back to #11655 as the canonical tracker.

## Best Possible Solution

Keep #11655 open as the canonical bundled Slack App Home tracker. Implement it in `extensions/slack` by adding `app_home_opened` to setup/docs, registering a monitor handler under `extensions/slack/src/monitor/events`, publishing a safe default Home view through Slack `views.publish`, deciding the supported customization path, and adding focused tests for event filtering, payload shape, error handling, manifest/docs coverage, and config/tool behavior.
