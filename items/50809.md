## Summary

Keep open. Current main has Android `sms.send` and `sms.search`, plus BlueBubbles support for `sms:` targets, but it does not implement the requested `sms.read` command or an Android-node-mediated SMS messaging channel. The linked implementation PR #50808 was closed unmerged, so it does not supersede the issue.
