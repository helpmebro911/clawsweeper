## Summary

Keep open. Current main does not implement the requested macOS wallboard shell, no related item in the provided context supersedes it, and the PR is a focused additive feature. The existing WebChat and Canvas WKWebView surfaces are not equivalent to a fullscreen external-display wallboard shell. The PR should remain open for maintainer review and author fixes to the unresolved review comments.

## Best Possible Solution

Keep PR #69999 open as the current REL-433 implementation candidate. Ask the author to address the safe Bundle.main-first resource lookup, activation/KVC/reload review comments, and then verify the packaged macOS app with --wallboard and an external display before maintainer merge consideration.
