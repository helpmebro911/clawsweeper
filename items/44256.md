## Summary

Current `main` still contains the exact `@_all` early-return behavior, now in the extracted Feishu helper, so this PR is addressing a real unresolved bug rather than something already fixed. The implementation likely needs a rebase because the mention logic moved from `bot.ts` to `bot-content.ts`, but the underlying issue remains open on `main`.
