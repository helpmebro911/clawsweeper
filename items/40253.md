## Summary

Current `main` does not implement Telegram inbound-message mirroring into Control UI/WebChat, and the PR's March 2026 approach is stale against today's telegram plugin layout and current chat event contract. This should stay open for a fresh rework rather than be closed as already fixed or obsolete.
