## Summary

Keep open. Current main has useful auth-path improvements, but I do not see high-confidence evidence that the reported macOS app + local Docker gateway schema-unavailable/token-mismatch flow is fully fixed. The config UI can still degrade to generic "Schema unavailable" when schema fetch auth fails, direct remote-style connections require the app-side remote token to be set, and the linked #64973 remains open with a more detailed token-mismatch/config-write reproduction against a local token-auth gateway.
