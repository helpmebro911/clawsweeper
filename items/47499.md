## Summary

Keep #47499 open. Current main has adjacent pieces: sandbox-browser noVNC observer URLs, node screen/canvas capabilities, and plugin/node-host extension surfaces. It does not implement the central request: an embedded Control UI VNC viewer plus agent-initiated full desktop takeover with a dashboard permission/token handoff. The linked PR #47497 was closed unmerged, and the provided context does not identify a canonical superseding issue. This is also too core/product/security-sensitive to auto-close as ClawHub cleanup because the missing part is dashboard integration plus a takeover authorization protocol, not just a standalone plugin idea.
