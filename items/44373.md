## Summary

Keep open. Current `main` only partially covers the PR intent: overview navigation cards are now real buttons, but the broader affordance mismatch still exists because generic `.card` and `.account-card` hover styling remains on non-interactive panels. The PR also carries substantial unrelated changes, so there is not strong evidence for a conservative close.
