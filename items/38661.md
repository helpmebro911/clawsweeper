## Summary

Keep open. Current main still lacks the remaining TUI Ctrl+C active-run hint from #38501, and this PR contains a concrete patch for that UX bug. The bundled PR should not merge as-is because its web_search rename conflicts with current main's managed/native web-search policy, while the Feishu, bootstrap-cache, and Telegram pieces are already handled or superseded on main.

## Best Possible Solution

Keep the PR open as contributor evidence for #38501, but do not merge the bundled branch as-is. The maintainer path is a narrow TUI-only patch in `src/tui/tui.ts` that adds `run still active (Esc or /abort to stop)` when `activeChatRunId` is set, plus a regression test for that status text. Preserve current main's `web_search` behavior and the existing Feishu, bootstrap-cache, and Telegram implementations.
