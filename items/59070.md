## Summary

Do not close. Current `main` still shows the same core mismatch the report described: the config contract defines the default agent via `agents.list[].default`, but the Control UI still has a code path that writes `agents.defaultId`, so there is not strong evidence that #59070 is fully fixed on `main`. The linked PRs in the issue timeline are useful context, but this checkout does not prove their intended fix is present end-to-end.
