## Summary

Keep #59509 open. It is a recent, concrete feature request from a non-maintainer with no protected labels, and current main still appears to expose provider/model config without a `location` field on model definitions or provider config. The existing public docs only state that `google-vertex` uses gcloud ADC; they do not document a per-model Vertex location override or the requested resolution order.
