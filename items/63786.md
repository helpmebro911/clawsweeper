## Summary

Keep this PR open. Current main still routes GitHub Copilot Claude replay through the generic sanitizer that preserves latest assistant thinking blocks, while the Copilot replay policy asks for thinking-block dropping. The PR is still a plausible fix, but it needs maintainer review for normalized provider selection and for the bundled ACP/guard-script changes.

## Best Possible Solution

Keep this PR open. Ask the author to refresh it against current main, select the Copilot replay sanitizer using normalized provider or explicit replay-policy state, apply the same strategy to session-history replay, tool-continuation replay, and context-pruning estimates, update the Copilot regression tests, and split or justify the ACP/guard-script changes before maintainer review and merge.
