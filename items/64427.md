## Summary

Keep open. Current `main` still documents and types `channels.slack.dmHistoryLimit`, but the Slack runtime only threads `historyLimit` through room/group-DM handling and never consumes `dmHistoryLimit` for 1:1 DM session bootstrap.

## Evidence

  - file: [src/config/types.slack.ts:153](https://github.com/openclaw/openclaw/blob/11804a484ded89374afa55020813cfa265aa143a/src/config/types.slack.ts#L153)
