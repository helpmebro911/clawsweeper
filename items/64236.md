## Summary

Keep open. Current `main` still supports reasoning visibility defaults only at the session/per-agent level; there is no implemented `agents.defaults.reasoningDefault` field and no per-model `reasoningDefault` config in the model catalog.

## Evidence

- **Global agent defaults omit `reasoningDefault`:** `AgentDefaultsConfig` includes `thinkingDefault`, `verboseDefault`, and related global defaults, but no `reasoningDefault` field.
  - file: [src/config/types.agent-defaults.ts:297](https://github.com/openclaw/openclaw/blob/11804a484ded89374afa55020813cfa265aa143a/src/config/types.agent-defaults.ts#L297)
  - command: `nl -ba src/config/types.agent-defaults.ts | sed -n '290,310p'`
  - sha: [11804a484ded](https://github.com/openclaw/openclaw/commit/11804a484ded89374afa55020813cfa265aa143a)
- **Per-model config has no `reasoningDefault` slot:** Model catalog entries only expose `alias`, provider `params`, and `streaming`; there is no per-model `reasoningDefault` property in the config type.
  - file: [src/config/types.agent-defaults.ts:31](https://github.com/openclaw/openclaw/blob/11804a484ded89374afa55020813cfa265aa143a/src/config/types.agent-defaults.ts#L31)
  - command: `nl -ba src/config/types.agent-defaults.ts | sed -n '18,36p'`
  - sha: [11804a484ded](https://github.com/openclaw/openclaw/commit/11804a484ded89374afa55020813cfa265aa143a)
- **Runtime reasoning resolution ignores global/per-model defaults:** Reasoning visibility resolves from session override, then `agents.list[].reasoningDefault`, then hard-falls back to `"off"`; it does not consult `agents.defaults` or model-specific config.
  - file: [src/auto-reply/reply/directive-handling.levels.ts:42](https://github.com/openclaw/openclaw/blob/11804a484ded89374afa55020813cfa265aa143a/src/auto-reply/reply/directive-handling.levels.ts#L42)

- **Docs still document per-agent-only reasoning default:** User-facing docs say `/reasoning` resolves via inline directive, session override, per-agent default, then fallback `off`. The config docs also describe `reasoningDefault` only under `agents.list[]`.
  - file: [docs/tools/thinking.md:99](https://github.com/openclaw/openclaw/blob/11804a484ded89374afa55020813cfa265aa143a/docs/tools/thinking.md#L99)
