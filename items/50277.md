## Summary

Keep open. Current main still positions the chat delete confirmation popover absolutely above the delete button with `bottom: calc(100% + 6px)`, which is the mechanism reported to overflow near the top of the chat viewport. The related fix PRs in the provided GitHub context are closed and unmerged, and current tests only cover left/right side classes, not viewport containment.
