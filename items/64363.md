## Summary

Keep open: current `main` does not implement Anthropic Advisor Tool support. The bundled Anthropic provider only maps existing extra params like `anthropicBeta`, `context1m`, `serviceTier`, and `fastMode`, while the native Anthropic transport still sends ordinary `/v1/messages` requests without any advisor-specific transport switch, tool injection, or system-hint shaping. The issue also has an active linked PR (`#64064`), so this is not conservative cleanup material.
