## Summary

Keep open. The request is still valid on current main: channel agent tool factories still only receive cfg, while core OpenClaw tools receive requesterSenderId and senderIsOwner in the same tool composition path. The linked PR #58808 remains relevant work rather than grounds for closing the issue.
