## Summary

Keep open. Current main already covers much of the macOS side through `system.run`, bundled app-specific skills, and Peekaboo-based UI automation, but it does not show an iOS native-app control surface. The issue's broader macOS/iOS cross-app-control request is only partially implemented, so there is not strong enough evidence to close it.
