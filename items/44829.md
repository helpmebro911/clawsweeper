## Summary

Keep open: current `main` still does not escape trailing `:` in Feishu post parsing, and the current Feishu regression tests do not cover the reported case. This PR still appears to address a live gap; the only visible concern is the review note that multiline text may want `/:$/gm` instead of `/:$/g`.
