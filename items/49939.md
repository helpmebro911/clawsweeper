## Summary

Keep #49939 open. Current main still keeps the generic fallback at 200K and only upgrades Anthropic Opus/Sonnet 4.x to 1M when `params.context1m: true` is set; the test suite explicitly asserts that `anthropic/claude-opus-4-6` without that flag resolves to 200K. The current 1M-by-default work applies to Opus 4.7, not the requested Opus/Sonnet 4.6 default behavior. The linked PR #49941 was closed unmerged, and the later related ClawSweeper reports also describe the same remaining 4.6 gap rather than proving it is fixed or superseded.
