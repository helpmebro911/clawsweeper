## Summary

Current `main` still installs/manages two distinct macOS LaunchAgents: `ai.openclaw.mac` for app launch-at-login and `ai.openclaw.gateway` for the local gateway. The app docs and code make that split intentional, but the doctor/status extra-service scan still appears to classify `ai.openclaw.mac` as an extra gateway-like service, so this issue is not safely closable.
