## Summary

Keep open. The specific `isolatedSession` config key is now accepted and present in the latest release tag, but the broader reported bug is docs/release drift. Current main still syncs docs to the publish repo on pushes to `main`, main is already at `2026.4.25` while the latest release tag is `v2026.4.24`, and there are post-release docs changes including a page that explicitly requires unreleased OpenClaw 2026.4.25.
