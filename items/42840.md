## Summary

Keep open. Current main still routes Control UI chat and reasoning text through the shared sanitized markdown renderer without a MathJax, KaTeX, or markdown math layer. The user discussion and duplicate reports describe a real Control UI rendering gap; the related open PR #67103 is only an implementation candidate, and the earlier PR #61510 was closed unmerged.

## Best Possible Solution

Keep this issue open as the canonical Control UI math-rendering request. Review and land #67103, or an equivalent focused Control UI implementation, adding sanitizer-safe KaTeX or MathJax support in ui/src/ui/markdown.ts with regression coverage in ui/src/ui/markdown.test.ts; close this issue only after the implementation merges to main.
