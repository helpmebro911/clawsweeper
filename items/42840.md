## Summary

Keep this issue open. Current main still has no MathJax/KaTeX/LaTeX rendering in the Control UI chat markdown path, and the related open PR #67103 is only an implementation candidate. The earlier PR #61510 used closing syntax but was closed unmerged, so it does not make the requested behavior shipped.

## Best Possible Solution

Keep this issue open as the canonical user request. Review and land #67103, or a similar focused Control UI change, by adding sanitizer-safe KaTeX or MathJax support in `ui/src/ui/markdown.ts`, adding regression coverage in `ui/src/ui/markdown.test.ts`, and closing this issue only after the implementation is merged to main.
