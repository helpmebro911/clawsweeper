## Risks / Open Questions

- The PR executes an operator-configured runner through a shell and launches Claude Code with edit permissions; this is intended but security-sensitive and should be explicitly reviewed before merge.

- Incident bundles include health/status output, gateway log tails, and runner output; docs should warn operators about possible sensitive local diagnostics before sharing bundles.
