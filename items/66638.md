## Summary

Keep open. Current main already has resilient final completion announce delivery for subagents, but it does not implement the requested per-run lifecycle contract. Spawn visibility is still just the tool result plus an internal session lifecycle event, subagent task notifications default to `done_only`, progress/state-change delivery is opt-in, terminal task auto-delivery is explicitly disabled for subagents, and the announce queue is in-memory rather than durable.

## Evidence
