## Summary

Keep #59225 open. It is a concrete, recent feature request for per-agent parameter-level tool schema filtering, and current main still exposes `sessions_spawn` as one schema with many optional parameters rather than supporting `allowedParams`/`deniedParams`. Related discussion points to narrower `streamTo` cleanup work, but that does not cover the requested per-agent schema filtering feature.

## Evidence
