## Summary

Keep #39232 open. Current main still has no plugin hook equivalent to `on_raw_inbound` / `before_channel_filter`, and the existing WhatsApp `message_received` path only emits after access control and group mention gating. Related PR #39240 was a useful WhatsApp-specific attempt, but it closed unmerged and did not land the requested core/plugin API surface.

## Evidence
