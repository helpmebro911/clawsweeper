## Summary

Keep open. Current main still renders the Control UI chat with `autoExpandToolCalls: false`, so the session/global verbose setting is not wired into the tool-card expansion path. The linked PR #50034 appears to have been closed unmerged, and the attempted fix is not present in this checkout.
