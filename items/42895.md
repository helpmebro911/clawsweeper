## Summary

Keep open. Current `main` still resolves the home directory only from `OPENCLAW_HOME`, `HOME`, `USERPROFILE`, or `os.homedir()`, so the Android/Termux fallback described in #42828 is not implemented on `main` or in the latest release. The PR also appears to address the review feedback about `OPENCLAW_HOME=~/...` by extracting a shared Termux helper in commit `3ad26c7` per the discussion provided.
