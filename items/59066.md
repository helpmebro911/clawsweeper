## Summary

Keep open. Current `main` still has the unguarded ACP projector path that can call `.trim()` on an undefined status text, and the ACP translator still emits `usage_update` snapshots without any `text` field. That matches the reported crash shape closely enough that there is not strong evidence to close this issue.
