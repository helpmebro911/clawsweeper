## Summary

Keep #49665 open. Current main still has the reported lazy-loader failure mode: `createLazy()` caches the dynamic import promise without a rejection handler, all lazy Control UI views including Channels still use it, and `lazyRender()` still renders `nothing` while the module is unavailable. The related PR #49697 was closed unmerged, and its referenced head commit is not present in this checkout.
