## Summary

Keep open. Current main still has the Control UI custom-theme importer fetching `https://tweakcn.com/r/themes/{id}` from the browser, while the shipped Control UI CSP `connect-src` only allows `'self'`, `ws:`, and `wss:`. The PR is a narrow, relevant fix rather than an obsolete or external item.

## Best Possible Solution

Keep the PR open and land this narrow fix, or an equivalent maintainer-authored change, by allowing the exact `https://tweakcn.com` origin in Control UI `connect-src` and keeping a regression assertion. The broader configurable CSP request in #47565 does not supersede this immediate core UI regression because tweakcn import is already a shipped Control UI feature.
