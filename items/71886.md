## Summary

Keep open. Current main does not include the proposed CSP allowlist entry, and the current Control UI custom theme path directly fetches `https://tweakcn.com/r/themes/{id}` from the browser. PR #71886 is a narrow, relevant fix against the implicated Gateway CSP builder with a matching regression assertion.

## Best Possible Solution

Keep PR #71886 open for normal maintainer review and likely merge, or land an equivalent narrow fix that adds `https://tweakcn.com` to the Control UI CSP `connect-src` directive with a regression assertion in the Gateway Control UI HTTP/CSP tests. The current browser-local tweakcn importer already exists, so the missing piece is the Gateway CSP allowance rather than a ClawHub/plugin boundary or duplicate cleanup.
