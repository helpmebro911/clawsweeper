## Summary

Keep open. Current main still documents and implements the reported mismatch: the bundled session-memory hook writes slugged daily files, while memory promotion/backfill and dreaming docs describe daily-note ingestion from date-only memory/YYYY-MM-DD.md files. The issue is recent, author is not a maintainer, and it has no protected labels, so there is no conservative close reason.
