## Summary

Current main does not implement Telegram bot avatar sync. The Control UI avatar upload is a browser-local assistant avatar override, and the bundled Telegram plugin exposes send/edit/react/poll/sticker/topic actions but no bot profile-photo update path. The official Telegram Bot API now documents setMyProfilePhoto/removeMyProfilePhoto, so this remains a feasible Telegram plugin enhancement rather than a stale, duplicate, or ClawHub-only close. ([core.telegram.org](https://core.telegram.org/bots/api%20))
