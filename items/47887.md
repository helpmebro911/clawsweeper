## Summary

Keep open. Current main has partial bootstrap-onboarding mitigation, but it does not fully implement the issue’s requested behavior: iOS manual gateway connects still force TLS for non-loopback hosts, and setup-code generation still emits bootstrap-token-only payloads rather than shared token/password credentials. The linked PR #65185 is also still open and explicitly covers only part of the report.
