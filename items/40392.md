## Summary

Keep #40392 open. Current main still uses numbered config backup rotation, while this PR is a focused implementation of the still-open datetime backup request. It is not implemented on main and is not clearly superseded by a canonical maintainer-chosen item, but it is not merge-ready because review discussion identifies migration-time deletion of legacy .bak.N backups.

## Best Possible Solution

Keep the PR open unless a maintainer deliberately chooses another canonical item.
