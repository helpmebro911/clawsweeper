## Summary

Keep #39457 open. Current main still has no post-update lifecycle hook that runs after a successful update and before gateway restart, and the current macOS refresh path can still rewrite the LaunchAgent from the generated install plan rather than preserving custom ProgramArguments wrappers. This is core update/service lifecycle work, not a ClawHub-only plugin request, because plugin hooks run inside the Gateway and do not cover the updater-before-restart moment requested here.
