## Summary

Keep open. Current `main` supports browser profile isolation only at the global `browser.profiles.*` level, not per agent. The reporter’s attempted `agents.list[].browser.userDataDir` shape is still unsupported, and the available workarounds (`profile` per tool call or separate gateway profiles/instances) do not provide enforced per-agent browser-profile isolation.

## Evidence
