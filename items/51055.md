## Summary

Keep #51055 open. Current main still hardcodes 128k/8192 for both Copilot Proxy and LiteLLM onboarding, and the LiteLLM/Copilot Proxy plugin code does not query `/v1/model/info` or parse LiteLLM `model_info` token limits. The provided GitHub context shows PR #50968 is the active proposed fix, but it is open/unmerged, and PR #51238 was closed unmerged, so the bug is not implemented on main.
