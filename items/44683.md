## Summary

Current `main` does not contain the proposed Helm chart or an equivalent in-tree Helm deployment path. It still ships only the Kustomize-based Kubernetes manifests, and a new Kubernetes doc added on April 24, 2026 explicitly says OpenClaw prefers Kustomize and that a Helm chart can be layered on top externally. That means the PR is not implemented on `main`, but it is also coherent and still potentially useful, so none of the allowed close reasons fit.
