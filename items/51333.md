## Summary

Keep open. Current main now has adjacent documentation for hybrid cloud/local model setup, subagent orchestration, and text-only local CLI fallbacks, but it still does not document the exact first-class pattern requested in #51333: a strong cloud orchestrator/tool-user delegating bounded text-only work to local worker models with cookbook-style prompting guidance.
