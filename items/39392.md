## Summary

Keep #39392 open. Current main does not contain the proposed/expected fix for the stale Control UI model selector: the Agents overview panel is rendered directly rather than keyed/remounted per selected agent, and agent switching resets files/skills/tools state but not the shared config form state implicated by the issue comments. The issue has concrete reproduction details and two open linked PRs (#39401 and #39495) that still represent useful pending fixes, so this is not stale or safely closable.
