Keep open. ## Best Possible Solution

Keep the PR open but require a rebase/rewrite against current main: remove the obsolete no-ACP-model-field claim, correct Codex settings references to `config.toml`, and align Claude/Codex settings inheritance wording with the current acpx plugin permission and runtime-option contracts. If maintainers still want this docs clarification, land it as a focused update to `docs/tools/acp-agents.md` and related setup docs after those corrections.
