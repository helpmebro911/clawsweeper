## Summary

Keep open. Current main has an image-attachment input and attach button, but the review did not find strong evidence that the reported Chrome/Windows WebChat button click failure is fixed or covered by a regression test. The current implementation also still limits uploads to image attachments, while the issue describes files more generally.
