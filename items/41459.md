## Summary

Keep #41459 open. Current main has a narrower Discord thread-binding persona path, but it does not implement the requested per-channel `bindings[].nickname` override for normal Discord route bindings or top-level binding config. This should not be closed as implemented or ClawHub: the remaining work is in the bundled Discord binding/outbound behavior or requires a deliberate plugin API seam, not just an already-expressible external plugin.
