## Summary

Keep #48432 open. Current main still applies long-token splitting to the full TUI-rendered text and only preserves URLs, paths, underscore file-like tokens, and credential-like mixed alnum tokens. Hyphen-only package names such as the reported Ubuntu package example are still not protected, and the directly related fix PR #69340 is open/unmerged while #48445 was closed unmerged.
