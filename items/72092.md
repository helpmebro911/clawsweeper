## Summary

Keep this PR open. Current main still rejects empty `aws-sdk` auth in the media image and audio/video paths, while the PR is a focused implementation candidate for the Bedrock credential-chain regression tracked by #72031. The latest PR head also addresses the earlier empty-key rotation gap, but it still needs maintainer review and green validation before merge.

## Best Possible Solution

Keep this PR open for maintainer review and CI. The best path is to validate or iterate on the focused implementation that carries the `aws-sdk` missing-key allowance through media image, MiniMax fallback, and audio/video execution, including the empty-key rotation contract, then merge it to close #72031 if checks pass. The closed unmerged #72066 should not supersede this broader PR.
