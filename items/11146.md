## Summary

Current main still lacks the two requested dashboard UX improvements. The Control UI has only the Cmd/Ctrl+K global command-palette shortcut, the New messages affordance is still button-only, and agent creation exists as lower-level gateway/CLI capability rather than a dashboard New Agent wizard. The existing maintainer-side Codex comment on #11146 already reached the same keep-open conclusion, and the current checkout at 4a3030df9efa72c44bab567a4390ac0c4876a73f still matches it.

## Best Possible Solution

Keep #11146 open or split it into two maintainer-tracked issues. The keyboard work should add a dashboard-level shortcut dispatcher with focus guards and tests for chat/navigation actions. The agent work should add a Control UI New Agent wizard that reuses agents.create, agents.files.*, config, and session APIs, supports workspace/bootstrap customization such as SOUL.md and USER.md, and includes focused UI tests plus user-facing docs.
