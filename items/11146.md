## Summary

Keep #11146 open. Current main still lacks the requested dashboard keyboard shortcuts and the Control UI New Agent wizard; only the existing Cmd/Ctrl+K palette shortcut and lower-level agent creation APIs/CLI flows are present.

## Best Possible Solution

Keep #11146 open or split it into two maintainer-tracked issues. The keyboard work should add a dashboard-level shortcut dispatcher with focus guards and tests for chat/navigation actions. The agent work should add a Control UI New Agent wizard that reuses agents.create, agents.files.*, config, and session APIs, supports workspace/bootstrap customization such as SOUL.md and USER.md, and includes focused UI tests plus user-facing docs.
