## Summary

Keep #11146 open. Current main still has only the existing command-palette shortcut plus partial CLI/backend agent creation support; the requested dashboard keyboard navigation and dashboard New Agent wizard are not implemented.

## Best Possible Solution

Keep #11146 open or split it into two focused tasks. The keyboard task should add a dashboard-level shortcut dispatcher with focus guards for chat/navigation actions. The agent task should add an Agents dashboard creation wizard that uses the existing `agents.create` and agent file APIs, supports workspace/bootstrap customization such as `SOUL.md` and `USER.md`, and optionally creates or spawns the initial session via existing session APIs. Cover both with focused Control UI tests and update user-facing docs.
