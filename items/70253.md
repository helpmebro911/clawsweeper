## Summary

Keep open. Current main still hardcodes Mattermost block-streaming off in the main inbound reply path, so the config-respect part of the report remains live. The bot-to-bot portion is less clear as product intent, but the current Mattermost routing/gating code still lacks an `allowBots` config surface and still drops unmentioned group traffic, so there is not enough evidence to conservatively close this item.
