## Summary

Keep #39659 open. Current main has manual Docker/profile primitives and ClawDock convenience helpers, but it does not implement the requested first-class named Docker instance lifecycle (`openclaw instance create/list/delete/update`) or equivalent managed create/list/update/delete flow for multiple Docker installs. This is not a stale issue, not maintainer-authored, and has no protected labels.

## Evidence

- **Requested surface remains absent from CLI command catalog:** The core command descriptor list includes setup/onboard/config/backup/doctor/dashboard/reset/uninstall/message/mcp/agent/agents/status/health/sessions/tasks, but no `instance` command or Docker instance lifecycle command.
  - file: [src/cli/program/core-command-descriptors.ts:6](https://github.com/openclaw/openclaw/blob/81a41fe5bee350c6a035bfcc458eb8be380afe2c/src/cli/program/core-command-descriptors.ts#L6)
