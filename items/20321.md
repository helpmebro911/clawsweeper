## Summary

Current `main` does not implement the requested Telegram user-membership group policy. The shipped config/type/schema surface still only supports `groupPolicy` values `open | disabled | allowlist`, and the Telegram docs still require either explicit `channels.telegram.groups` entries or an existing sender allowlist rather than automatic full-member verification for new groups.
