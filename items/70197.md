## Summary

Keep open. Current `main` still starts the default-enabled Bonjour discovery plugin on gateway startup, that plugin still resolves to `@homebridge/ciao` 1.3.6, and the repo has no local patch or Windows-specific mitigation for the reported console-flash path. The reporter's upstream `homebridge/ciao#64` link and local workaround still fit the current code layout, even though the advertising code has since been moved into the bundled `bonjour` plugin.
