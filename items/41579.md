## Summary

Keep open. Current main does not implement a permanent Mission Control/Control UI File Gallery left-nav entry, and the existing file UI only edits a small allowlisted set of agent bootstrap files. A ClawHub close is not well-supported because the current plugin API exposes gateway/agent capabilities and HTTP routes, but not a Control UI left-nav or panel extension point for this requested in-app surface.
