## Summary

Keep open. Current `main` at `608c08fc54da2f86338366f8eeafd7e2a02713aa` still appears to have the Feishu bot reply-dispatch path bug this PR fixes: the bot never computes agent-scoped `mediaLocalRoots`, `createFeishuReplyDispatcher` does not accept/retain them, and its `sendMediaFeishu` call omits them. That means local media generated in agent workspaces can still miss the allowed-roots context required by Feishu media loading.
