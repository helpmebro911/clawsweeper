## Summary

Keep open. Current main has partial session management, including a Sessions table and a chat session selector, plus a gateway `sessions.create` API, but it does not implement the requested ChatGPT-style Control UI sidebar or the requested `+` behavior. The chat `+` still sends `/new` to the current session, and the UI has no `sessions.create` call for an independent create-and-switch flow. The KeepMyClaw and Jupyter comments are useful workarounds/context, but they do not supersede the core Control UI request.
