## Summary

Current `main` still throws session lock errors with the literal text `session file locked (timeout ...)`, classifies generic `timeout` text as a model-fallback timeout, and `runWithModelFallback()` continues across remaining candidates. The reported cascade therefore still appears valid on `cc0f3e0e40364023cc3a39227a411898ee68ce3b`, including the `infer model run` trigger path that reuses `agentCommand()`/agent runtime.
