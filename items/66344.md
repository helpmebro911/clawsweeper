Related PR #66568 is closed and unmerged in favor of this PR, and broader PR #63460 is still open/unmerged, so neither is shipped superseding evidence. Security review found no CI, dependency, lockfile, script, permission, secrets, download, or release-surface changes in this PR.

## Best Possible Solution

The best path is to land a table-span implementation wired into `src/agents/pi-embedded-block-chunker.ts`, add the missing `subscribeEmbeddedPiSession` integration coverage requested by #66605, and resolve the remaining parser/chunker edge-case review comments. If #63460 lands first and fully covers this exact streaming block chunker behavior, revisit this PR as superseded by that landed SHA.
