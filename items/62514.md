## Summary

Keep #62514 open. Current main still exposes the `image` tool to vision-capable runs and still ignores HTTP(S) image URLs during native prompt-image detection, so the reported class of weak-model tool misuse and mixed-source image loss is not implemented/fixed on main. The linked #63710 is an open, unmerged PR tracking part of the remaining work; #62523 was closed unmerged.
