## Summary

Keep open. Current main has important phase-aware suppression for `commentary` assistant output, but it does not implement the requested generic/per-agent or per-channel suppression of any unphased text that precedes tool calls. The remaining #50985 ask is still meaningful for providers or paths that do not mark pre-tool narration as `commentary`.
