## Summary

Keep open. Current `main` still preserves the active-tab exception that this PR tries to remove, so the reported sidebar-collapse behavior is not clearly implemented on `main`. The sidebar code was also rewritten after this PR was opened, which means the old one-line patch likely needs to be rebased onto the newer `nav-section` implementation instead of being closed as already fixed.
