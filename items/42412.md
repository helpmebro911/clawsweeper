## Summary

Keep open. Current `main` does not implement skill-usage statistics: the `openclaw skills` CLI only supports search/install/update/list/info/check, the skill status/report types expose eligibility/setup metadata but no invocation counters or last-used timestamps, and existing session analytics summarize tool/model usage rather than skill usage.
