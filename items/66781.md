## Summary

Keep open. Current main narrowed the Active Memory recall run to `memory_search`/`memory_get`, so the issue's concrete reproduction is partly outdated, but the broader least-privilege-by-default request is not implemented: current docs and tests still describe/verify broad default tool access for sub-agents unless explicit allow/deny policy is configured.
