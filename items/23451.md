## Summary

Keep open. Current main has exec approvals and plugin-driven before_tool_call approvals, but it does not implement the requested built-in configurable tool-level confirmation gate, and the existing plugin hook surface does not clearly cover the text-aware same-response auto case before the first tool executes.
