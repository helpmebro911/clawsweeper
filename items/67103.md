## Summary

Keep this PR open. Current main still does not implement Control UI LaTeX/KaTeX/MathJax rendering, #42840 remains the canonical open feature request, and this PR is a plausible implementation candidate, but the PR head has merge-blocking functional and security regressions that need rework before it can land.

## Best Possible Solution

Keep this PR open as a rework candidate for #42840. The best path is to update it against current main, preserve the existing markdown cache and DOMPurify link-hardening hooks, add robust delimiter parsing that avoids currency and code-span/code-fence false positives, narrow any KaTeX-required style allowance, include the lockfile/dependency review for KaTeX, and add focused tests for inline math, display math, invalid LaTeX, mixed markdown, currency text, and backtick-delimited code. If the author does not continue, a successor PR can reuse the same shared Control UI markdown-renderer surface.
