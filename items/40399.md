## Summary

Keep open. Current `main` has error-based model failover and pre-run plugin model overrides, but not the requested quality-based adaptive routing flow. There is no `adaptiveRouting` config surface, no `adaptive-routing` command/files on `main`, and the related PR #30185 was closed unmerged as stale.
