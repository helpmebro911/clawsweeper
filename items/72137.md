## Summary

Keep PR #72137 open. Current main still has the missing-vs-empty-object diff behavior that produces phantom `plugins.entries.*.config` changes, and those paths still classify as gateway-restart changes. The PR remains a focused fix with regression coverage for #72061.

## Best Possible Solution

Keep PR #72137 open for maintainer review and normal landing, or land an equivalent narrow fix on main. The best implementation is a targeted `diffConfigPaths` change that collapses missing vs empty plain-object normalization artifacts, preserves explicit empty-array diffs, and keeps regression tests for the #72061 skill-toggle scenario.
