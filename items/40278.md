## Summary

Current `main` already covers the reported Anthropic runtime miss: bundled provider plugins are now force-compatible with restrictive `plugins.allow` on the runtime provider path, and Anthropic provider/model configs are normalized to `api: "anthropic-messages"` with regression coverage.

## Evidence

- **Anthropic is still a bundled provider enabled by default:** The bundled Anthropic plugin manifest declares provider id `anthropic` and `enabledByDefault: true`, so current main expects it to be active without extra manual enablement.
  - file: [extensions/anthropic/openclaw.plugin.json:2](https://github.com/openclaw/openclaw/blob/d4e93e791bc5093ea14666bf4d8202b45a5132b4/extensions/anthropic/openclaw.plugin.json#L2)
  - command: `nl -ba extensions/anthropic/openclaw.plugin.json | sed -n '1,80p'`
- **Runtime provider loading now injects bundled-provider allowlist compatibility:** The runtime provider load path calls `resolveBundledPluginCompatibleActivationInputs` with `allowlist: params.bundledProviderAllowlistCompat` before resolving enabled provider plugin ids, which is the exact guard needed when a bundled provider was omitted from a restrictive `plugins.allow`.
  - file: [src/plugins/providers.runtime.ts:193](https://github.com/openclaw/openclaw/blob/d4e93e791bc5093ea14666bf4d8202b45a5132b4/src/plugins/providers.runtime.ts#L193)
  - command: `nl -ba src/plugins/providers.runtime.ts | sed -n '150,240p'`
- **Provider hook/runtime resolution enables bundled-provider allowlist compat by default:** `resolveProviderPluginsForHooks` passes `bundledProviderAllowlistCompat: true` and `bundledProviderVitestCompat: true` into both the in-flight check and the actual runtime provider resolution path, so request-time provider hooks no longer bypass the compatibility layer.
  - file: [src/plugins/provider-hook-runtime.ts:120](https://github.com/openclaw/openclaw/blob/d4e93e791bc5093ea14666bf4d8202b45a5132b4/src/plugins/provider-hook-runtime.ts#L120)

  - file: [src/plugins/bundled-compat.ts:5](https://github.com/openclaw/openclaw/blob/d4e93e791bc5093ea14666bf4d8202b45a5132b4/src/plugins/bundled-compat.ts#L5)
  - command: `nl -ba src/plugins/bundled-compat.ts | sed -n '1,120p'`
