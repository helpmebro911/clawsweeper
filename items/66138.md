## Summary

Keep open. Current main has meaningful partial UX work for configuration, including Quick Settings, schema-backed config editing, and ClawHub skill search/install in the Skills panel, but it still does not implement the broader guided dashboard and secure dashboard-based ClawHub plugin install/configuration experience requested here.

## Best Possible Solution

Keep this open as a product/UI umbrella or split it into narrower maintainer-owned issues: guided Control UI reconfiguration flows, contextual help for complex config sections, secure dashboard ClawHub plugin discovery/install/configuration, and any additive plugin manifest/setup/config-schema API needed for third-party plugins to expose safe dashboard configuration.
