## Summary

Keep open. Current main has runtime `toolsAllow` support in the embedded runner and cron-related plumbing, but `sessions_spawn` still does not expose a per-run tool allowlist or forward one through `spawnSubagentDirect`. The related merged work (#62264 for `lightContext`, #62675 for cron `toolsAllow`) is useful adjacent context but does not implement this requested spawn-time API.
