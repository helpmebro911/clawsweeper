## Summary

Keep open. Current main still does not document the exact `model.usage` JSON field names requested in #49046. The public logging docs only describe the event at a category level, while the actual field contract is only discoverable from TypeScript and emitter code. This remains an actionable docs/API-contract clarification rather than an obsolete or ClawHub-only request.
