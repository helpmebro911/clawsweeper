## Summary

Keep open. Current `main` does not implement the PR's core reload-semantics split: explicit reloads still call plain `loadConfig(...)`, `applyConfigSnapshot(...)` still has no discard/preserve option, and the bundled Nostr/import guard changes are also absent.
