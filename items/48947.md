## Summary

Keep open. Current main has useful partial fixes for live tool visibility, abort handling, and loop detection, but it does not fully satisfy the reported default behavior: generic tool-loop protection is still opt-in, there is no documented `tools.maxToolCalls` agent config, and the issue’s broad default-runaway/no-feedback complaint remains salvageable.
