## Summary

Keep open. Current `main` still builds a debounced Discord synthetic message by object-spreading the Carbon `Message` instance without materializing `mentionedUsers`, `mentionedRoles`, or `mentionedEveryone`, while the current preflight path still relies on those getter-backed fields for mention gating. The PR’s core fix remains relevant, but it now needs retargeting to `extensions/discord/...` after the Discord monitor move on 2026-04-24.

That is the exact failure mode described in the PR.
