## Summary

Keep open. Current main still duplicates the native subagent task text: the task is embedded in the subagent system prompt and also sent as the first child user message. The linked ClawHub skill is only a workaround, and open PRs #72028 and #72053 are active implementation candidates with closing syntax, so this issue should remain open until a canonical fix merges.

## Best Possible Solution

Keep this issue open and review one canonical fix, likely #72053 or #72028. The fix should single-source the native subagent task text, preserve multiline task formatting, and add a regression test that inspects the child system prompt and first gateway user message together.
