## Summary

Keep #39317 open. The related fix PR #39460 was closed unmerged, and current main still has no `sandbox_denied`/`sandboxDenied` detection path. On exit code 0, the CLI runner parses stdout and returns a successful run result, so the reported semantic-success bug still appears actionable.
