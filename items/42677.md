## Summary

Keep open. Current `main` still ships bundled `session-memory` as an explicit `/new`/`/reset` hook only; automatic stale-session rollover now preserves/archive the prior session transcript, but no internal rollover hook is emitted for bundled memory persistence. The PR is therefore not obsolete, and the discussion also surfaces a plausible ACP double-hook edge case that still merits maintainer review rather than cleanup-closing.
