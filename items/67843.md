## Summary

Keep this PR open. The checked-out main still omits `agentDir` and `workspaceDir` from the provider stream wrapper context, while the OpenAI/Codex wrapper and Codex auth lookup need `agentDir` to detect persisted `openai-codex` auth. The PR is a focused implementation candidate with a regression test, and the related #65909 remains open.

## Best Possible Solution

Review and land this narrow PR, or cherry-pick the same forwarding of `agentDir` and `workspaceDir` into `wrapProviderStreamFn` plus the regression test. After applying it, run the targeted `src/agents/pi-embedded-runner-extraparams.test.ts` coverage and the normal changed TypeScript gate.
