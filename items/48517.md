## Summary

Keep open. Current main has subagent and sandbox primitives, but the plugin hook/runtime contracts still do not let a plugin request quarantined execution of an arbitrary operation or receive standardized trust/provenance metadata back. This remains an unimplemented core/plugin-SDK security API proposal, not ClawHub cleanup or stale insufficient info.
