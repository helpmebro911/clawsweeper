## Summary

Keep open. The request is for missing core plugin hook runtime semantics: per-hook timeout/fallback/critical behavior and an async context-provider pattern. Current main has typed plugin hooks and prompt context injection, but the prompt-modifying hooks still run as awaited sequential work with no timeout/race/fallback configuration, and the public SDK/config surfaces do not expose the requested controls.
