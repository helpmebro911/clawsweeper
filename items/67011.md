## Summary

Keep this PR open. Current main still has the traversal-wide `WeakSet` behavior that misclassifies shared acyclic object references as `[Circular]`, and cache tracing still uses that sanitizer. The PR is a plausible focused fix with a review-response memoization update, but it needs maintainer review and cleanup against current main before landing.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to land the active-recursion-path plus `WeakMap` memo fix in `src/agents/payload-redaction.ts`, add focused regression coverage for shared sibling references and real cycles, update the test patch for current exported sanitizer names and the `QueuedFileWriter.flush` contract, and separately decide whether cache-trace digest `stableStringify()` needs the same semantics.
