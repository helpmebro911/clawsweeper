## Summary

Keep open. Current main has mitigations for two adjacent failure modes, but I do not see high-confidence evidence that #41304’s broader third-party plugin/tool invocation problem is fully implemented or obsolete. The discussion reports AgentMail, GitHub, weather, and Composio/custom plugin cases across multiple models and installs; the reporter later got some skills working by switching from a small local model to Gemini, which points to model/provider behavior but does not prove the core/plugin path is fixed.
