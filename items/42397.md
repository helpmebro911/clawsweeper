## Summary

Keep open. Current `main` still applies a single `heartbeat.model` override to the whole heartbeat run, and `HEARTBEAT.md` task blocks only support `name`, `interval`, and `prompt`. Due heartbeat tasks are batched into one prompt and executed in one run, so there is no per-task model override in `HEARTBEAT.md` today.
