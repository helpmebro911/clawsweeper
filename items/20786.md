## Summary

Keep open. Current `main` only partially matches the report: Telegram business update types are already allowed via `extensions/telegram/src/allowed-updates.ts`, and that list is passed to webhook/polling setup in `extensions/telegram/src/webhook.ts:423` and `extensions/telegram/src/monitor.ts:34`. But the Telegram runtime still does not implement Telegram Business handling: `extensions/telegram/src/bot-handlers.runtime.ts:1940` and `extensions/telegram/src/bot-handlers.runtime.ts:1979` register `message` and `channel_post` handlers only, `extensions/telegram/src/bot-updates.ts:17` does not model business update fields for dedupe, and repo-wide search found no `business_connection_id` handling for outbound replies. This means the feature request is not implemented on `main`, even if the issue’s original `allowed_updates` theory is now partly outdated.

## Evidence

- **Business update types are already in the allowed update set:** `business_connection`, `business_message`, `edited_business_message`, and `deleted_business_messages` are present in the fallback/default update arrays, so current `resolveTelegramAllowedUpdates()` already preserves them.
  - file: [extensions/telegram/src/allowed-updates.ts:3](https://github.com/openclaw/openclaw/blob/a958b6e72328d1d922a6d133b1a70e866faf4013/extensions/telegram/src/allowed-updates.ts#L3)
  - sha: [a958b6e72328](https://github.com/openclaw/openclaw/commit/a958b6e72328d1d922a6d133b1a70e866faf4013)
- **Webhook and polling both use the resolved allowed update set:** Telegram setup paths pass `resolveTelegramAllowedUpdates()` through to Telegram, so current `main` is not missing these update names at registration time.
  - file: [extensions/telegram/src/webhook.ts:423](https://github.com/openclaw/openclaw/blob/a958b6e72328d1d922a6d133b1a70e866faf4013/extensions/telegram/src/webhook.ts#L423)

A repo search for `business_message|business_connection|edited_business_message|deleted_business_messages` under `extensions/telegram/src` only returned `allowed-updates.ts`.
  - file: [extensions/telegram/src/bot-handlers.runtime.ts:1940](https://github.com/openclaw/openclaw/blob/a958b6e72328d1d922a6d133b1a70e866faf4013/extensions/telegram/src/bot-handlers.runtime.ts#L1940)
