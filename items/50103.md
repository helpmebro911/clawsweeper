## Summary

Keep issue #50103 open. Current main does not implement `agents.defaults.systemPrompt`, `agents.defaults.rules`, or per-agent `systemPrompt`/`rules`; it only has `systemPromptOverride`, which replaces the assembled prompt rather than appending config-injected rules after bootstrap files. The linked implementation PR #52038 is closed/unmerged, and the remaining capability is not clearly superseded by another canonical open item in the provided context.
