## Summary

Keep open. Current `main` at `608c08fc54da2f86338366f8eeafd7e2a02713aa` still does not contain the PR’s installer changes: gum raw-mode fallback only checks stderr for `setrawmode`, and the npm `@next` retry still happens before `ensure_openclaw_bin_link`, so the reported behavior is not implemented on `main` and remains plausible.
