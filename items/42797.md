## Summary

Keep open. Current `main` still does not narrow gateway `models.list` to authenticated/usable providers; it returns the allowlisted catalog or the full catalog. That endpoint is still the source for browser/chat model pickers, so the requested gateway behavior is not already implemented on `main`.

## Evidence
