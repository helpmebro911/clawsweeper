## Summary

Keep open. Current `main` still uses glob-only `SKILL.md` watcher targets with `chokidar` 5 and only refreshes on `add`/`change`/`unlink`, so the macOS whole-directory deletion case described in the PR is not implemented on `main`. The PR remains relevant, and the review discussion also surfaced follow-up concerns about the proposed directory-watch scope and unrelated QA/script changes.
