## Summary

Keep open. Current main still suppresses text-only tool-result summaries in ordinary group chats, and I did not find a channel/default or Telegram/WhatsApp group config option that lets a trusted group opt back in. The related PR #44027 is open and unmerged, and it proposes honoring `/verbose` rather than adding the requested schema-backed config override.

## Best Possible Solution

Keep this issue open. The best path is a narrow, typed, schema-backed opt-in such as `channels.defaults.allowVerboseToolResults` plus provider/group/topic overrides, defaulting to false. Feed that policy into `dispatch-from-config`'s `shouldSendToolSummaries` and verbose-progress gates, add regression tests proving ordinary groups stay silent by default but send text summaries when explicitly enabled, preserve forum/media/exec-approval behavior, and update public docs with a trusted-group warning while coordinating with #44027.
