## Summary

Current `main` does not implement this PR. Inline skill `dispatch.kind === "tool"` still builds its tool list with `createOpenClawTools()` and then applies only `applyOwnerOnlyToolPolicy()`, so it does not go through the full coding-tool policy/profile/group/subagent filtering path used by normal coding sessions.
