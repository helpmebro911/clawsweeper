## Summary

Keep open. Current `main` still returns the updated `skills.entries.<skillKey>` object directly from `skills.update`, so `apiKey` and `env` secrets would still be echoed in the gateway response. I found no merged fix on `main`, and the existing test coverage only checks API-key normalization, not response redaction.
