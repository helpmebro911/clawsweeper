## Summary

Keep open. Current `main` does not contain the proposed lowercase normalization in the system-events queue, and the heartbeat/session-key path still canonicalizes agent session keys to lowercase elsewhere. That means the PR is not implemented on `main`, and there is no strong evidence the reported mixed-case channel-session bug is obsolete.

## Evidence
