## Summary

Keep this PR open. Current main still lacks shared default redaction for Basic auth headers, standalone JWTs, and x-pomerium-jwt-assertion values on the logging path used by logs.tail, and the linked bug remains open. The PR is a relevant focused fix, but it needs a small amendment before merge.

## Best Possible Solution

Keep this PR open as the implementation candidate for #66832. Ask for a small amendment: remove or replace the Basic-auth trailing word boundary, add focused regression tests for padded Basic auth, standalone JWTs without a Bearer prefix, x-pomerium-jwt-assertion, and the logs.tail/redactSensitiveLines path, then run the targeted logging tests before merge.
