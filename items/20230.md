## Summary

Keep #20230 open. Current main has shipped the related active assistant name/avatar display and distinct user-vs-assistant bubble styling, but the central remaining request is still missing: no per-agent chat bubble color contract and no per-message/per-agent assistant attribution for mixed multi-agent chat history.

## Best Possible Solution

Keep the issue open until maintainers either implement or explicitly reject per-agent chat styling and per-message assistant attribution. A complete fix likely belongs in the Control UI and shared chat/session contracts: persist assistant identity per message or group, add a validated optional color/styling field or render hook, render assistant-specific CSS variables/classes, and add focused tests for mixed-agent and historical-session attribution.
