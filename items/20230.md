## Summary

Keep open. Current `main` already gives user messages a distinct bubble style and shows the active assistant name in assistant message footers, so part of the report is obsolete. But the multi-agent part is still missing: the UI keeps only one session-wide assistant identity, only user messages retain `senderLabel`, and the surfaced UI config exposes a global seam color plus one assistant identity, not per-agent bubble colors or per-message assistant labels.
