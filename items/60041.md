## Summary

Keep open. Current `main` still contains the reported Twitch restart-loop bug: the Twitch gateway startup path returns immediately after registering handlers, and the channel manager treats that normal return as a crashed channel and schedules auto-restarts. The config/schema mismatch is also still present because `clientId` is optional in the Twitch schema but required at runtime.
