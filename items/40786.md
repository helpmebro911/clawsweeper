## Summary

Current `main` does not implement `.gitignore`-style excludes for `openclaw backup create`, and the provided GitHub context shows open PR #44288 explicitly targets this issue with GitHub closing syntax. Keep this issue open while that PR or a narrower replacement is reviewed; the linked Keep My Claw plugin is a useful workaround, but it does not remove the native backup CLI gap.

## Best Possible Solution

Keep this issue open and use it as the user-facing tracker for native backup exclude support. Review #44288, or replace it with a smaller implementation, but the final core change should add CLI flags in `src/cli/program/register.backup.ts`, typed options and filtering in `src/infra/backup-create.ts`/`src/commands/backup-shared.ts`, regression coverage in the backup CLI/archive tests, and docs updates under `docs/cli/backup.md`.
