## Summary

Keep open. Current `main` at `c2d82b87eee7a86bbd0e851169e0ca5559c44fe6` still has the reported mismatch: `openclaw doctor` warns about missing recent session transcripts but suggests a cleanup preview command without `--fix-missing`, while `sessions cleanup --dry-run` only counts missing transcript pruning when that flag is set.

## Best Possible Solution

Land a narrow commands fix in `src/commands/doctor-state-integrity.ts` so the missing-transcript `Preview cleanup impact` command includes `--dry-run --fix-missing`, then update `src/commands/doctor-state-integrity.test.ts` to assert that exact hint. Keep the explicit `sessions cleanup --fix-missing` behavior flag-gated unless maintainers choose a broader CLI behavior change.
