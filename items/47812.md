## Summary

Keep #47812 open. Current main has partial session hygiene support: manual `/new`, daily/idle resets, and a Control UI high-context notice with a compact action. It does not implement the requested topic-change detection, a "start a new session?" reminder on topic switch, or automatic session switching at very high context usage, and the issue is recent enough that stale closure does not apply.
