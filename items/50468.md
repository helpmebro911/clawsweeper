## Summary

Keep open. Current main still constructs chokidar watch targets as `SKILL.md` and `*/SKILL.md` glob strings, and the colocated test still asserts those glob targets. With chokidar locked at 5.0.0, the reported incompatibility has not been fixed on main; the existing session refresh path still depends on the watcher bumping the skills snapshot version.
