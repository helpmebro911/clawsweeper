## Summary

Keep open. Current main still lacks the requested `openclaw timeline` command, `/timeline` slash command, TUI or Control UI timeline panel, and JSON-exportable cross-agent activity timeline. Existing trajectory export, subagent status, usage timelines, live tool cards, and gateway agent events are adjacent but do not satisfy the proposed interactive agent activity timeline. Related #60814 is an open draft PR for a web activity dashboard and only explicitly closes #60810, so it is an implementation candidate rather than a reason to close this issue now.

## Best Possible Solution

Keep this issue open until maintainers explicitly own the full scope. The best path is either to broaden/re-scope #60814 so it covers the CLI command, `/timeline`, TUI/Control UI timeline surface, event persistence/export contract, and tests/docs, or to split this into a dedicated core/gateway activity-timeline implementation over existing agent events, subagent run metadata, sessions, tasks, and trajectory data.
