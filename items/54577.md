## Summary

Keep this issue open. Current main has adjacent observability pieces such as task lifecycle timestamps, session change events, subagent metadata, and per-session trajectory export, but it still does not ship the requested built-in agent activity timeline surface: no `openclaw timeline` CLI command, no `/timeline` chat command, no Control UI/TUI timeline route, and no persisted or JSON-exportable cross-agent lifecycle timeline. Related #60814 is an open draft Control UI dashboard candidate, not a merged implementation of this full scope.

## Best Possible Solution

Keep this open until maintainers scope and ship the built-in agent activity timeline product. A good path is to land or replace #60814 with a narrower core/gateway implementation that adds durable lifecycle event capture, JSON export, `openclaw timeline` filters, `/timeline`, a TUI or Control UI view, docs, and focused tests over existing session/task/subagent/trajectory data.
