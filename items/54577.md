## Summary

Keep this issue open. Current main still does not ship the requested built-in agent activity timeline: no `openclaw timeline` CLI command, no `/timeline` chat command, no Control UI/TUI timeline surface, and no JSON-exportable cross-agent lifecycle timeline. Existing agent events, session lifecycle metadata, usage timelines, subagent/task status, and trajectory export are useful adjacent observability pieces but do not satisfy the requested product surface. Related #60814 is an open draft Control UI dashboard candidate and remains only a partial implementation path.

## Best Possible Solution

Keep this issue open until maintainers scope and ship the agent activity timeline product. The likely path is to either broaden and land #60814 with the missing CLI command, `/timeline` command, TUI/Control UI surface, lifecycle event persistence/export contract, filters, tests, and docs, or split a narrower core/gateway activity-timeline implementation over the existing session lifecycle, subagent, task, agent-event, and trajectory data. This should be treated as core observability UX rather than ClawHub-only work because the request is for built-in CLI/chat/TUI/Control UI surfaces over OpenClaw's own agent lifecycle.
