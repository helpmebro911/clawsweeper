## Summary

Current `main` contains related cache-boundary infrastructure, but it does not implement this PR’s raw prompt reordering. `buildAgentSystemPrompt()` still emits dynamic `## Messaging` and `## Reactions` content before `# Project Context`, and multiple transport paths strip the internal cache-boundary marker before sending the prompt, so the large static project context still follows dynamic content in shipped prompt text.
