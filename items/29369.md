## Summary

Keep #29369 open. Current main still does not expose or honor a spawn-time `tools` allowlist on `sessions_spawn`; tool policy remains configured through global/provider/agent/group/sandbox/subagent config rather than per call.

## Best Possible Solution

Keep #29369 open until maintainers design and implement a spawn-time tool policy for `sessions_spawn`, covering the TypeBox schema, native and ACP spawn parameter types, child session/run metadata if needed, effective tool-policy resolution, docs, and tests. The design should document precedence against `tools.allow`, provider policies, `agents.list[].tools`, group policy, sandbox policy, `tools.subagents.tools`, and deny-wins behavior.
