## Summary

Current `main` does not implement this PR’s `tools.sessions.ownedAcp.enabled` behavior. Session-tool visibility still only supports `self|tree|agent|all`, cross-agent access is still blocked unless visibility is `all`, and current tests still assert tree-mode rejection for parent/ACP-adjacent access. The PR remains actionable rather than obsolete.
