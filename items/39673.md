## Summary

Keep #39673 open. Current main has systemd user-bus hardening, but the direct `openclaw gateway install` path can still fail with the same `systemctl --user unavailable: Failed to connect to bus` class when the user systemd scope remains unavailable. The stale close reason does not apply yet, and the provided related #14 is unrelated to gateway/systemd install.
