## Summary

Keep this issue open. Current main implements the global `agents.defaults.thinkingDefault` path, but the requested global `agents.defaults.reasoningDefault` is still absent from the defaults type/schema, generated schema/help, runtime resolution, and status/current-level helpers. The current supported reasoning default is per-agent (`agents.list[].reasoningDefault`) only. Open PR #39115 is the likely implementation candidate; open PR #71817 is only a partial per-channel/per-agent fix.

## Best Possible Solution

Keep this issue open until maintainers either land a complete global `agents.defaults.reasoningDefault` implementation across config types, strict schema, generated schema/help, directive resolution, current-level/status surfaces, docs, and regression tests, or explicitly decide reasoning visibility should remain per-agent/session-only and close with that product decision. Review #39115 as the likely complete candidate; treat #71817 as a useful but partial per-channel/per-agent fix.
