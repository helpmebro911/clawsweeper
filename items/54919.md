## Summary

Keep this issue open. Current main supports the global `agents.defaults.thinkingDefault` path, but the body also asks for global `agents.defaults.reasoningDefault`, which is still missing from the defaults contract and runtime resolution. The currently supported reasoning default is per-agent (`agents.list[].reasoningDefault`) only; #39115 is the likely implementation candidate and #71817 is a partial channel/per-agent fix.

## Best Possible Solution

Keep this issue open until maintainers either land a complete global `agents.defaults.reasoningDefault` implementation across the defaults type, strict Zod schema, generated schema/help/labels, runtime directive resolution, current-level/status surfaces, docs, and regression tests, or explicitly decide that reasoning visibility should remain per-agent/session-only. Review #39115 as the likely complete implementation candidate; treat #71817 as useful but partial channel/per-agent work.
