## Summary

Current main does not implement the requested /agent workspaceAccess:"ro" read-tool guard mapping. The guard still maps only containerWorkdir, the sandboxed read-tool wiring only passes sandbox.containerWorkdir, and current regression coverage explicitly expects /agent mounted paths to be rejected under the workspace-only guard. Related PRs #39525, #39571, and #41033 were closed unmerged; merged PR #44307 explicitly scoped out #39497, so it does not supersede this issue.
