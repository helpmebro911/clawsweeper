## Summary

Keep #11057 open. Current main still does not provide native 1M Opus 4.6 support for Claude Max/browser-session auth: 1M remains config-gated with `params.context1m`, OAuth/setup-token credentials strip the 1M beta header, docs describe fallback to the standard window, and tests assert that limitation.

## Best Possible Solution

Keep this issue open until OpenClaw either implements native Claude Max/browser-session 1M support for Opus/Sonnet 4.6 in the built-in Anthropic provider, documents a maintainer-approved API-key/custom-provider-only decision, or designates a narrower canonical follow-up. The likely path spans `extensions/anthropic/stream-wrappers.ts`, `src/agents/context.ts`, model listing/status behavior, Anthropic provider docs, and OAuth/API-key regression tests.
