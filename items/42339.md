## Close Comment

Codex review: closing this as `clawhub`. Current project direction is explicit that core should stay lean, optional capability should usually ship as plugins, and new skills should go to ClawHub first (`VISION.md:52-70`). Main already has the extension seams needed for this feature: plugin slash commands via `api.registerCommand()` (`docs/plugins/sdk-overview.md:85-89`, `docs/tools/slash-commands.md:80-85`), plugin command auth (`src/plugins/commands.ts:192-199`), and plugin-owned config/distribution (`docs/plugins/sdk-entrypoints.md:44-70`, `docs/tools/plugin.md:11-15,43-57`). This PR instead adds a vendor-specific Resemble integration directly to core command/config surfaces, so it is better handled as a ClawHub/community plugin rather than merged into OpenClaw core.
