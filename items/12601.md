## Summary

Keep #12601 open. Current main at 4a3030df9efa72c44bab567a4390ac0c4876a73f still does not expose Slack Canvas content reading in the bundled Slack plugin: action discovery, dispatch, runtime handling, scopes, docs, and tests cover existing message/file/pin/member/emoji actions only. The related implementation PR #18752 was closed unmerged, and the maintainer comment already noted this should remain open for Slack plugin work.

## Best Possible Solution

Keep #12601 open for the bundled Slack plugin. A complete fix should add a Slack Canvas read action or dedicated Canvas tool under extensions/slack, expose it through message action discovery and dispatch, add canvases:read to setup/docs/config guidance, call Slack Canvas APIs through the existing token/account model, return bounded markdown or section data, and cover discovery, dispatch, runtime behavior, docs, tests, and live Slack validation. PR #18752 remains useful prior art but is not a merged canonical fix.
