## Summary

Keep open. Current main has a narrow Anthropic thinking-block recovery path, but it does not implement the requested framework-level guarantee to never expose raw LLM/API error details on user messaging channels. The issue discussion is still actionable: the reporter described repeated WhatsApp and Telegram leaks after the 2026.3.7 update, and current tests still assert user-visible delivery of sanitized-but-specific exception text.
