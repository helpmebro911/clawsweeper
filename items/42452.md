## Summary

Keep this PR open. Current main still does not expose provider rate-limit headroom in `openclaw models status --probe`, and this PR remains a concrete implementation candidate for the linked feature request #42401. It is not merge-ready: the patch needs a rebase onto current main, docs/tests, routed CLI wiring, and a safer provider transport design before maintainer review.

## Best Possible Solution

Keep this PR open as the implementation candidate, but require a rebase onto current main. The best version should preserve the current lazy probe import guard, update Commander and routed CLI parsing, document `--rate-limits` at https://docs.openclaw.ai/cli/models, and capture rate-limit headers through the existing provider runtime/transport path or a narrow generic seam so provider config, auth, headers, transport policy, and plugin-owned behavior are honored. Add focused tests for CLI routing, JSON/table output, endpoint/config normalization, and transport safety before maintainer review.

## Risks / Open Questions

- The PR’s direct provider `fetch` path can bypass normalized provider config lookup, plugin transport normalization, provider-owned usage/auth hooks, configured headers/authHeader, and request transport policy; with a custom provider shape it could probe the wrong endpoint with a real API key.
- The patch does not update the routed `models status` argument parser; current `parseModelsStatusRouteArgs` has no `rateLimits` field, so the fast routed path may drop the new flag even if Commander registration is added.
- The PR has no docs update for the opt-in extra provider request, its billable/quota impact, or the JSON `rateLimit` contract.
- The PR tests cover parsing/formatting helpers but not CLI flag forwarding, routed args, JSON/table output, provider base URL/config normalization, transport hook behavior, or preserving the current lazy embedded-runner import guard.
- The `.gitignore` change for `openclaw-*.log` is unrelated to the feature and should be removed or separately justified.
