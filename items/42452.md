## Summary

Current `main` does not implement this PR's `--rate-limits` probe feature, and the shipped `models status --probe` surface still lacks the CLI flag, option plumbing, probe result fields, output columns, docs, and tests described in PR #42452. That makes `implemented_on_main` unsupported, and there is no basis for another close reason.
