Keep this issue open. Current main still does not support client-provided Chat Completions tools, and PR #66278 is an open, unmerged implementation candidate that explicitly closes this issue.

## Best Possible Solution

Keep this issue open and use PR #66278 as the active implementation candidate. The maintainer path is to review, revise, and land that PR or an equivalent change in src/gateway/openai-http.ts with tests in src/gateway/openai-http.test.ts covering tools, tool_choice, non-stream message.tool_calls, streaming delta.tool_calls, and role:"tool" follow-up messages with tool_call_id.
