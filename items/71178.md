## Summary

Keep #71178 open. Current main still lacks the `update.run` continuation producer path that the maintainer identified, while the restart continuation consumer and restart action path already exist. The linked PR #71573 is the canonical fix attempt in the provided context, but it is open/unmerged there and the current checkout still has tests asserting that `update.run` writes no continuation.

## Best Possible Solution

Keep #71178 open until the `update.run` continuation fix lands on main, then have maintainers decide whether the remaining graceful-drain and post-reset bridge behaviors stay in this issue or move to narrower follow-ups. The concrete code path remains `src/gateway/protocol/schema/config.ts`, `src/gateway/server-methods/restart-request.ts`, `src/gateway/server-methods/update.ts`, and `src/agents/tools/gateway-tool.ts`, with tests updated away from the current `continuation`-is-undefined assertions.
