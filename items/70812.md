## Summary

Keep open. Current main still has the reported behavior: the session artifact helper treats any non-archived, non-checkpoint `.jsonl` file as a primary transcript, so `*.trajectory.jsonl` files still flow into doctor orphan detection and related usage/disk-budget scans. The PR is recent, active, linked to open issue #70680, and includes regression coverage plus follow-up handling for the valid `debug.trajectory` session-id concern raised in review. There is no merged or implemented-on-main replacement; the related #72047 was closed unmerged.
