## Summary

Keep this PR open. Current main still hardcodes shell completion profile paths, lacks ZDOTDIR/XDG_CONFIG_HOME regression coverage, and the PR is paired with the same author's open bug report #63069.

## Best Possible Solution

Keep the PR open as the implementation candidate for #63069. The maintainer path is to adapt the unified profile resolver into `src/cli/completion-runtime.ts`, use it for install/check/doctor messages, respect `ZDOTDIR` and `XDG_CONFIG_HOME`, preserve consistent bash fallback behavior, and add focused regression tests for zsh, fish, and bash profile selection.
