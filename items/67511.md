## Summary

Keep open. Current main still lacks the requested per-session workingContext capsule, runtime/status integration, and docs terminology cleanup. The issue discussion includes a concrete WSL/cloud/Docker compaction-loss report, and the related #71135 PR is open, draft, and explicitly storage-only rather than a shipped fix.

## Best Possible Solution

Keep this issue open as the product/API tracker. Review #71135 as a possible storage primitive, then land follow-up work that updates the capsule from run/exec/subagent/ACP paths, exposes cwd/repo/branch/sandbox/temp-clone/push-target state in /status, and updates workspace plus compaction docs to distinguish semantic continuity from working-state continuity.
