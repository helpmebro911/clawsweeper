## Summary

Keep #8495 open. Current main still blocks or skips unauthorized WhatsApp group messages, but does not auto-leave those groups, expose a WhatsApp group-join hook/event, or provide a WhatsApp leave-group message action. The related #11489 is broader and open, not a shipped replacement, and the existing maintainer-side Codex review comment also kept this issue open.

## Best Possible Solution

Keep #8495 open for a deliberate bundled WhatsApp plugin feature: add opt-in config/schema/docs for unauthorized-group auto-leave, implement a verified WhatsApp runtime leave operation with account and target authorization, cover unauthorized group admission and allowlist behavior with tests, and decide whether join notification belongs in this issue or the broader open #11489 group-pairing design.
