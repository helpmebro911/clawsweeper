## Summary

Keep open. Current main has related hook/plugin surfaces, including a bundled bootstrap-extra-files hook, but the specific requested capability remains missing: there is no agents.defaults.extraBootstrapFiles config, and the bundled extra-file loader rejects arbitrary custom filenames such as CONTEXT_BRIEF.md.

## Evidence

- **No agent-defaults config key:** The agent defaults schema includes skipBootstrap, contextInjection, bootstrapMaxChars, bootstrapTotalMaxChars, and bootstrapPromptTruncationWarning, but no extraBootstrapFiles-style key in this config surface.
  - file: [src/config/zod-schema.agent-defaults.ts:85](https://github.com/openclaw/openclaw/blob/81a41fe5bee350c6a035bfcc458eb8be380afe2c/src/config/zod-schema.agent-defaults.ts#L85)

- **Agent defaults type has no extraBootstrapFiles field:** The public agent defaults type documents the existing bootstrap controls but does not expose a custom bootstrap file list or per-agent/default extra file override.
  - file: [src/config/types.agent-defaults.ts:222](https://github.com/openclaw/openclaw/blob/81a41fe5bee350c6a035bfcc458eb8be380afe2c/src/config/types.agent-defaults.ts#L222)
