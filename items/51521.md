## Summary

Keep open. Current `main` has reasoning visibility controls and a generic full system-prompt override, but there is no dedicated reasoning-language setting and the provider request builders only pass reasoning effort/summary, not a language hint. That means the requested behavior is not clearly implemented on `main`.
