## Summary

Keep open. Current `main` does not implement this PR’s behavior: `wrapWebContent` still omits the warning for `web_search`, and the runtime before-tool-call path has no built-in external-content approval guard. The prior PR discussion also already identified the main blocker: the proposed guard was not wired into production dispatch.
