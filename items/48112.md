## Summary

Keep open. Current `main` still has the same hybrid fallback shape this PR is trying to fix, now under the bundled memory plugin path, and the relevant regression coverage is still skipped/missing. The PR is stale against the current file layout and has review nuance to resolve, but the underlying bug is not implemented on `main` or superseded by a canonical merged fix.

## Best Possible Solution

Keep this PR open, ask for a rebase/port to `extensions/memory-core/src/memory/manager.ts` and `extensions/memory-core/src/memory/index.test.ts`, and resolve the fallback threshold semantics with regression tests for sub-1.0 BM25 scores and temporal-decay/MMR-adjusted merged scores.

## Evidence
