## Summary

Keep open. Current main still lacks this PR's model-agnostic conversationalFreeform flag and per-turn reply-shaping hint. Existing GPT-5 overlay and GPT chat brevity guards partially address chat style, but they do not implement the PR's specific prompt-pipeline behavior, and the PR has substantive review comments to resolve rather than a cleanup-close reason.

## Best Possible Solution

Keep the PR open for maintainer review. If the product direction is accepted, revise the branch to compute conversational classification from the final current user message after inline rewrites, apply the hint consistently to immediate and queued prompt variants, and keep focused tests for flagged/unflagged prompt paths plus the queued and skill-command rewrite cases.
