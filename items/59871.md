## Summary

Keep open. Current `main` already fixes the TUI display half of the report by stripping inbound metadata from `command: true` messages before the TUI renders them, but the async exec follow-up half is still present on `main`: OpenClaw still injects a verbose agent-facing completion prompt with repeated behavioral instructions, and current tests explicitly assert that behavior.
