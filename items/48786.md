## Summary

Keep open. Current main still fetches quoted Feishu messages through getMessageFeishu() without reading a mentions array or applying normalizeMentions(), while the current inbound event path does normalize mentions. The linked PR #48976 is still open/unmerged in the provided GitHub context, so the real bug remains unshipped on main.
