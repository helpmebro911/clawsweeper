## Summary

Keep #70493 open. Current main has browser Playwright runtime packaging, browser.request/node-proxy plumbing, and browser doctor/CLI smoke coverage, but I found no isolated sub-agent regression coverage for browser navigate/snapshot/act through the spawned-session path. The issue also has an explicit maintainer comment from steipete saying not to close until isolated-session coverage exists.

## Best Possible Solution

Keep the issue open and fix it by adding an isolated sub-agent regression test that enables the browser tool, spawns an isolated sub-agent/session, and calls browser snapshot or navigate through that runtime. Then either ensure the isolated runtime includes the browser plugin Playwright dependency or explicitly delegates browser operations to the host/node browser proxy that owns Playwright availability.
