## Summary

Keep open. Current main still has the read-tool diagnostic guard manually checking only `path` and `file_path`, so the PR’s `filePath` alias fix is not already implemented on main. The PR is small, targeted, and the existing discussion supports that it reduces duplication by using the shared resolver.
