## Summary

Keep open. Current `main` already contains the cron payload-kind idempotence fix, but it does not contain the daemon/systemd inferred user-bus retry described in PR #42342. The remaining systemd behavior is still distinct and unshipped, so there is not strong evidence to close this PR as implemented or obsolete.
