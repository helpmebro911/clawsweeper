## Summary

Keep this open. Current main still supports session listing, switching, table filtering/sorting, bulk delete, and label edits, but it does not implement the requested ChatGPT/Claude-style left-side conversation management: manual drag ordering, persistent folders/tags/custom categories, move-between-groups behavior, or double-click/context-menu rename in the chat sidebar.

## Best Possible Solution

Keep this open as a Control UI conversation-management enhancement. The best path is to design a persistent session organization contract, extend SessionEntry/GatewaySessionRow and additive gateway protocol APIs for folder/tag/category assignment, manual ordering, and move operations, then add a left conversation list with drag ordering, move/group actions, and double-click/context-menu rename. Reuse existing label support where it fits, with focused UI/gateway tests and docs once the product contract is settled.
