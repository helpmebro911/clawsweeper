## Summary

Keep this issue open. Current main still has session listing, filtering, sorting, delete, and label editing, but it does not implement the requested left-side conversation-list management: manual drag ordering, persistent folders/tags/custom categories, move-between-groups behavior, or double-click/context-menu rename in a ChatGPT/Claude-style sidebar.

## Best Possible Solution

Keep this open as a Control UI conversation-management enhancement. The best path is to design a persistent session organization contract, extend SessionEntry/GatewaySessionRow plus additive gateway protocol APIs for folder/tag/category assignment, manual ordering, and move operations, then add a left conversation list with drag ordering, move/group actions, and double-click/context-menu rename. Existing label support can be reused where appropriate, with focused UI/gateway tests and docs once the product contract is settled.
