## Summary

Keep open. `main` does not already contain this change, and the PR's new SQLite dot-parameter path has a verified correctness bug: `.parameter set ?N '5551234567'` binds an integer expression, so `WHERE digits IN (?N)` can fail against text-valued `digits`. The prior cross-reference to PR #63584 is useful context, but this patch still needs revision rather than conservative auto-close.
