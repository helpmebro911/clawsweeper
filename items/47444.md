## Summary

Keep open. Current main has partial assistant-output sanitization, but the exact XML-style `<function_calls><invoke ...>` leak class in #47444 is not fixed with high confidence; the sanitizer tests currently assert that a matching `<function_calls><invoke ...></invoke></function_calls>` payload is preserved.
