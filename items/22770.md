## Summary

Keep open. Current `main` does not implement an external-editor compose flow for the TUI, and the documented/implemented shortcut set has no `$EDITOR`/`$VISUAL` handoff.

## Evidence

- **Documented TUI shortcuts omit external-editor compose:** `docs/web/tui.md` lists Enter, Esc, Ctrl+C, Ctrl+D, Ctrl+L, Ctrl+G, Ctrl+P, Ctrl+O, and Ctrl+T only. There is no Ctrl+E or external-editor workflow documented.
