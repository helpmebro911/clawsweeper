## Summary

Keep open. Current main has meaningful progress on skill discovery, ClawHub installs, plugin-packaged skills, and gateway-backed skill install scanning, but this issue is a recent broad ecosystem report and several central asks remain unresolved or only partially covered. In particular, ClawHub skill downloads still use a separate install path without the gateway-backed installer security scan/hook path, the watcher still constructs glob targets while the repo depends on chokidar 5, and requested metadata/composition/trust features such as trigger examples, skill types, permission manifests, signed skills, and versioned dependencies are not present as a complete core contract. Related comments and linked issues are useful narrower follow-ups, but no single canonical item fully supersedes this umbrella.
