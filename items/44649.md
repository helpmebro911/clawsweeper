## Summary

Current `main` already carries requester delivery context through subagent completion announce routing, and the bundled Feishu plugin now resolves the original DM/topic target for completion delivery. The exact Feishu DM/topic/sender-scoped cases from the report are covered by regression tests, and the implementation shipped in `v2026.4.22`.
