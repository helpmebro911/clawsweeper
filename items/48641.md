## Summary

Keep open. Current main has Discord inbound/gateway hardening, but the specific multi-account Discord DM allowlist issue discussed here is not conclusively fixed: per-account `allowFrom` still overrides top-level `allowFrom`, and the linked fix PR #49365 was closed unmerged in the provided GitHub context.
