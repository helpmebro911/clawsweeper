## Summary

Keep #7424 open. Current `main` still provides a global `agents.defaults.models` catalog/allowlist plus per-agent primary/fallback model overrides, but it does not provide an ordinary per-agent model allowlist schema or agent-scoped `/models`/`sessions.patch` enforcement. Related #9010 is closed as a duplicate, so #7424 remains the canonical tracker.

## Best Possible Solution

Keep #7424 open as the canonical feature request. Add an agent-scoped model allowlist, likely under `agents.list[].model.allowlist` or `agents.list[].models`, define how it combines with `agents.defaults.models`, then enforce it anywhere models are listed or selected: gateway `models.list`, `/model`/`sessions.patch`, OpenAI-compatible override paths, Control UI model pickers, config schema/types/generated docs, and regression tests. #9010 can remain closed as a duplicate pointing here.
