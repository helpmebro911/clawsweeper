## Summary

Current `main` does not implement a per-agent model allowlist. It supports per-agent default model and fallback overrides, but model authorization is still derived from the global `agents.defaults.models` allowlist plus fallback models, so an agent can still switch to any globally allowlisted model.

## Evidence

- **Agent config only exposes per-agent primary/fallback overrides:** `AgentConfig.model` is typed as `string | { primary, fallbacks }`; there is no per-agent `allowedModels`/allowlist field in the agent schema.
  - file: [src/config/types.agents.ts:80](https://github.com/openclaw/openclaw/blob/7e16e3d07723cc3903f40ce94f53a3920c88b1fc/src/config/types.agents.ts#L80)
- **Docs describe per-agent model as primary or { primary, fallbacks }:** The configuration reference for `agents.list` documents `model: "provider/model" // or { primary, fallbacks }` and says object form overrides `primary` and `fallbacks`; it does not define a per-agent model allowlist.
  - file: [docs/gateway/configuration-reference.md:1775](https://github.com/openclaw/openclaw/blob/7e16e3d07723cc3903f40ce94f53a3920c88b1fc/docs/gateway/configuration-reference.md#L1775)
- **Allowed model set is built from the global allowlist:** `buildAllowedModelSetWithFallbacks` reads `cfg.agents?.defaults?.models` into `rawAllowlist`, sets `allowAny` from that global map, and only adds fallback models on top. No per-agent allowlist is consulted.
  - file: [src/agents/model-selection-shared.ts:468](https://github.com/openclaw/openclaw/blob/7e16e3d07723cc3903f40ce94f53a3920c88b1fc/src/agents/model-selection-shared.ts#L468)
- **Runtime enforcement uses that global allowed set:** `agent-command` builds `allowedModelKeys` via `buildAllowedModelSet(...)` and rejects explicit overrides only when the selected model is outside that set. Because the set is global-plus-fallbacks, a globally allowlisted expensive model remains selectable for any agent.
  - file: [src/agents/agent-command.ts:685](https://github.com/openclaw/openclaw/blob/7e16e3d07723cc3903f40ce94f53a3920c88b1fc/src/agents/agent-command.ts#L685)
- **Checkout stayed clean during review:** `git status --short` returned no output before and after inspection.
  - command: `git status --short`
