## Summary

Keep #9987 open. Current main has the gateway plumbing to list pending exec approvals, but the requested user-facing `openclaw approvals pending` command is still absent from the CLI registration, CLI tests, and docs. The related implementation attempts #10013 and #10341 were closed unmerged, so there is no shipped CLI surface to close against.

## Best Possible Solution

Keep #9987 open until OpenClaw adds `openclaw approvals pending` in `src/cli/exec-approvals-cli.ts`, backed by the existing `exec.approval.list` gateway method. The initial implementation should include human-readable table output with command details, `--json` output for monitoring, focused tests in `src/cli/exec-approvals-cli.test.ts`, and docs updates in `docs/cli/approvals.md` plus `docs/cli/index.md`. Treat `--watch` as a follow-up unless maintainers explicitly want it in the first CLI surface.
