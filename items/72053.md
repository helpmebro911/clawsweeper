## Summary

Keep open. Current main still sends the subagent task in both the subagent system prompt and the initial child user message, which is the behavior this PR is trying to fix. The linked bug remains open, and the related alternative PR is also open rather than a landed canonical replacement.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to select one canonical fix for #72019, ensure the first subagent model request contains exactly one long-form copy of the task, preserve multiline and structured task text, and land regression coverage for both the system-prompt and initial-user-message surfaces.
