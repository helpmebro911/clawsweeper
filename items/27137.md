## Summary

Keep open. Current main has shared DM/group-policy primitives, but it has not implemented the requested unified `authorizeIngress()` gate, and representative channels still perform authorization in channel/event-specific paths. The request is core security architecture work, not optional ClawHub/plugin scope.

## Evidence
