Keep this PR open. ## Best Possible Solution

Keep this PR open for maintainer review. If `agents.defaults.subagents.model` is intended to outrank an agent's own `model.primary`, land a narrow fix in `src/agents/model-selection.ts` and `src/cron/isolated-agent/model-selection.ts`, update the regression tests, and explicitly validate or document allowlist behavior for resolved subagent models before persistence.
