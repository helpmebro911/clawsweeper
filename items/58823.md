## Summary

Keep open. Current `main` still gives a parent agent's own model higher priority than `agents.defaults.subagents.model`, which matches the reported regression and contradicts the current docs. The PR also has an unresolved security concern in discussion around explicit spawn-time model overrides, so there is not strong evidence for conservative closure.
