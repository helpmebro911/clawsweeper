## Summary

Keep open. Current main still matches the narrower bug in #50590: sandbox setup copies visible skills into the sandbox workspace, but the actual run path can still prefer a pre-sandbox skills snapshot whose prompt and file paths were built from the normal agent workspace/global managed-skill locations. That means the related #46257 sync work is useful, but it is not enough evidence to close this issue.
