## Summary

Keep open: current `main` does not implement per-agent compaction overrides. The agent-entry schema, resolved agent config, runtime compaction paths, and docs all remain `agents.defaults.compaction`-only, while the issue discussion points to replacement PR #69988 that is still off-main.

## Evidence

- **Agent schema has no per-agent compaction field:** `AgentEntrySchema` defines per-agent overrides like `heartbeat`, `identity`, `subagents`, `embeddedPi`, `sandbox`, `tools`, and `runtime`, but no `compaction` key, so `agents.list[].compaction` is not accepted on current `main`.
  - file: [src/config/zod-schema.agent-runtime.ts:812](https://github.com/openclaw/openclaw/blob/24e9924d6abd93560fb7bf721909257de533e57a/src/config/zod-schema.agent-runtime.ts#L812)
