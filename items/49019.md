## Summary

Keep open. Current main still implements `POST /hooks/agent` as an async trigger that returns `{ ok, runId }`, with no request fields or response shape for waiting on completion and returning a deterministic machine result. Related PR #67433 appears to be an active implementation attempt, but it is still open/unmerged, so this issue is not implemented or cleanly superseded yet.
