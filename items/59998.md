## Summary

Keep open. Current `main` still hardcodes Windows exec through PowerShell/pwsh, the exec runtime always uses that helper, and the config/schema/docs expose `tools.exec.host/node/pathPrepend/...` but no `tools.exec.shell` field. This request is not implemented on `main` and remains a core exec-config feature rather than a ClawHub/plugin item.
