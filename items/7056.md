## Summary

Current `main` already has first-class sandboxing for agent operations: configurable sandbox modes, Docker/SSH/OpenShell backends, tool routing into the sandbox runtime, and sandbox-root path confinement. That covers the issue's requested container/VM-only operational layer well enough.

Since the requested container/VM-only operational layer already exists in core, there is no separate open feature here to track.
