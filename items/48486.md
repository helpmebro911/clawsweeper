## Summary

Keep issue #48486 open. Current main still matches the reported gap: `sanitizeHtml()` only treats exact zero font size and transparent/zero-alpha colors as hidden, and there is no near-white color parsing, tiny non-zero font threshold, or foreground/background color equivalence check.
