## Summary

Keep #71285 open. Current main still has no `memorySearch.ignorePaths` or equivalent config surface, and both builtin and QMD memory indexing still include recursive `memory/**/*.md` by default. Existing dreaming fixes exclude `memory/dreaming/**` from short-term promotion, but they do not stop dream report Markdown files from being indexed, watched, or returned by `memory_search`.

## Best Possible Solution

Keep the issue open and implement a real memory-search exclusion contract across config schema/types/docs, builtin file enumeration, builtin chokidar watching, QMD default/custom collection handling, and tests. The narrow default could either exclude generated `memory/dreaming/**` by policy or add documented `memorySearch.ignorePaths` semantics that work consistently for both indexing and watching.
