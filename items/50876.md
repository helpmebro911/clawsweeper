## Summary

Keep open. Current main does not implement the requested `agents.defaults.bootstrapFiles` / per-agent config key, and the existing bundled `bootstrap-extra-files` hook is narrower than the reported RULES.md use case because it only accepts recognized bootstrap basenames. The related model-aware AGENTS/SUBAGENTS work is adjacent but does not supersede this arbitrary workspace bootstrap-file extension request.
