## Summary

Keep open. Current main has adjacent mechanisms for isolated agents/workspaces, per-agent memory indexing, and manually injecting extra project context files, but I did not find a current implementation of the issue's central request: same-session project context declaration/switching with project-level memory to prevent stack/context bleed between projects.
