## Summary

Keep open. Current main still appears to have both reported Google Chat UX gaps: the typing indicator display-name fallback does not read `agent.identity.name`, and inbound reply delivery sends/update Google Chat text chunks and media captions without a markdown-to-plain-text/Google Chat formatting step. The related discussion is useful context, but PR #49361 only tracks the typing-name sub-issue and remains open, while PR #49406 attempted the markdown fix but was closed unmerged.

## Evidence

- **Typing indicator fallback omits identity.name:** `resolveBotDisplayName` checks account config name and `agent.name`, then falls back to `OpenClaw`; it does not check `agent.identity?.name` before the fallback. The typing placeholder then sends `_${botName} is typing..._`.
  - file: [extensions/googlechat/src/monitor.ts:77](https://github.com/openclaw/openclaw/blob/a1090b604312ac7a49cf2d2ce7276b334786bb79/extensions/googlechat/src/monitor.ts#L77)
  - command: `nl -ba extensions/googlechat/src/monitor.ts | sed -n '1,380p'`
  - sha: [a1090b604312](https://github.com/openclaw/openclaw/commit/a1090b604312ac7a49cf2d2ce7276b334786bb79)
- **Agent config supports identity:** `AgentConfig` includes both `name?: string` and `identity?: IdentityConfig`, so an identity-only configured display name is a real current config shape that the Google Chat fallback can miss.
  - file: [src/config/types.agents.ts:74](https://github.com/openclaw/openclaw/blob/a1090b604312ac7a49cf2d2ce7276b334786bb79/src/config/types.agents.ts#L74)
  - command: `nl -ba src/config/types.agents.ts | sed -n '70,115p'`

- **Inbound Google Chat replies send markdown chunks unchanged:** `deliverGoogleChatReply` chunks `reply.text` with `chunkMarkdownTextWithMode` and passes each chunk directly to `updateGoogleChatMessage` or `sendGoogleChatMessage`; media captions are likewise passed directly as `text: caption`.
  - file: [extensions/googlechat/src/monitor-reply-delivery.ts:66](https://github.com/openclaw/openclaw/blob/a1090b604312ac7a49cf2d2ce7276b334786bb79/extensions/googlechat/src/monitor-reply-delivery.ts#L66)

  - file: [extensions/googlechat/src/channel.adapters.ts:145](https://github.com/openclaw/openclaw/blob/a1090b604312ac7a49cf2d2ce7276b334786bb79/extensions/googlechat/src/channel.adapters.ts#L145)
