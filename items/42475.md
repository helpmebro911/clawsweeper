## Summary

Keep open. Current `main` still has cost reporting and session disk-budget enforcement, but no per-agent cost-budget config or gateway-side spend gate. The request is coherent, recent, and aligned with an existing documented security recommendation rather than being a ClawHub/plugin-only idea.
