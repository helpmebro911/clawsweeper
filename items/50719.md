## Summary

Keep open. Current main has better vLLM support and a separate 404 failover classifier, but I did not find high-confidence evidence that the reported `local-vllm` GLM 4.7 regression is fully fixed. The issue is only 36 days old, the linked fix PRs were closed unmerged, there is no exact #50719 regression test/changelog entry, and current docs still contain conflicting guidance for vLLM/custom local proxies.
