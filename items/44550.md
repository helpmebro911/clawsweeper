## Summary

Keep open. Current `main` still does not carry per-model Ollama `params.num_ctx` through to the runtime request path. `agents.defaults.models.*.params` is read generically, but the Ollama plugin does not translate that config into model metadata or request payloads, and the native/compat Ollama paths still derive `num_ctx` from `contextWindow` instead.
