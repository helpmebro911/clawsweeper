## Summary

Keep #13962 open. Current main only implements pending WhatsApp group-history injection for messages skipped by mention gating; it still lacks per-mention/background model routing, does not process unmentioned group messages through a cheap/background model path, and does not expose per-group mention context sizing like `mentionContextMessages`. The existing maintainer-posted Codex comment on the issue reached the same keep-open conclusion on April 26, 2026, and current main still matches that assessment.

## Best Possible Solution

Keep the issue open until OpenClaw either implements mention-state-aware model routing plus per-group mention context sizing, or exposes a clear pre-gating plugin/API path that lets a trusted plugin process unmentioned group messages with a background model before the main mention-triggered reply path.
