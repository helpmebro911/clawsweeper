## Summary

Keep open. Current main still has the documentation bug: the QMD + bridge example nests `memory-wiki` inside `memory`, while the documented and implemented plugin config contract is `plugins.entries["memory-wiki"].config`. PR #70278 is a focused docs-only correction rather than obsolete work.

## Best Possible Solution

Keep PR #70278 open and merge this docs fix, or land an equivalent maintainer patch, so the QMD + bridge example matches the shipped `plugins.entries["memory-wiki"].config` contract.
