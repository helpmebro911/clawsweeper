## Summary

Keep open. Current `main` has a malformed tool-call argument repair path, but it is only enabled for `kimi` on `anthropic-messages` and for `openai-completions`. Amazon Bedrock Claude runs as `bedrock-converse-stream`, and the Bedrock plugin wrapper adds cache/guardrail handling only, not equivalent tool-argument repair or retry logic. The reported Bedrock `{}` tool-call failure is therefore not clearly fixed on `main`.
