## Summary

Keep open. Current `main` already covers several sub-requests from this umbrella issue (exec approvals, session transcript persistence, auto-compaction, and hook systems), but the main security ask is still unresolved: host exec is still unsandboxed by default when sandboxing is off, and the docs still describe that as a critical residual risk. The issue discussion also points to follow-up PR work for macOS `sandbox-exec`, which reads as ongoing rather than already-landed upstream behavior.

## Evidence
