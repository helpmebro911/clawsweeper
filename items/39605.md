## Summary

Keep open. Current main has a fresh-config path for Telegram native commands, but Discord and Slack slash command routing still use startup/captured config when calling resolveAgentRoute, so the reported cross-channel bug is not implemented on main. The related PR #39617 is still open in the provided GitHub context, so it is useful evidence but not a reason to close the issue yet.
