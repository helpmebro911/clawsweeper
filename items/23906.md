## Summary

Keep open. Current main has coarse subagent tool policy and some gateway-bound plugin tool support, but the requested per-spawn `sessions_spawn` tool profile override is still missing: the tool schema does not expose `tools`, `toolsAllow`, or `disableTools`, and the native spawn path does not forward any per-call tool selection into the child agent run.

## Evidence
