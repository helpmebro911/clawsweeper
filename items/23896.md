## Summary

Keep open. The issue describes a concrete Discord feature gap and cites NanoBot's require_mention_from_ids as prior art, but current main still only supports guild/channel-wide requireMention plus sender allowlists; it cannot require mentions only from selected sender IDs while exempting others such as bridge bots.
