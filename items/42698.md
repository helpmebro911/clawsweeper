## Summary

Keep open. Current `main` has moved Bonjour advertising into the bundled `bonjour` plugin, but the code does not wire the ciao unhandled-rejection classifier into the runtime path, and the classifier that exists only matches the `defined -> undefined` IPv4 assertion direction. The PR's reported `undefined -> defined` crash is therefore not clearly fixed on `main`.
