## Summary

Keep #25592 open. Current main has targeted hardening for commentary/final_answer phase separation, but it does not implement the broader requested policy that intermediate or unintentional bare assistant text must never reach messaging channels. A current-main regression test still expects buffered assistant text to be delivered via onBlockReply when tool execution starts, and the proposed replyMode/tool-only explicit-send config is not present in the agent defaults schema.
