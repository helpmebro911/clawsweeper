## Summary

Keep #48510 open. Current main still has no structured capability descriptor on agent tool definitions, plugin tool registration, or the before_tool_call hook payload. This is a clear, actionable core/plugin-SDK feature request rather than a ClawHub-only plugin idea, because it asks for a shared tool metadata contract and hook exposure that external policy plugins cannot add by themselves. The related #48503 is complementary and still open, not a merged or canonical replacement.
