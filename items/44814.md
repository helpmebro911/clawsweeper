## Summary

Keep open. Current `main` and the latest shipped release still hardcode a 5 MB attachment limit in both Control UI upload paths, and the requested `gateway.webui.attachmentMaxBytes` config surface is not present. The PR discussion also shows the author updated the patch to cover the `chat.send` path after review, which matches the live gap on `main`.
