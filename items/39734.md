## Summary

Keep open. Current main appears to address the concrete 1M Bedrock Claude 4.6 context-window metadata path for discovered models, and the issue discussion notes that the original 1M beta flag may no longer be required. But the requested capability, configurable custom Bedrock `anthropic_beta` / provider-field passthrough, is not implemented on main, and the issue is not old enough for stale-insufficient-info closure.
