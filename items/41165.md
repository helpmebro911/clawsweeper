## Summary

Current main does not strongly resolve #41165. Default-account Telegram DMs can still resolve to `agent:main:main` under `session.dmScope: "main"`, and the heartbeat runner still prioritizes a forced/originating session before `heartbeat.session`, matching the later system-event heartbeat reports. Related PR #56092 remains open, while #41939 was closed unmerged in the provided GitHub context.
