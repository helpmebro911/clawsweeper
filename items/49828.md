## Summary

Keep #49828 open. Current main still uses a chat-header session/agent dropdown for WebChat switching, while the only chat sidebar implementation is a tool/markdown detail pane. The requested left-side agent conversation list is a core Control UI UX change, not an optional ClawHub/plugin cleanup target, and no duplicate/superseding item in the provided related context fully tracks this sidebar request.
