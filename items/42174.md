## Summary

Keep open. Current `main` still rejects Windows drive-letter bind sources during both config validation and runtime sandbox validation, while adjacent sandbox parsing code already understands Windows bind specs. The PR targets a real gap, but the review thread raised unresolved correctness/security concerns that need maintainer judgment rather than auto-close.
