## Summary

Keep open. Current `main` does not implement this PR's behavior: Telegram DM draft preview transport is still forced to `"message"` for all DM lanes, and the regression tests still assert `"message"` for both answer and reasoning lanes. That means the requested answer-lane `"auto"` transport restoration is not already on `main`, so there is not strong evidence for a conservative close.
