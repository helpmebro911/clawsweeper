## Summary

Keep #27526 open. Current main has useful building blocks: sessions, labels/search, model/thinking overrides, checkpoints/branching, multi-agent isolation, and sub-agent background runs. It still does not implement the requested WebChat conversation directory with folders/tags/archive/session pinning/recommendations, nor the requested multi-AI collaboration UX for side-by-side model comparison, roles, consensus/conflict handling, agendas/minutes, committees, or project tracking. The issue discussion also explicitly marks this as the canonical tracker after duplicate #27603 was consolidated here.

## Best Possible Solution

Keep #27526 open as the canonical product tracker. The best path is to split it into explicit milestones: extend the session protocol/storage with folder, tag, archive, persistent pin, richer search, and recommendation metadata; add WebChat/Control UI directory navigation; then design the multi-model collaboration UX with roles, side-by-side comparison, consensus/conflict handling, agendas/minutes, committee templates, and task extraction/project tracking. Existing sessions, labels/search, checkpoints, multi-agent routing, and sub-agents should be treated as foundations, not completion.
