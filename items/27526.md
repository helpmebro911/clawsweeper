## Summary

Keep #27526 open. Current main still has only foundational session management and agent primitives: flat session list/search/labels, per-session overrides, checkpoint branching, multi-agent isolation, and sub-agent spawning. It does not implement the requested WebChat conversation directory with folders/tags/archive/pinning/recommendations, nor the multi-AI collaboration UX for side-by-side model comparison, roles, consensus/conflict handling, agendas/minutes, committees, or project tracking. The provided GitHub context also includes a maintainer comment marking this as the canonical tracker after duplicate #27603 was consolidated here.

## Best Possible Solution

Keep #27526 open as the canonical product tracker. The best path is to split it into explicit milestones: extend session protocol/storage with folder, tag, archive, persistent pin, richer search, and recommendation metadata; add WebChat/Control UI directory navigation; then design the multi-model collaboration UX with roles, side-by-side comparison, consensus/conflict handling, agendas/minutes, committee templates, and task extraction/project tracking. Existing session labels/search, checkpoints, multi-agent routing, and sub-agents should be treated as foundations, not completion.
