## Summary

Current main partially mitigates the common retry-success case by clearing a stale mutating tool error after the same mutating action succeeds, but the requested configurable suppression policy is still missing. Main still intentionally surfaces mutating tool warnings even when the assistant has a user-facing success reply, and the related PR #39636 was closed unmerged.
