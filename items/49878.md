## Summary

Keep open. Current main has partial fixes: heartbeat preflight now reads HEARTBEAT.md on each run for empty-file gating and task parsing. However the bootstrap/context path can still reuse a session-keyed workspace snapshot, and I did not find a regression proving that editing non-empty HEARTBEAT.md content is refreshed without session deletion. That leaves the original stale-content drift plausible.
