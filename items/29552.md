## Summary

Keep open. Current `main` contains partial mitigations for the reported symptom set, but not strong enough evidence that the underlying cross-component version-skew bug from #29552 is fully resolved. In particular, gateway auth now supports a trusted device-token retry after shared-token mismatch, and global package updates now verify the installed version/dist contents, but real version-skew warnings still exist and there is no clear proof in current `main` that companion/app/CLI version drift can no longer happen.
