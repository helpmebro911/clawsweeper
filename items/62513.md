## Summary

Current main still has the reported behavior: memory search calls `hasIndexedContent()` before preflight, and that method directly probes SQLite. The linked PR #62597 appears to track the same optimization, but it is open and unmerged in the provided GitHub context, so the requested fix is not implemented on main yet.
