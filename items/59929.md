## Summary

Keep open. The PR is still useful because current main still resolves the compaction safeguard model as `ctx.model ?? runtime?.model`, so a session model can override the configured runtime compaction model. The PR changes that precedence and adds a focused regression test. This is not a conservative cleanup close candidate.
