## Summary

Keep open. Current `main` and the latest release still resolve the compaction safeguard model as `ctx.model ?? runtime?.model`, so a session model can override the configured `agents.defaults.compaction.model` runtime target. That conflicts with the documented contract and the newer runtime-context tests, so this PR is not implemented on `main` yet.
