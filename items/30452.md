## Summary

Keep open. Current main has implemented the compaction-only model override, but the issue also asks for memory-flush model selection and a flush-then-reset mode. I found no `memoryFlush.model` or `memoryFlush.resetAfterFlush` surface in the current config types, schema, or docs, so the request is only partially implemented on `main`.
