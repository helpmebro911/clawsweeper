## Summary

Keep #11955 open. Current main has adjacent memory search, optional session transcript indexing, usage/cost/session aggregation, first-turn startupContext, and an opt-in BOOT.md startup hook, but the umbrella still has meaningful missing work: an agent-facing self-evaluation metrics API, automatic conversation chaining/summary handoff, and default gateway-restart preload before first user input. The provided discussion also includes a recent maintainer comment explicitly keeping this issue open as the canonical umbrella after duplicate #11956 was closed.

## Best Possible Solution

Keep #11955 open as the canonical umbrella unless a maintainer splits it. Preserve the shipped memory and usage surfaces, then track or implement the remaining product/API work: an agent-facing self-evaluation metrics API, documented conversation continuation or compact-summary handoff, and explicit gateway-startup preload/prewarm of core workspace context before first user input.
