## Summary

Keep #11955 open. Current main implements adjacent pieces: memory search over MEMORY.md and memory/*.md, configurable extra paths, opt-in transcript indexing, usage/cost/session aggregation, first-turn startup context for bare /new and /reset, and an optional BOOT.md startup hook. The umbrella still has meaningful unimplemented core work: an agent-facing self-evaluation metrics API with task/satisfaction/success signals, automatic conversation chaining or compact-summary handoff for new sessions, and default gateway-restart preload/prewarm of core context before first user input. The provided GitHub context also shows #11955 is the canonical tracker after duplicate #11956 was closed, and includes an earlier maintainer-authored Codex review comment explicitly keeping it open.

## Best Possible Solution

Keep #11955 open as the canonical umbrella unless a maintainer splits it. Preserve the shipped memory_search, extraPaths, sessionMemory, sessions.usage, startupContext, and boot-md pieces, then define narrower follow-up work for an additive agent-facing metrics/self-evaluation API, a documented conversation continuation or compact-summary handoff mode, and an explicit gateway startup preload/prewarm mechanism for core workspace context before first user input.
