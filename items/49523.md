## Summary

Keep open. Current main still implements the reported transcript header path with `cwd: process.cwd()` inside `ensureSessionHeader()`, and the mirrored append path calls that helper without passing an explicit agent workspace. The linked follow-up PRs are useful context, but they are closed/unmerged or still open, and the requested fix is not present in the checked-out main SHA.
