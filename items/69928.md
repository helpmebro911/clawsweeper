## Summary

Keep open. Current `main` still routes `claude-opus-4.7` through the adaptive-thinking branch and unconditionally emits `params.output_config = { effort: options.effort }` on the shared Anthropic transport, with no `github-copilot` or model-specific guard. The bundled `github-copilot` provider still advertises `high` for Copilot models, so `github-copilot/claude-opus-4.7` can still reach the unsupported payload shape described in the issue.
