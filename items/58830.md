## Summary

Keep open. Current main has some transparency pieces for compaction, including optional user notices, /status counts, manual /compact, docs, and plugin hooks/providers, but it does not implement the requested confirmable/preview/rollback workflow or the proposed notify-only/require-confirmation modes. This is a plausible core UX/control-plane feature request, not stale, not maintainer-authored, and not a ClawHub-only plugin idea.
