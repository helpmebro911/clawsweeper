## Summary

Keep open. Current main still fail-closes chained interpreter script commands like `cd <dir> && python3 script.py` outside the gateway `security=full` + `ask=off` path. The related #67621 is useful exec-preflight context, but it is about source-file `$` token scanning, not this command-pattern failure.
