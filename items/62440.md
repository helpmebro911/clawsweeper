## Summary

Keep #62440 open. Current main still routes the local `agent --local --json` envelope through `runtime.log()`, while the gateway JSON path uses `writeRuntimeJson()`. The two linked fix PRs are useful and directly relevant, but they are unmerged in the provided GitHub context and the requested behavior remains missing on main.
