## Summary

Keep #27574 open. The request is a concrete core Dashboard/Control UI feature, the author/labels are not protected, and current main still lacks a live browser preview panel. Existing browser support provides CLI/gateway screenshot, console, request, and action surfaces, but not a Dashboard route/panel that streams what the browser sees. This also should not be closed as clawhub yet because the documented plugin API does not expose a Control UI view/panel registration seam, so the requested Dashboard integration would require core UI work or a new plugin API first.
