## Summary

Keep open. Current main partially addresses the related microphone-only problem by allowing same-origin microphone access, but #47565 asks for configurable Control UI Permissions-Policy and CSP overrides, including adding local inference servers to `connect-src`. The gateway still builds those headers from hardcoded source, and `gateway.controlUi` has no `permissionsPolicy` or `cspOverrides` config fields in the exported type or strict schema.
