## Summary

Keep open. Current `main` still matches the reported bug: memory-flush and preflight gating compute thresholds from `reserveTokensFloor`, while the sibling preemptive-compaction path still uses `reserveTokens`. The issue is recent, has active follow-up discussion, and already has open linked fix PRs in the timeline, so there is no conservative basis to close it.
