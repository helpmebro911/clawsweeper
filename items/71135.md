## Summary

Keep open. Current main does not contain the PR's session working-context capsule or an equivalent runtime feature, and the linked #67511 remains an open core session-continuity request with supporting user discussion. This draft PR is recent, focused, and potentially useful, so it needs maintainer design review rather than automated cleanup.

## Best Possible Solution

Keep #71135 open for maintainer review or fold it into the canonical #67511 implementation. The best product path is a core-owned working-context design that persists enough machine-readable state to answer “where were we working?”, then wires updates from execution/runtime metadata, compaction/new flows, `/status`, and subagent spawn, with focused tests and docs terminology updates.
