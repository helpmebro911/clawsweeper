## Summary

Keep open. The issue is recent, has concrete macOS LaunchAgent reproduction evidence from the reporter plus corroborating follow-up comments, and current main still appears to contain the three risky mechanics described: auto-update/update subprocess output is captured through pipes, update restart refresh runs `gateway install --force` before the restart handoff, and LaunchAgent install still performs `launchctl bootout` before bootstrap.
