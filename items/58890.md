## Summary

Keep open. Current `main` still matches the reporter's core failure shape: auto-update runs as a pipe-backed child, `maybeRestartService` still performs `refreshGatewayServiceEnv()` before detached restart handoff, and the macOS install path still calls `launchctl bootout` from `gateway install --force`. The discussion's distinction between the hardened restart path and the still-risky install/update path is supported by current source.
