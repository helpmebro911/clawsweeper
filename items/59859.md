## Summary

Keep PR #59859 open. Current main still does not include a native Linux companion app, public docs still describe native Linux companion apps as planned, and the gateway client registry/handshake paths do not yet recognize an `openclaw-linux` native UI client. The PR is large and has review risk, but it is a plausible implementation track for maintainer-owned issue #75 rather than something already shipped, external to core, or clearly superseded.

## Best Possible Solution

Keep the PR open and route it through maintainer product review for issue #75. The best path is to decide whether the C/GTK track in #59859, the alternative #61576 track, or a smaller staged implementation should become the canonical Linux companion effort, then require the chosen track to resolve gateway-contract, lifecycle, security, and packaging review issues before merge.
