## Summary

Keep open. Current main still only implements the native signal-cli JSON-RPC/SSE path for the bundled Signal plugin: there is no `apiMode` config, no adapter/container client, no REST `/v2/send`, and no WebSocket `/v1/receive/{account}` stream. The linked issue #10240 tracks the same still-missing containerized Signal support, and this PR remains potentially useful rather than obsolete or superseded.
