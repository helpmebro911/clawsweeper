## Summary

Keep open. Current `main` still accepts empty or whitespace-only `name` and `description` values in `skills/skill-creator/scripts/quick_validate.py`, and the regression tests proposed by this PR are not present. One claim in the PR body is outdated on current `main` because `package_skill.py` names the archive from the directory basename, not the frontmatter `name`, but the validation bug itself remains unaddressed.

## Evidence
