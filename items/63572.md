## Summary

Keep open. Current main still lacks the requested doctor stale workspace path detector/repair and the migration guide cross-OS workspace path warning. The provided GitHub context also shows PR #63803 is open, unmerged, and uses closing wording for this issue, so the issue should remain open until that PR or an equivalent fix is reviewed and lands.

## Best Possible Solution

Keep this issue open and review PR #63803 as the focused implementation candidate. The best maintainer path is a doctor-owned detector/repair for stale home-shaped absolute workspace paths in `agents.defaults.workspace` and per-agent `workspace`, backed by regression tests and a migration guide warning. After that PR or an equivalent fix lands, close this issue as implemented.
