## Summary

Keep open. Current main has improved stale-index freshness defaults, but the linked FTS/hybrid failure is still plausible in source: provider-backed hybrid search passes the current embedding model into keyword search, and the FTS query still adds an `AND model = ?` filter. The directly linked PR #48328 that proposes removing that filter is still open/unmerged in the provided context, so this is not conservative to close as implemented.
