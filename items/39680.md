## Summary

Keep open. Current main still contains the reported broad base64 heuristic in `validateEnvVarValue`, and `sanitizeEnvVars` still applies it to values without key-name context. The related fix PRs in the provided GitHub context were closed unmerged, so they do not supersede the remaining work.
