## Summary

Keep PR #70647 open. Current main already has the production empty-response guard that would surface a terminal error for empty assistant turns, but main does not contain the PR's specific regression coverage for non-strict-agentic NVIDIA/Nemotron shapes: unsigned-thinking-only `stopReason="stop"` and fully empty `content: []` without retry. The PR is recent, focused, test-only, linked to concrete external NemoClaw hang reports in the body, and the only review feedback is a minor issue-reference naming cleanup rather than evidence that the coverage is obsolete or duplicated.

## Best Possible Solution

Keep the PR open for maintainer review or merge after addressing the minor naming/reference nit. The best outcome is to add the targeted non-strict-agentic NVIDIA/Nemotron regression tests to `src/agents/pi-embedded-runner/run.incomplete-turn.test.ts` while leaving production behavior unchanged.
