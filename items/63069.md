## Summary

Keep this issue open. Current main still hardcodes zsh and fish completion profile paths, still has the bash check/install mismatch, and the same-author PR #63082 remains the implementation candidate with closing syntax for this issue.

## Best Possible Solution

Keep this issue open and review, land, or replace #63082. The fix should centralize shell profile resolution in `src/cli/completion-runtime.ts`, respect `ZDOTDIR` for zsh and `XDG_CONFIG_HOME` for fish/PowerShell, make bash check/install use the same `.bashrc`/`.bash_profile` fallback, update doctor/update source hints to show the resolved path, and add focused unit tests for those profile-resolution cases.
