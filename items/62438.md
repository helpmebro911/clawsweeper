## Summary

Keep open. Current main still has the reported audit-path mismatch: env substitution is resolved before the security audit runs, but `collectSecretsInConfigFindings` still receives only the resolved `cfg` and checks `looksLikeEnvRef()` against that value. The related #67171 write-path cleanup does not supersede this audit false-positive path.
