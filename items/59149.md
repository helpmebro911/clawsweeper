## Summary

Keep open. The issue is a concrete, recent feature request for per-agent session visibility and agent-to-agent scoping, has positive follow-up discussion, and the current documented configuration still exposes these controls as global `tools.agentToAgent` and `tools.sessions` settings rather than the requested per-agent allowlists/scopes. The linked PR #59170 is still open per the supplied timeline, so this should not be closed as implemented.
