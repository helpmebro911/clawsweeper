## Summary

Keep open. Current `main` still documents `clawhub package publish --dry-run` in multiple plugin/ClawHub docs, so the reported docs/client mismatch remains real in this repo. Separately, the checked-out OpenClaw repo does not contain the `clawhub package publish` CLI implementation; it only links ClawHub as a separate service/repo and includes release-planning helpers that query ClawHub metadata. That means the publish-timeout portion is not disproven or resolved from `main`, and there is still an actionable docs bug here.
