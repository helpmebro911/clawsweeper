## Summary

Keep open. Current main has a generic `compat.requiresStringContent` path that can flatten text-only OpenAI-compatible message arrays, but NVIDIA's bundled provider catalog and the issue's documented config do not set that flag. The linked PR #50152 directly tracks the broader automatic normalization fix and remains relevant; current main does not strongly prove the NVIDIA report is fixed out of the box.

## Evidence
