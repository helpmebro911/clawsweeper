## Summary

Keep open. Current `main` does not contain this PR's streaming tar implementation, and the PR as written is stale against the refactored backup code plus current CLI/docs. The backup creation logic moved into `src/infra/backup-create.ts`, still uses `tar.c({ file: ... })`, and current CLI/docs do not support the PR's `--exclude-logs` / `--exclude-media` guidance. That means the PR is neither already implemented nor safe to auto-close as obsolete; it still represents potentially useful work that needs rebasing and revision.
