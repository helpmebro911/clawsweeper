## Summary

Keep open. Current main still creates Feishu docs via `createDoc()` only and does not chain `writeDoc()` when `content` is supplied. The linked PR #48377 tracks the same fix but is open/unmerged, so the reported behavior is not implemented on main.

## Evidence
