## Summary

Keep #10029 open. Current main still renders WebChat fenced and indented code as escaped monospace HTML with language labels, copy-button chrome, and JSON collapse behavior, but no syntax highlighter is wired into the WebChat markdown path. Related #30390 copy-button work is present and useful, but it does not satisfy syntax highlighting.

## Best Possible Solution

Keep #10029 open until WebChat implements real syntax highlighting in ui/src/ui/markdown.ts, using an approved highlighter or markdown-it highlight integration, Control UI theme CSS, and focused tests that prove highlighted fenced code while preserving the current language label, copy button, and JSON collapse behavior.
