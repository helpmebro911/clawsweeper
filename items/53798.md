## Summary

Keep open. Current main still routes Feishu `.opus` / `.ogg` message-tool media to native audio, but the Feishu upload path does not compute or pass `duration`, so the reported missing Feishu UI duration remains unfixed.

## Best Possible Solution

Keep this issue open and fix it in the bundled Feishu plugin media path. Compute a finite audio duration for native `.opus` / `.ogg` sends and for `audioAsVoice` transcode outputs, pass the millisecond duration into `uploadFileFeishu`, and add focused Feishu media regression tests proving `im.file.create` receives `duration` for both direct Opus/Ogg audio and voice-intent transcodes.
