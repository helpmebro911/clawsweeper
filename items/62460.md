## Summary

Keep open. Current main still has the reported behavior: QMD boot update calls `runUpdate("boot", true)`, and `force` still makes `shouldRunEmbed` return true. The requested `memory.qmd.update.embedOnBoot` config field is not present in the public config type, schema, help, or memory config docs.
