## Summary

Keep this PR open. Current main still lacks the unified Amazon plugin, the branch is a recent active implementation candidate, and a maintainer explicitly said the feature direction is valid while requiring rebase, docs/metadata, CI, and runtime/protocol fixes. The earlier Polly PRs are closed unmerged attempts consolidated by this PR, not a reason to close it.

## Best Possible Solution

Keep this PR open as the active consolidated Amazon voice/speech integration candidate. The contributor should rebase onto current main, add provider docs/changelog/labeler metadata, fix disabled Transcribe behavior, align Nova Sonic with AWS bidirectional event schemas and OpenClaw realtime audio contracts, address the outstanding Polly/Nova review comments, and return with green changed-surface CI for maintainer review. If maintainers ultimately decide AWS voice/speech should not be bundled, make that an explicit ClawHub/plugin placement decision rather than an automated cleanup close.
