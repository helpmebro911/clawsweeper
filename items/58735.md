## Summary

Keep open. Current `main` still hardcodes the control-plane write limiter at `3` requests per `60s`, does not expose a `gateway.controlPlane.rateLimit` config surface in the gateway config types/schema/docs, and therefore does not implement this PR's feature on shipped code.
