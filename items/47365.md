## Summary

Keep #47365 open. Current main still makes `openclaw logs` local-time output opt-in, while this PR is a focused implementation for the linked timezone bug and its Greptile dead-code feedback was addressed in the second PR commit. This is not implemented on main and should not be conservatively closed.
