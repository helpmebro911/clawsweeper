## Summary

Keep open. Current main supports live tool output cards inside the Control UI chat flow and a generic side panel for tool details, but it does not implement the requested split-view live exec/log panel with exec-specific stdout/stderr, duration, exit status, background monitoring, and controls. The related ClawSweeper items cover exec notification suppression/leaks or approval completion, not this Control UI layout request.
