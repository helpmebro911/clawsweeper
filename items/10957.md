## Summary

Keep #10957 open. Current main still documents Android as not publicly released, and the Android app remains in alpha/release-hardening state. The requested Android node capability surface exists, but that does not satisfy the reporter's request for Android beta/public access for an existing Gateway setup.

## Best Possible Solution

Keep #10957 open until maintainers either provide and document an Android beta/public distribution path for existing-Gateway phone nodes, or explicitly move Android beta-access requests to a support/auto-response process with clear public instructions. The Android node capability work can continue separately from the access/distribution decision.
