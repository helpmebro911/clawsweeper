## Summary

Current `main` has the generic exec-based secrets foundation, but it does not include Bitwarden/Vaultwarden-specific documentation or the proposed bundled `bw-exec-resolver` helper. This PR's follow-up work is therefore not already implemented on `main`, and the item is still a plausible, scoped enhancement rather than something safe to close conservatively.
