## Summary

Keep open. Current `main` still ships the shebang-based npm CLI entrypoint and still does unconditional `chmodSync` during task/task-flow registry restore, so the macOS 26 `com.apple.provenance`/`EPERM` report remains plausible and there is no strong evidence that `main` or `v2026.4.23` fixed it.
