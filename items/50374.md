## Summary

Keep open. Current main still lacks a strict message-key sanitizer for direct `models.providers` entries using `api: "openai-completions"`. Main has related compat knobs such as `supportsStore`, `supportsTools`, and `requiresStringContent`, plus provider/plugin hooks that could work around this. Ryce's KeepMy Claw gateway comment is a plausible workaround, but it does not make the core/config path obsolete.
