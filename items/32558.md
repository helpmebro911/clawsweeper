## Summary

Keep this PR open. Current main still lacks the Microsoft Teams Graph upload-session fallback: OneDrive and SharePoint helpers still do single `/content` PUT uploads, while the PR is a focused implementation candidate with tests for upload sessions. It needs maintainer review and a rebase, not cleanup closure.

## Best Possible Solution

Keep this PR open for Microsoft Teams maintainer review. Rebase it onto current main, preserve the current User-Agent behavior, fix the stall-count guard, confirm the fallback threshold, then land a focused `extensions/msteams/src/graph-upload.ts` implementation with regression tests for OneDrive and SharePoint upload-session behavior.
