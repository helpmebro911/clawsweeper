## Summary

Keep #7456 open. Current main still exposes only forward/cancel onboarding wizard navigation, with no typed Back result, no Clack Back handling, no wizard history/previous API, and no Gateway `wizard.previous`/back contract. The related Back PR #7762 was closed unmerged, #5880 is a distinct interrupted-resume attempt, and the dialog PR #18286 is not present on current main.

## Best Possible Solution

Keep #7456 open. The best path is a typed wizard navigation contract: add an explicit Back/previous-step result through `WizardPrompter`, the Clack adapter, `WizardSession`, Gateway wizard schemas/handlers, and macOS/Control UI rendering; refactor onboarding into a state machine that preserves answers while moving backward and forward; and add CLI/RPC regression coverage. PR #7762 is useful prior art but should not be treated as shipped.
