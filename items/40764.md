## Summary

Keep this PR open. Current main still does not provide an Android app setting to override talk-mode/STT locale, and the PR remains a focused implementation candidate, but it needs Android maintainer review, rebase/validation, and coverage before merge.

## Best Possible Solution

Keep this PR open as the active implementation candidate. The best path is to rebase onto current main, replace the unconditional reconnect with an endpoint-aware option refresh that does not surface a false failure while offline, add focused Android/Robolectric coverage for persistence and locale propagation, and get Android maintainer review on the language list and Settings placement.
