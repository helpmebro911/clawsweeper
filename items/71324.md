## Summary

Keep #71324 open. Current main has the #71229 baseline display projection, but it still does not cover the PR's central legacy cases: quote-wrapped internal runtime wrappers, persisted background-task status rows, gateway restart status blocks, and the added regression coverage. The related #71312 was closed unmerged and is superseded by this PR, not a canonical replacement.

## Best Possible Solution

Keep #71324 open and review it as the active gateway/WebChat history-projection fix. The best path is a narrow sanitizer in the chat-history display projection that hides only known legacy runtime-only rows, preserves following real user text and raw transcript sequence accounting, and lands with the added regression tests for quote-wrapped internal context, background-task status, gateway restart status, startup prompts, and subagent completion events.
