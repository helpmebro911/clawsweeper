## Summary

Keep #62492 open. Current main still matches the reported bug: `update.run` can return `status: "skipped"`, the gateway still wraps that as `ok: true`, and the UI only shows feedback when `ok === false`. The related PR #66865 is useful context and explicitly tracks this issue, but it is still open, so the bug is not implemented on main or superseded by merged work.
