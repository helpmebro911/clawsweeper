## Summary

Current main has not resolved the issue. The subagents guide still omits `streamTo` from the `sessions_spawn` params, still omits `announceTimeoutMs` from the sample/config discussion, still presents the depth table as 0-2 with depth-2 as never spawning, and still describes auto-archive as depth-1/depth-2 specific. Source confirms the reported fields and max depth range exist, so this remains actionable docs drift rather than a cleanup close.
