## Summary

Keep open. Current main has generic per-message timestamp/sender envelopes for thread/group history, but it does not implement the requested image-specific timestamp plus position annotation, nor the proposed `annotateImages` config. The related #48725 PR attempted a Slack-only implementation but was closed unmerged in the provided GitHub context, so there is no canonical implemented or superseding item to close against.

## Evidence

- **No annotation implementation found:** A targeted search for the proposed API/output strings found no `annotateImages`, `buildImageAnnotation`, image annotation line, or `thread.annotateImages` implementation in source, docs, packages, UI, or GitHub config.
  - command: `rg -n "annotateImages|buildImageAnnotation|Image — sent|Image - sent|message [0-9]+ of [0-9]+ in thread|thread\\.annotateImages|channels\\.slack\\.thread\\.annotateImages" src extensions docs packages ui .github`
