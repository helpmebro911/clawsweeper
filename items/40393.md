## Summary

Keep open. Current `main` and the latest release still silently ignore `thinking` for `sessions_spawn(runtime:"acp")`. `model` is now forwarded, but the issue is not fully implemented/fixed as reported because the ACP path still advertises `thinking` without accepting or applying it.
