## Summary

Keep open. Current main appears to address the later `is_from_me` echo-loop root cause, but the original cursor/backlog replay request is still not implemented: the iMessage monitor subscribes without any persisted rowid/since/start cursor, and repo search found no cursor persistence path for the legacy imsg monitor.
