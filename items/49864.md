## Summary

Keep open. Current main still resolves Telegram inbound mention gating with `activationOverride` ahead of `topicConfig?.requireMention`, so an explicit per-topic `requireMention: false` can still be overridden before the topic setting is applied. The linked PR #49980 is open/unmerged and directly tracks the requested fix, but it does not make the issue obsolete on main yet.
