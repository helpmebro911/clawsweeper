## Summary

Keep open. Current `main` still bakes the reported mismatch into the bare `/new` and `/reset` startup prompt: it tells the agent to mention `default_model` when the runtime model differs, while the runtime selection code and current status/model UX still distinguish selected/default from active runtime.
