## Summary

Keep open. Current main does not implement Anthropic's native `memory_20250818` tool or the requested client-side command handler. The bundled memory plugin currently exposes `memory_search` and `memory_get` only, while the related #70049 tracks a narrower generic `memory_add` direction rather than the full Anthropic native tool registration/CRUD behavior requested here. This remains a meaningful provider/runtime design request, not conservative cleanup material.
