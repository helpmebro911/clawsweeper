## Summary

Current `main` has adjacent ACP env-sanitizing work, but it does not implement this PR's per-agent ACPX spawn-env plumbing. The new runtime type exposes `env`, yet the session manager never passes it, `AcpInitializeSessionInput` cannot accept it, and the proposed helper symbols from this PR are absent on `main`. That leaves the PR's requested behavior unresolved rather than already implemented.
