## Summary

Keep open. Current `main` still has the underlying global-skill-env mutation, and the ACP control-plane paths on `main` do not thread any per-agent env overrides into `runtime.ensureSession()`. The repo has partial ACP client-side env stripping, but not the per-agent ACP runtime wiring described by PR #42269.
