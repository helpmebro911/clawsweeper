## Summary

Keep #47555 open. Current main handles many profile-aware service paths, but the reported agent-session failure can still occur on macOS because gateway-managed processes export `OPENCLAW_LAUNCHD_LABEL`, agent host exec inherits that environment, and launchd lifecycle resolution still gives that inherited label precedence over a later `--profile rescue`. The related fix PR #47594 is closed and unmerged, and current main does not contain equivalent coverage for this inherited-label conflict.

## Evidence

- **CLI profile parsing is not enough to clear inherited service overrides:** `runCli` applies `--profile` by setting `OPENCLAW_PROFILE`, but `applyCliProfileEnv` only updates profile/state/config/dev port and does not clear `OPENCLAW_LAUNCHD_LABEL`, `OPENCLAW_SYSTEMD_UNIT`, or `OPENCLAW_WINDOWS_TASK_NAME`.
  - file: [src/cli/run-main.ts:186](https://github.com/openclaw/openclaw/blob/d228463120a5057f7ccf9a5c31af62fe4a922088/src/cli/run-main.ts#L186)
  - command: `nl -ba src/cli/run-main.ts | sed -n '180,192p'; nl -ba src/cli/profile.ts | sed -n '72,99p'`
  - sha: [d228463120a5](https://github.com/openclaw/openclaw/commit/d228463120a5057f7ccf9a5c31af62fe4a922088)
- **Managed gateway environments include a fixed launchd label:** Gateway service environment construction sets `OPENCLAW_LAUNCHD_LABEL` from the active profile label, so commands executed from within the main gateway/agent process can inherit `ai.openclaw.gateway`.
  - file: [src/daemon/service-env.ts:292](https://github.com/openclaw/openclaw/blob/d228463120a5057f7ccf9a5c31af62fe4a922088/src/daemon/service-env.ts#L292)

- **Agent host command execution inherits the gateway process environment:** The bash tool builds host command env from `process.env` via `sanitizeHostExecEnvWithDiagnostics`; that sanitizer drops dangerous keys but otherwise preserves inherited keys, so `OPENCLAW_LAUNCHD_LABEL` is not removed before spawning `openclaw --profile rescue gateway restart`.
  - file: [src/agents/bash-tools.exec.ts:1536](https://github.com/openclaw/openclaw/blob/d228463120a5057f7ccf9a5c31af62fe4a922088/src/agents/bash-tools.exec.ts#L1536)
