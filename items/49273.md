## Summary

Keep open. Current main still defaults `commands.nativeSkills` to `"auto"`, Discord and Telegram still opt into native skill-command registration under auto mode, and the user-facing docs still describe that as the default. Main has quota mitigations for Telegram/Discord overflow, but the requested opt-in/default-off behavior remains unimplemented; the linked default-flip PR #49849 is closed unmerged and the narrower warning PR #50191 is still open.
