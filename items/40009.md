## Summary

Keep open. Current main still renders the Raw config editor as a native textarea, styles it as a textarea, and has browser tests that explicitly expect a textarea after sensitive-value reveal. I found no CodeMirror/Monaco/Ace integration or folding-capable JSON editor in the relevant UI surface, so the requested capability remains missing.
