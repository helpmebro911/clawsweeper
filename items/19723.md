## Summary

Keep open. Current `main` does not implement an `upload` action on `feishu_drive`. The only related upload capability is `feishu_doc.upload_file`, which requires a `doc_token` and uploads with `parent_type: "docx_file"`, so it is document-scoped rather than arbitrary Feishu Drive folder/root upload as requested.
