## Summary

Keep this PR open. Current main still documents the SearXNG JSON-format prerequisite but does not surface it during `openclaw onboard` or `openclaw configure --section web`, and the current PR remains a focused implementation candidate after moving the hint before credential collection.

## Best Possible Solution
