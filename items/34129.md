## Summary

Keep #34129 open. Current main still preserves the reported blank tool-call name plus `call_auto_*` fallback-id shape, and the linked sentinel fix remains represented by open PR #56391 rather than shipped code.

## Best Possible Solution

Keep #34129 open until PR #56391 or an equivalent maintainer-approved fix lands. The fix should prevent blank or whitespace-only model tool names without an inferable id from reaching dispatch as raw blank names with `call_auto_*` ids, and it should update regression coverage so this exact loop cannot return.
