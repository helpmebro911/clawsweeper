## Summary

Keep open. Current main does not implement the requested chat composer Expand button/modal. The composer has an auto-growing textarea capped at 150px, and the toolbar/run controls expose attach, voice/talk, token count, new session, export, send/queue/stop, but no large editor modal with Apply/Send semantics. The request is clear, user-facing, and belongs to core Control UI ergonomics rather than ClawHub/plugin scope.
