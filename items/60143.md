## Summary

Keep open. Current `main` appears to have fixed the narrow Brave legacy-path incident by auto-migrating legacy web-search config before validation, but it still does not implement the requested behavior of warning and disabling only the affected plugin when plugin config schema validation fails. Plugin config errors are still recorded as fatal `issues`, and `loadConfig()` still turns any such issue into `INVALID_CONFIG` and aborts startup.

## Evidence
