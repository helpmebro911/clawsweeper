## Summary

Keep open. Current main still does not provide a built-in Accessible Control UI theme preset. Main has partial related work: improved default dark muted contrast, three built-in theme families, and a browser-local tweakcn custom theme slot. That does not satisfy this issue's request for a first-class high-contrast, colour-vision-deficiency-safe preset in the theme picker.
