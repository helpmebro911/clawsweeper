## Summary

Keep #49162 open. Current main has adjacent reliability pieces, including provider/query embedding timeouts, hard-coded embedding retry loops, QMD search timeout config, and an embedding cache. It does not implement the requested schema-backed `agents.defaults.memorySearch.query.timeoutMs`, `query.cacheTtlMs`, or `query.retry` controls, nor a configurable query-result cache TTL. Ryce's comment links related durability work (#49056 and #49233), but this issue still tracks a distinct core config/runtime reliability surface.
