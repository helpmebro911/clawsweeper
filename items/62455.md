## Summary

Keep open. Current main still rejects `channels.discord.guilds.<guildId>.channels.<channelId>.agentId`; I did not find support for that shape, an actionable validation hint for migrating it to `bindings[]`, or a doctor repair rule. The issue has no maintainer/protected-label override in the provided context, but the requested usability fix remains missing.
