## Summary

Keep open. Current main has useful tracked-tab cleanup for idle/excess sweeps, session reset/delete, and subagent/cron lifecycle end, but it still does not close primary-agent browser tabs/windows automatically at ordinary task completion.

## Best Possible Solution

Keep this issue open for a focused browser lifecycle change: add an ephemeral marker for automation-opened tabs, preserve explicit/labeled/pinned/user-owned tabs, wire cleanup into ordinary primary-agent task or turn completion, and keep the current idle/excess sweep plus reset/lifecycle cleanup as fallback behavior.
