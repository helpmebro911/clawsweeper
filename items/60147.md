## Summary

Keep open. Current `main` does not contain the dependency/lockfile updates from PR #60147, so this is not `implemented_on_main`. The shipped A2UI build path still points at the vendored lit renderer, but most of the PR's churn is in vendored angular/spec-eval subtrees that current build and Docker paths do not reference, so this needs maintainer scoping rather than an automatic cleanup close.
