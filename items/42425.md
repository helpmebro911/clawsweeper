## Summary

Current `main` still loads internal hooks only from `defaultWorkspaceDir` during gateway startup, so the non-default-agent workspace hook gap described in #42072/#42425 is still present. The PR also shows substantive follow-up fixes in discussion, so this is not a stale or already-implemented cleanup candidate.
