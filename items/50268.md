## Summary

Keep open. Current main added preservation plumbing, but the concrete bundled-provider path described by the issue can still overwrite the existing primary before that preservation logic sees the original value. The external Keep My Claw comment is a recovery workaround, not a core fix for the silent primary-model mutation.
