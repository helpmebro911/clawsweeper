## Summary

Keep open. Current `main` has a newer generated Control UI i18n pipeline and already-shipped zh-CN locale output, but it still does not implement this PR's global top-bar locale switch and still contains hardcoded English in the dashboard shell/update banner. The only current locale selector also still saves settings before `setLocale()` finishes, so the review thread's async-state concern remains materially relevant. This PR needs maintainer rebase/rework, not conservative auto-close.
