## Summary

Keep #12554 open. Current main has managed exec/process cancellation for individual background exec sessions and subagent-run kill plumbing, but it still lacks the requested session-wide operator surface to list all exec processes for an OpenClaw session and terminate them as a unit.

## Best Possible Solution

Keep #12554 open and implement an additive Gateway/CLI/RPC operator surface: resolve the canonical session key, list managed exec runs for that session scope, terminate matching runs through process supervisor cancellation plus process-tree fallback, return per-process status, document the contract, and cover same-session kill, unrelated-session isolation, background exec, HTTP/RPC/CLI validation, and subagent cancellation interactions in tests.
